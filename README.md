# 量子情報網理論 (Quantum Relational Network, QRN)

**本リポジトリは完成した万物理論を主張しない。** 量子情報・創発時空・ゲージ制約・
標準模型的スペクトルに関する既存理論・toy model・制限付き全数探索を、
**再現可能な計算体系 (executable atlas)** として整理した研究プログラムである。
寄与は「実行可能な機構・仮定・有限探索・反証可能な予想のアトラス」であり、
全主張は [claims.yml](claims.yml) で 6 等級 (C0 前提 / C1 再現 / C2 制限付き計算定理 /
C3 機構 / C4 現象論 / C5 解釈) に分類され、証拠と限界つきで機械検証されている
(`v61_ledger`)。

動機は「群盲、象を評す」— 一般相対論・量子力学・場の量子論・統計力学はそれぞれの領分で
驚異的に正しいのに、全体像が見えていない。本プロジェクトは、現代物理の各理論を
**Rust による第一原理シミュレーション**(外部依存なし)で一つずつ再現・検証しながら、
それらが収束する統一的描像の作業仮説

> 時空・重力・力・物質は、単一の量子相関網の異なる読み出しである (C5)

を段階的に検証してきた記録である。公理系と反証条件は [docs/uft-v1.0.md](docs/uft-v1.0.md)、
最新の統合と未解決問題の残高は docs/ の最新 vX.0 文書を参照。

## 実行方法

```bash
cd sim
cargo build --release          # 外部依存なし (std のみ)
./target/release/v01_qm        # 各バージョンのバイナリを実行
```

全シミュレーションの出力は `results/` に保存済み。各実験には厳密解・観測値との比較
([PASS]/[FAIL] 判定) が組み込まれている。

## リポジトリ構成

```
CLAUDE.md        プロジェクトの指示
PROMPT/          原点の指示と改良方針
claims.yml       主張台帳 (全主張の等級・証拠・限界。v61_ledger が機械検証)
docs/            バージョン付き理論文書
sim/             Rust ワークスペース (src/lib.rs: 乱数/複素数/Jacobi固有値/Bessel/SHA-256 等を自作)
  src/bin/       各バージョンのシミュレーション
results/         実行出力 (文書中の数値の一次ソース)
```
