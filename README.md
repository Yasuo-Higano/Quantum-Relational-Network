# 統一場理論プロジェクト — 量子情報網理論 (QRN)

「群盲、象を評す」— 一般相対論・量子力学・場の量子論・統計力学はそれぞれの領分で驚異的に
正しいのに、全体像(象)が見えていない。本プロジェクトは、現代物理の各理論を **Rust による
第一原理シミュレーション**で一つずつ再現・検証しながら、それらが収束する統一的描像

> **時空・重力・力・物質は、単一の量子相関網の異なる読み出しである**

を段階的に組み上げた記録である。最終成果は [docs/uft-v1.0.md](docs/uft-v1.0.md)(公理系・
数値的裏付け・予言と反証可能性・未解決問題)。

## 実行方法

```bash
cd sim
cargo build --release          # 外部依存なし (std のみ)
./target/release/v01_qm        # 各バージョンのバイナリを実行
```

全シミュレーションの出力は `results/` に保存済み。各実験には厳密解・観測値との比較
([PASS]/[FAIL] 判定) が組み込まれている。

## リポジトリ構成

```
CLAUDE.md        プロジェクトの指示
docs/            バージョン付き理論文書
sim/             Rust ワークスペース (src/lib.rs: 乱数/複素数/Jacobi固有値/Bessel 等を自作)
  src/bin/       各バージョンのシミュレーション
results/         実行出力 (文書中の数値の一次ソース)
```

## 知的誠実性について

- 各文書は「教科書的に確立された事実」「確立された研究成果」「本プロジェクトの仮説」を区別する。
- これは完成した万物の理論ではない(標準模型のパラメータは導出できていない)。
  v1.0 §6 に未解決問題を、§4 に理論を殺せる反証条件を明示してある。
- 設計ミスや失敗した検証(例: v0.7 の「1 量子は無限小でない」)も記録している。
