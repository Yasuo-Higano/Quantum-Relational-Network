## 1.3 現在のモデルは Fermi 面ではなく、点ノード型に見える

[
D_y=(-1)^x\frac{T_y+T_y^{-1}}2,

[

となり、バルク分散は

[
E(\mathbf k)
=\pm\sqrt{
\cos^2 k_x+

したがって零エネルギー集合は

[
\cos k_x=\cos k_y=\cos k_z=0

という点ノードの集合であり、余次元1の Fermi 面ではありません。これはソースコードからの解析的推論であり、有限開放 x 境界はバルク分類を変えません。 ([GitHub][3])

[

は Fermi 面を持つ自由フェルミオンに対する結果です。一般のノード系では余次元が重要で、少なくとも3次元 nodal line でさえ通常の面積則に戻り、係数は非普遍的です。相対論的 Dirac 真空についても UV 正則化の下で面積則が示されています。 ([arXiv][6])

[

は、このモデルに対する適切な二択ではない可能性が高いです。後者は

[

[
S(N)=\alpha N^2+\text{subleading terms}
]

です。滑らかな曲面で universal logarithm が存在する場合でも、(s_{\rm area}=S/N^2) では候補形は概ね

[

===============

\alpha+
\frac{\beta\log N+\gamma}{N^2}

# 2. 「エンタングルメント第一法則」と重力を分離する

任意の基準状態 (\rho_0) に対して

[
K_0=-\log\rho_0
]

とすると、

[
\delta S
=-\operatorname{Tr}(\delta\rho\log\rho_0)
=\operatorname{Tr}(\delta\rho K_0)
=\delta\langle K_0\rangle
]

です。ここでは (\operatorname{Tr}\delta\rho=0) を使っています。

つまり一次の entanglement first law は、正しい (K_0=-\log\rho_0) を使えば一般の密度行列で成立する恒等式です。数値的に比が1になることは、

* RDM 構成
* 行列対数
* 有限差分
* 固有値処理

の検査としては有益ですが、それ単独では重力の証拠ではありません。v22.3 自身も、[g] を「modular 機構の器械恒等式」であり「曲率・力学的重力ではない」と明記しています。 ([GitHub][9])

Jacobson の導出には、単一領域の第一法則以外に、

* 各時空点を通るすべての局所 Rindler horizon
* horizon area と entropy の比例
* matter のエネルギーフラックス (\delta Q)
* 加速観測者の Unruh 温度
* causal geometry の変形

が必要です。小球による entanglement equilibrium 版でも、固定体積、小さな geodesic ball、局所的 maximally symmetric vacuum、一次変分などの条件があります。 ([arXiv][10])

また holographic な導出では、すべての球領域に対する第一法則に加え、Ryu–Takayanagi/Wald 型の面積辞書と semiclassical bulk geometry が必要です。 ([arXiv][11])

現在の v23.6 は、

1. Jacobson の熱力学的論理
2. raw lattice entanglement area coefficient
3. Sakharov 型 induced gravity の解釈

を一つの式で接続しています。しかし Jacobson と Sakharov は必要な仮定も計算対象も異なります。次期では明示的に二つの経路に分けるべきです。

### 経路A: Jacobson 型

複数位置・複数向き・複数形状の局所領域で、同一の温度、stress tensor、area variation が成立し、そこから共通の幾何応答方程式が得られるかを検証する。

### 経路B: Sakharov 型

背景計量 (g_{\mu\nu}) または vierbein を格子作用へ導入し、

[
\Gamma[g]=-\log Z[g]
]

を直接計算して、

[
\Gamma[g]
=========

\int d^4x\sqrt g
\left[
\Lambda_{\rm ind}
+\frac{1}{16\pi G_{\rm ind}}R
+c_2R^2
+c_3R_{\mu\nu}R^{\mu\nu}
+\cdots
\right]
]

の係数を抽出する。

少なくとも、

[
G_{\rm entropy}=G_{\rm effective\ action}
]

が同じ renormalization scheme で一致しなければ、「誘導された Newton 定数」とは呼べません。宇宙定数項 (\Lambda_{\rm ind}) と高曲率項を無視することもできません。

# 3. 統一場理論としての完成条件

QRN を「統一場理論候補」と呼べる最低条件は、次の5つです。

## 3.1 単一の microscopic theory

単一の Hilbert 空間、局所代数、Hamiltonian/action、有限個の無次元パラメータから、物質・ゲージ・幾何が同時に出ること。

単に既知の表現

[
(3,2,1/6)
]

を入力し、その代数を読み戻すことは「標準模型の導出」ではなく「標準模型表現の符号化」です。

## 3.2 制御された 3+1D 連続極限

次を別々に定義する必要があります。

[
\begin{aligned}
\text{thermodynamic:}&\quad L/a=N\to\infty,\ a\text{ fixed},\
\text{continuum:}&\quad a\to0,\ L\text{ fixed},\
\text{massless:}&\quad ma\to0,\ mL\text{ controlled},\
\text{BW window:}&\quad \xi\to\infty,\ \xi/N\to0.
\end{aligned}
]

現在はこれらが「(N\to\infty)」という一語で混在しています。

## 3.3 カイラル標準模型

v22.3 は (N=4) の 1+1D core であり、弱左手性は doubling の外に残されています。これは機構実演としては有効ですが、3+1D SM の連続理論ではありません。 ([GitHub][9])

次に必要なのは、

* 3+1D
* (SU(3)\times SU(2)\times U(1))
* 全 Weyl multiplet
* 正しい hypercharge
* gauge anomaly cancellation
* light mirror fermion の不在
* Higgs/Yukawa
* 動的 gauge field
* 正しい RG と低エネルギースペクトル

です。

Ginsparg–Wilson 関係は格子上の厳密なカイラル対称性と doubling 回避の制御された出発点になります。anomaly-free U(1) には非摂動的な gauge-invariant 構成があり、一般コンパクト群にも全摂動次数での構成があります。QRN 独自の方式を試す場合も、まずこの既知基準と比較すべきです。 ([arXiv][12])

## 3.4 力学的重力

最低でも、

* 低エネルギー massless spin-2 mode
* 2つの物理 helicity
* diffeomorphism Ward identity
* matter 種に依存しない universal coupling
* backreaction
* Einstein–Hilbert 項
* 高曲率補正の制御
* Lorentz symmetry の回復

が必要です。

entropy coefficient 一つだけでは、このどれも確認できません。

## 3.5 凍結後の新規予測

QRN の予測台帳は失敗も残している点で優れています。一方、凍結された幾何湯川・ニュートリノ枝には (|V_{td}|)、(\beta)、(\theta_{12})、特に (\theta_{23}) の miss が記録されています。広い帯による (\theta_{13}) hit も弱い hit と自己評価されています。従って、現行の物質 phenomenology は統一理論を支持する証拠ではなく、構造変更が必要な反証結果として扱うべきです。 ([GitHub][13])

失敗後に同じデータへ再調整するのではなく、構造仮説を変更し、別の observable を新しい holdout として凍結する必要があります。
