# 統一場理論 v12.3 — 磁束族の走査: N=6 の傾き窓に深い階層はない

**Version**: v12.3
**Date**: 2026-07-05
**シミュレーション**: `sim/src/bin/v123_fluxscan.rs` → `results/v123_fluxscan.txt` (機械可読: `results/v123_fluxscan.json`)
**対応残高**: 残高 2 — 傾きスレッドの境界確定

---

## 1. 問題

v12.2 の陰性 (磁束 (2,2,1,−1) で lnZ₉ = −94.5) は 1 点の結果だった。同じ N=6 格子で
指数 Pf(F) = Q₁Q₂ + ts = 3 を保つ磁束は他にもある — 深いタワー (Q=3) と傾きの
両立点を系統的に探す。

## 2. 結果 — 走査表 (全 6 点で指数 3 は厳密)

| 磁束 | 縮退幅 / ギャップ | 深さ代理 ln r₁ | MI(x₁:x₂) |
|---|---|---|---|
| (3,1,0,0) 因子化参照 | 6×10⁻¹³ / 0.334 | −2.14 | 0 |
| **(2,2,1,−1)** | 7×10⁻¹³ / 0.320 | **−3.58** (最深) | 0.0067 |
| (3,2,3,−1) | 5×10⁻¹³ / 0.199 | −2.17 | 0.0013 |
| (3,2,1,−3) | 4×10⁻¹³ / 0.199 | −2.17 | 0.0135 |
| (3,3,2,−3) | 3×10⁻¹³ / 0.166 | −1.50 | 0.0006 |
| (3,3,3,−2) | 5×10⁻¹³ / 0.166 | −1.50 | 0.0006 |

- **反直観の発見**: 磁束を上げるほど浅くなる (Q=3 系で −1.50)。N=6 の小格子では
  磁束が強いほどモード幅が格子に対して太り、重なりの分離が失われるため。
  T² 系 (N=18) の「Q=3 が深い」直観は小格子では逆転する。
- 勝者 = 既検定の (2,2,1,−1)。その全 36 Wilson 証拠は −94.51 (v12.2 と一致 —
  同一機械の回帰にもなっている)。必要な深さは ln r₁ ≈ −11.3、走査窓の最深は −3.58。

## 3. 結論 — 境界の確定と道具の限界

**N=6 の指数 3 傾き族には、観測階層に届く点は存在しない** (走査で確定)。
より深い階層には格子の拡大が必要 (N=12 で site 20736 → 行列 41472² は現行の
稠密ヤコビ法では不可能)。**疎行列固有値ソルバ (Lanczos 系) が lib.rs にない**ことが
この糸の現在の道具の限界である — 傾き T⁴ スレッドはここで一区切りとし、
構造の成果 (指数 = Pf(F) の実証、v12.1) と境界 (本結果) を残高 2 に記録する。

## 4. 限界

- 走査窓: |磁束| ≤ 3・N=6・ヒッグス等方ガウス。深さ代理は cfg0 の r₁ (証拠の完全な
  下界ではないが、−3.58 vs 必要 −11.3 の隔たりは代理の粗さで埋まらない)。
