## 0. 第七期の方針転換 — 横に広げず、固める

第六期までに 32 本のシミュレーションと 87/87 PASS を積んだ。第七期は新しい物理項目を
**足さない**。代わりに、既存の成果を「査読可能な主張」に分解して固める(原点は
`PROMPT/1.改良アドバイス.md`)。理由は単純で、本プロジェクトの弱点は物理の広さではなく、

> 「どこまでが導出で、どこからが解釈か分からない」

という批判に一撃で答える仕組みがなかったことだからである。v6.1 はその仕組み
= **主張台帳 (claim ledger)** を導入する。

## 1. 等級の定義

全主張を次の 6 等級に分類する。

| 等級 | 意味 | 例 |
|---|---|---|
| **C0** | 教科書的・既存定理(本プロジェクトの前提) | QNEC の証明、Page 公式、Unruh 温度 |
| **C1** | 既存結果の再現実装(本リポジトリの計算) | 第一法則 δS=δ⟨K⟩ の 1+1d 数値検証 (v0.7) |
| **C2** | 制限付き計算定理(明示領域内の全数計算) | v3.1/v4.3/v5.2 のアノマリー探索 |
| **C3** | toy model による機構提示(適用は仮説) | 相互情報量からの円環復元 (v0.7) |
| **C4** | 現象論的 fit / オーダー推定 | FN 電荷で 9 量を 5 倍以内 (v3.2)、Λ〜1/√V₄ |
| **C5** | 解釈・哲学・スローガン(反証条件つき) | 「時空はもつれの夢」、各期の中心テーゼ |

各主張は `claims.yml` に 1 エントリで、**証拠**(コード・結果ファイル・文書。実在が機械検査
される)と**限界**(C0 以外は必須)を持つ。C2 は探索領域を `inputs:` として機械可読に固定する。

## 2. 集計 (v61_ledger による機械検証: 全 6 検査 PASS)

```
C0 前提 6 / C1 再現 27 / C2 計算定理 4 / C3 機構 5 / C4 現象論 6 / C5 解釈 5  (計 53)
証拠参照 95 件すべて実在 / 全 32 バイナリが台帳から参照されている
```

分布が語ること(正直な自己評価):

- 主張の過半 (27/53) は **C1 = 既存結果の再現**である。これは弱点ではなく本プロジェクトの
  性格そのもの — 「実行可能な計算アトラス」— だが、**新規性を主張できる場所ではない**。
- 新規性の候補は C2 の 4 件(アノマリー探索系)に集中している。だからこそ v6.2 で
  この 4 件を「証明に近い計算」へ格上げする(独立実装・証明書・陰性対照)。
- C4 (fit 系) は「選んだパラメータで合った」と「合うパラメータは稀」の区別がまだない。
  v6.5 がベイズ模型比較で白黒をつける。
- C5 は 5 件。スローガンは研究の推進力だが、台帳上は解釈であると明示された。

## 3. 主要な再分類(このバージョンで明確化されたこと)

1. **v3.1「SM 世代が最小・一意」は C2 であって定理ではない** — 正確には「明示された探索領域
   (表現 ≤(3,2)、|Y|≤3/2、≤5 多重項、全因子帯電) 内での計算定理」。ゲージ群は仮定。
   領域の物理的正当化は EFT 的仮定(v5.3 の脱結合が部分的に支える)。
2. **v0.7「空間の復元」は C3** — 円環という最単純ケースの機構実証。陰性対照(幾何を
   持たない状態で壊れること)が未検証だった → v6.4 で対処。
3. **v3.2 FN は C4 の M1 級** — 「文献の電荷を選んで合った」段階。電荷全探索・null model
   との比較・学習/予測の分離がない → v6.5 で対処。
4. **v4.1 QNEC は C1(既存定理の離散 toy 再現)** — 「QNEC の発見」ではない。しかも強形の
   最小ギャップ −3e-5 を経験的 tolerance 2e-4 で PASS としており、誤差予算(微分幅・
   有限サイズ・丸め)の分解がない → v6.3 で対処。
5. **BMV 反証条件 (v2.5) は 2025 年の議論で精密化が必要** — 「もつれが出れば量子重力」は
   言い過ぎになりつつある → v6.6 で更新。

## 4. 既存研究との差分表(何を加え、何を加えないか)

| 既存研究 | 既存研究の主張 | QRN が加えるもの | QRN が加えないもの |
|---|---|---|---|
| RT 公式 (Ryu–Takayanagi 2006) | もつれエントロピー=極小面積 | toy 模型群での再現・教育的実装 (v0.5) | AdS/CFT の新証明 |
| Van Raamsdonk 2010 | もつれ=時空の接続性 | 相互情報量幾何の計算実験 (v0.7, v1.2) | 一般時空の導出 |
| Jacobson 1995 / FGHMV 2013 | 第一法則⇔(線形化)Einstein 方程式 | 1+1d 平坦版の直接数値検証 (v0.7) | 非線形 Einstein 方程式の完全導出 |
| Almheiri–Dong–Harlow 2015 | バルク局所性=量子誤り訂正 | 最小模型の厳密検証 (v1.3) | 新しい code subspace |
| ER=EPR (Maldacena–Susskind 2013) | もつれた BH=ER 橋 | TFD の最小実演 (v1.2) | ER=EPR の証明 |
| QNEC (Bousso+ 2016 ほか) | S″ ≤ 2π⟨T_kk⟩ | 離散 toy での数値確認 (v4.1) | QNEC の新証明 |
| アノマリー消去 (標準) | SM 電荷の無矛盾性 | **制限領域での最小性・一意性・孤立性の全数探索 (v3.1/v4.3/v5.2)** | ゲージ群自体の導出 |
| Froggatt–Nielsen 1979 | 階層=電荷の算術 | O(1) 係数 MC での 9 量一括検証 (v3.2) | 電荷・ε の第一原理 |

| 因果集合 (Sorkin ほか) | 順序+数=時空、Λ〜1/√V₄ | 次元復元・ローレンツ両立の再現 (v0.6, v1.5) | 動力学の完成 |

この表は「既存研究の寄せ集めでは?」という批判への先回りである: **その通り大半は
寄せ集め(C1)であり、それを明示した上で、C2 の探索系だけが新規性を主張しうる**。

## 5. 運用規約(以後の全バージョンに適用)

2. `v61_ledger` が常に PASS であること(CI にも組み込む — v6.6)。
3. 文書中の等級表記は台帳を一次ソースとする。
