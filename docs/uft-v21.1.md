## 2. 装置 — スペクトル関数法 (v2)

シード (Ê_mean − ⟨E⟩)|0⟩ の 3 項 Lanczos (再直交なし・メモリ自明) の Ritz 値と
重み |s_j|² = Ê のスペクトル関数。**「E が結合する状態」だけが重みを持つ**ので、
多フレーバーの無質量擬モード (E と結合しない) を自動的に素通しする — ボソン同定が
重みとして内蔵される。{1}/{1,1}/{2} で極重み 0.96〜1.00。

開発記録 (run1, 保存): (i) 励起状態の素朴 Lanczos (k=8, m=170) は非収束 (res ~1e0) —
変分違反として発覚、(ii) {2} の E₁ はフラックス状態でボソンでない、(iii) cross-N
回帰帯 ±2% は根拠曖昧で 2.002% で落ちた — 全てゲートが捕捉し、v2 で根治。

## 3. 残高への効き

- 橋の完成度: **Σq² = 2 で実証・Σq² ≥ 4 は二種の窓 (格子・共鳴) の外**。窓を開ける
  正道は大きい系 = **§7R (DMRG)** — 本版が MPS 基盤の必要性を定量的に裏書きした。
- run1→v2 の教訓「スペクトル関数はシードで問いを選ぶ」は以後の分光の標準器械。

## 4. 限界

- N=10・x ≤ 4。重電荷は未読 (窓)。共鳴幅の測定は未実装。
- 1+1D の vector-like 理論 — カイラル割当そのもの (Σq_L² = Σq_R²) の動力学検定は
  格子カイラルフェルミオンの一般的困難 (doubling) の内側で、別途の定式化が要る。
