## 1. 構成 — 全て第一原理の数値計算

1. **世代 = 磁束のゼロモード**: 磁束 Q=3 の 18×18 トーラスを厳密対角化。縮退した
   3 つの最低モード (縮退幅 2e-13 ≪ ギャップ 0.115) が 3 世代。
2. **世代の住所**: 縮退空間で位置演算子を対角化すると、N/Q = 6 サイト間隔で並ぶ
   局在波束 (LLL のシータ関数的モード、幅 ≈ 2.9) が得られる。
3. **セクターの区別 = Wilson 線**: y 方向 Wilson 線 k_a ∈ Z₆ がゼロモード中心を
   k_a サイトずつ厳密に平行移動する (6 回の対角化で検証)。
4. **湯川 = 重なり積分**: Y_ij = Σ_x ψ_i*ψ_j φ_H (Higgs はガウス、幅 σ_H)。
   **乱雑な O(1) 係数は存在しない** — 行列は幾何が完全に決める。

パラメータ: Wilson 線整数 (単一 T²: 5 個 ∈ Z₆ / T²×T²: 10 個) + σ_H (4 通り)。
証拠は v6.5 と同一の尤度 (σ=ln2、学習 = 質量比 6 つ、CKM は out-of-sample)。
