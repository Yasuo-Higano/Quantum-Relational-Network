崩壊」(3 点) を、f64 床の生存窓 (d ≤ 1.5) 内の 6 点で検定した結果、
**合併単調性も残差ゲートも破れた**。これで R₂ 不足の幾何仮説は 3 本とも死に、
G 窓の障害は「ブースト重み同定の構造的問題 + f64 床」という装置の言葉に帰着した。

## 2. 何が死んだか — 幾何仮説の全滅

1. **横質量仮説** (v22.2 で反証): m_T = 0 でも R₂ 不変。
2. **d/ξ 崩壊仮説** (本版で反証): N=12 は d 非単調 (d=1.0 で 0.898 > d=1.5 の
   0.865 — v22.2 の「単調」は 2 点ゆえの錯覚)。N=14 は d·gap が 0.33→0.99 と
   3 倍動いても R₂ ≈ 0.80 で平坦。
3. **ξ/L_A スラブ仮説** (本版で同時に反証): gap(N=14) = 0.6632 (シェル構造は
   N mod 4 に敏感 — 12, 16 ≡ 0 は準ゼロ横モードを持ち、14 ≡ 2 は持たない)。
   ξ = 1.5 と最短・ξ/L_A = 0.21 と最小の N=14 が R₂ 最低域 (0.80) —
   スラブ仮説の予言 (短 ξ → R₂ → 1) と正反対。

## 3. 何が生き残ったか

- R₂ ∈ [0.79, 0.90] の不足は **シェル構造 (N mod 4, 横モードの縮退パターン) に
  敏感で、幾何の一変数関数ではない** → ブースト重み K_boost の格子離散化
  (v19.2 凍結規約) がシェル依存の O(1) 補正を持つ、が残る唯一の説明系。
- **器械知見 — カナリアの共通モード盲**: R₁ = δS/δ⟨K_A⟩ は分子分母が f64 床
  バイアスを共有して部分相殺する (だから d ≤ 1.5 で 1e-4 の精度)。一方 R₂ の
  分母 (核なしブースト重み) は床と無関係なので、δS 側の欠損を丸ごと露出する。
  カナリア有効は「核の自己整合」であって「δS の完全性」ではない。

## 4. 終端宣言 (堂々巡り条件)

幾何ノブ (m_T, N, d, ξ/L_A) の走査はこれ以上の情報を生まない — 3 仮説の反証で
プログラムは終端。G 窓を開ける残り路線は装置側の 3 本 (named limits):
(i) 縮退クラスタの DD-Gram 多倍長核、(ii) κ 床非依存の modular 読み出し、
(iii) K_boost 離散化の改良 (シェル依存補正の理論)。
