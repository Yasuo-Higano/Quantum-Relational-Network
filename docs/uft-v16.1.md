## 1. 結果 — CP は「傾き」では買えない

エンジンは v13.2 を完全回帰 (36 Wilson 配置の残差 max 8.8e-12・縮退幅 max 1.5e-12、
lnZ₉ = **−34.8804** = 公表値と 4 桁一致)。その上で:

| 量 | 傾き T⁴ (3,3,1,−6) 全窓 ~19 万点 | 陽性対照 (因子化・厳密積) | 測定 (PDG) |
|---|---|---|---|
| max \|J\| | **2.6×10⁻⁸** | **1.6×10⁻⁴ (格子スケール)** | 3.08×10⁻⁵ |
| MAP の J | −1.3×10⁻¹⁷ | — | — |
| P(\|J\| > 10⁻⁶) | 0.000 | — | — |
| MAP の \|V_td\| | 0.0053 | — | 0.0086 |

二つの発見が重なっている:

1. **因子化スカラー格子の J は 0 ではない** — 連続体 (長方形トーラス・実 Wilson) の
   CP 保存 (実化可能性) は格子で破れ、J は格子スケール ~1.6×10⁻⁴ (O(1/N²) 級の
   位相残差) に立つ。v15.7 の厳密零 (8×10⁻¹⁷) は Dirac 構成 + 局在基底 + Hadamard
   積という**別の構造**の性質だった — 「因子化 ⇒ J=0」は構成に依存する。
2. **傾き厳密縮退点の J はその格子スケールより 6000 倍深い** — 2.6×10⁻⁸ は装置の
   検出床 (陽性対照が 1.6×10⁻⁴ を現に検出) ではなく、**傾き点に固有の構造的な
   CP 抑制**である。ゼロモードが非因子化で湯川が複素なのに、CKM は格子
   アーティファクト水準を 4 桁下回って実に留まる。

深さ (v13.2: S₃ 対に 15 nats 負け) に続き、**CP でもこの窓は死んだ** — しかも
「たまたま小さい」のではなく、何かがそれを守っている。

## 2. 機構は開いた問い — 素数 3 の影

なぜ複素な湯川から実な CKM が出るのか。機構候補は v13.2 の「単一タワー」構造:
Pf(F) = 3 (素数) の厳密縮退点では 3 世代が単一の磁気並進軌道を成し、並進の
クロック構造が位相を拘束する — 深さ (2 タワーの掛け算の禁止) と CP (位相の消去)
が同じ群論的根を持つ、という推測である。ただし本版では**推測のまま残る**:
検証に使おうとした湯川位相プラケット arg(Y_ij Y_kl Ȳ_il Ȳ_kj) は、縮退空間の
モード基底 (Lanczos/jacobi が返す) の任意ユニタリ回転に依存する**基底依存量**で
あり、診断として無効だった (実測 ~π がその証拠 — 開発記録 §3)。基底不変な CP
検定量は J (∝ Im tr[H_u, H_d]³) 自身であり、それが床レベルというのが本版の全て。
機構の基底不変な診断の設計は次期の的である。

## 3. 開発記録 — 対照が三つの盲点と一つの発見を運んだ

対照 (因子化磁束 [3,1,0,0] → J=0 のはず) の構築が三度想定を裏切り、教訓 3 件と
発見 1 件 (§1 の格子スケール J) を残した:

1. **Lanczos の完全性の盲点**: 初版は対照を 4D Lanczos で解き「縮退幅 3.9e-2 の分裂」を
   得た。単一 T² の厳密対角化で切り分けると、T²(Q=3, N=18) の LLL は厳密 3 重 (1e-13)、
   T²(Q=1) は間隔 0.0385 の非縮退ラダー — Kronecker 和の底は必ず 3 重で分裂は不可能。
   真因は縮退部分空間の**取り逃し**: 残差は見つけたベクトルの質を測るが、取り尽くしを
   保証しない。Kronecker 和の密な踏み台スペクトルで縮退パートナーを逃し、次段の状態が
   3 番目として返っていた (v13.1「答えは残差とセットで読む」の限界の新しい顔)。
   → 対照は単一 T² の稠密厳密対角化 → 積波動関数の構成に変更 (完全性リスクなし)。
2. **CKM の物理的な組**: 対照の第 2 版は左場 (kQ) を u/d で共有しない非物理な組まで
   比較し偽の J (~0.1) を出した。CKM = U_u U_d† が物理量なのは**同じ左二重項**の
   基底変換が相殺するからであり、共有 kQ の組だけが意味を持つ ([2] の五重和と同じ
   添字構造に統一)。また Wilson 線は磁束量子ステップ (w = 2πQk/N², v10.1 規約) に
   変更 — 2πk/6 の大ステップは N·w ∈ 2πZ でゲージ自明になり対照の意味を失う。
3. **診断量の基底依存性**: 機構検証に使った湯川位相プラケット arg(Y_ij Y_kl Ȳ_il Ȳ_kj)
   は縮退空間のモード基底の任意回転に依存する量で、診断として無効だった
   (実測 ~π がその証拠)。基底不変な CP 検定量は J のみ。

修正後の対照は「J=0 の再現」でなく**格子スケールの J (1.6e-4) の検出**を返した —
これは対照の失敗ではなく前提の訂正である (§1 の発見 1)。「対照は装置の一部」(v6.4)
の実演: 対照が通らない限り本体を公表しない規律が、盲点 3 つを本体の解釈に混入する
前に捕まえ、最後に対照自身が新しい事実を運んだ。

## 4. 残高への効き — 「CP を作る窓」の地図

| 窓 | 深さ (lnZ₉) | CP (J) | 判定 |
|---|---|---|---|
| 長方形 T²×T² (Dirac+局在+Hadamard, v10.1) | −19.86 (S₃ 対) | **0 (構造零, v15.7)** | CP 失格 |
| 因子化スカラー T⁴ 格子 (本版の陽性対照) | — | 1.6e-4 (格子スケール) | 対照 (資格外) |
| 傾き T⁴, Pf=3 素数 (v13.2/本版) | −34.88 | **≤ 2.6e-8 (格子スケールの 1/6000)** | 深さ・CP とも失格 |
| 非素数 Pf (複数軌道) / 複素構造 τ / オービフォールドセクター | 未走査 | 未走査 | **次の的** |

CP の要求は「非因子化なモード」では**足りない**ことが判った — より深い構造
(世代を複数の並進軌道に分ける・トーラスの形の位相 τ・セクター射影) が必要である。
選択原理の探索空間 (v15.5 で 5 候補が落ちた) は、観測 (J≠0) によってさらに
絞られ続けている。

## 5. 次の的

- **複素構造 τ**: 非直交トーラス (格子ベクトルの傾き — 磁束の傾きとは別物) は
  湯川の theta 関数に本物の複素モジュライ位相を入れる — CP の最短候補。
  連続パラメータの追加になるため、Occam・深さ・CP の三つ巴を v6.5 の枠で検定する。
- 非素数 Pf (4 = 2+2 軌道など) での J 測定 — 「軌道が割れれば CP が立つ」の直接検定
  (ただし世代数が 3 でなくなるため、資格は機構の理解に限る)。
- 機構の基底不変な診断の設計 (縮退基底に依らない「位相因子化」の定式化)。
