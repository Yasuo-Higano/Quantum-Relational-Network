# 量子情報網理論 v24.5 — 離散化普遍性 I: 1+1D では BW が UV 層から成立する

**第二十五期の本丸 III。判定 (a) — 両離散化とも λ → 1**。同一の連続理論
(massless 1+1D Dirac 1 種, v = 1) の 2 つの離散化に同一の符号付き射影推定器
(v24.3/24.4 と同一規約) を適用し、「格子依存」と「連続 QFT 依存」を分離した。

## 1. 測定 (DD + クランプ 2 点梯子)

- **XX 鎖** (naive, ノード k = ±π/2), N = 128/256/512, 解析 sine モード経路
  (dense-DD と C_A が 2e-30 で一致 [D1])
- **Wilson 鎖** (m=0, r=1, ノード k=0, doubler は +2r), N = 96/160/256,
  実表現 σ_y 規約の dense-DD 経路 (反周期解析スペクトル ±2|sin(k/2)| と
  1.6e-14 一致 [D2a]・カイラル ± 対称 2e-14・開放 gap ≈ π/N [D2b])

結果 (N 最大, 信頼窓):

| ξ | 1 | 2 | 3 | 4 | 5 | 6 |
|---|---|---|---|---|---|---|
| λ_XX | 1.00000 | 1.00002 | 1.00004 | 1.00006 | 1.00010 | 1.00015 |
| λ_Wilson | 1.00002 | 1.00006 | 1.00015 | (1.0005±7e-4) | — | — |

**λ_UV(ξ≤3): XX = 1.00002 / Wilson = 1.00008。窓端: 両者 1.00015。**

## 2. 帰結 — 監査の三角測量が閉じた

1. **推定器は健全**: 正しく Lorentz 不変な連続極限を持つ系では、この推定器は
   UV 境界層 (ξ=1!) からすでに λ = 1 を 4–5 桁で返す。BW は 1+1D 格子で
   「窓を待たずに」成立する。
2. よって **3+1D staggered の λ_x = 1.18547 / λ_y,z = 1.22943 (v24.3–24.4) は
   3+1D staggered 離散化に固有の規格化構造** — BW 定理の破れでも推定器の
   欠陥でもなく、regulator の性質。PROMPT/6 の判定表「離散化を変えると値が
   変わる → regulator artifact」が 1+1D↔3+1D 比較で確定した
   (残る確認は 3+1D Wilson の直接対照 — 次期課題として登録)。
3. v23.3–v24.0 の「格子エンタングルメント温度」概念は最終的に棄却され、
   λ は「離散化・方向・次元に依存する NN 演算子規格化」に再分類される。

## 3. 器械知識

- Wilson (2 成分・領域 128 サイト) の DD 信頼域は ξ ≤ 3 — κ 床は成分数と
  領域サイズに応じて狭まる (開発記録: 初版ゲート ξ≥4 は 3D の較正の流用で誤発報)。
- 1+1D XX の解析 C 経路 (sine モード直和) は dense 対角化なしで N=512 を数秒 —
  1+1D 対照実験の標準器械として登録。
