## 1. 結果 — 判定 (c)、ただし窓分解が本当の発見

| 窓 | Δ_flat2D (binned) | Δ_gauss (二次傾き) | \|差\| | QQ-RMS | 判定 |
|---|---|---|---|---|---|
| **(36; 1,3) 非対称** | +3.18 | **+3.09** | **0.10 ✓** | **0.25 ✓** | **(a) 級 — 解析形成立** |
| (36; 3,3) 対角 | +2.05 | **−351** | 353 ✗ | 52.3 ✗ | 尾がモーメントを破壊 |
| (24; 3,3) 対角 (τ_im=2/3) | −0.43 | −318 | 318 ✗ | 53.5 ✗ | 同上 |

- **非対称 (1,3) では解析形が完全に立つ**: 深さ周辺分布はほぼ Gauss (QQ-RMS 0.25)、
  モーメント構成の二次傾き w = ½(x−μ)ᵀΣ⁻¹(x−μ) は binned 平坦化の証拠 +3.18 を
  0.10 nats 差で再現。**bin 規約なし・フィット自由度ゼロ・データ非参照** — S1 が
  この窓では完全に充足される。
- **対角では標本分散が尾に支配され (σ_d ~ 50)、二次傾きは深い尾へ質量を送り込んで
  自滅する (−351)** — v16.12 の hard-selection 壊滅と同じ機構が、モーメント推定の
  レベルで再現した。対角族の密度は**モーメントが実効的に定義できない重い尾**を持つ。
- つまり測度問題の核心構造はこうなった: **「測度補正クラスが機能する」ことと
  「その解析形が書ける」ことは幾何族で分かれる**。非対称族 = モーメント適合
  (Gauss 近似可)、対角族 = 尾支配 (有界重みの平坦化だけが機能し、非有界な
  傾き・二次形式は全て尾に殺される)。v16.11 の「対称の壊滅」・v16.12 の
  「hard selection の壊滅」・本版の「モーメントの壊滅」は同じ一つの事実 —
  **対角幾何の配位空間には観測より遥かに深い異常な尾がある** — の三つの顔である。
- 興味深い符合: 測度が机上で機能する族 (非対称) は、生存原理 (v16.11–13) が
  選んだ族でもある。「原理が選ぶ幾何」と「解析形が書ける幾何」が一致している —
  偶然かは開いた問い。

## 2. 装置

- 全 3 窓キャッシュ命中 — 全実行 2 分。一様 G0 回帰 3 本 ±0.001 ✓。
- QQ-RMS は 99 分位の分布フリー比較 (Beasley–Springer–Moro 逆正規近似)。
  二次傾きのパラメータは標本モーメントのみ (フィットなし)。
- (36;1,3) の局所勾配予言 −1/σ_d ≈ −0.57 は v16.13 の pooled 測定 −0.19 と
  オーダーは整合するが窓・σ が違う (直接比較は限界に記す)。

## 3. 残高への効き

- **MSR-FLATTEN の bin 規約は「非対称窓では」消去可能** — measures.yml に
  MSR-GAUSSFLAT を**資格条件つき** (QQ < 0.6 のモーメント適合窓のみ) で追記した。
  対角窓の解析形は未達のまま (ロバスト・モーメント [中央値/MAD]・切り詰め尾の
  変種が次の候補)。
- 対角族の重い尾は、測度の解析形にとっての障害であると同時に、**なぜ非対称が
  選ばれるのかの構造的説明の候補**になった (尾の異常さ自体が族を特徴づける)。
- PROMPT/3 v18.2 は部分返済 — 「密度の解析形」は族に依存するという答え。
