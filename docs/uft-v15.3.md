## 1. 模型と拘束の解法

**1+1D Z₂ 格子ゲージ + staggered フェルミオン (環, L=14)**:

```text
H = −w Σ (c†_n σ^z_{n,n+1} c_{n+1} + h.c.) + m Σ (−1)^n c†_n c_n − h Σ σ^x_{n,n+1}
Gauss 拘束: G_n = σ^x_{n−1,n} σ^x_{n,n+1} (−1)^{q_n} = +1
q_n = 占有 XOR (n 奇数) XOR 外部電荷   (staggered 真空 = 電荷 0)
```

拘束は**基底の構造として厳密に解く**: 電場は E_{n,n+1} = ε·(−1)^{P_n} (P_n = Σ_{k≤n} q_k) と
電荷から一意に決まり、残る自由度は (占有 bitmask, 巻き付き電場 ε=±1) のみ。
環の整合性から**全 Z₂ 電荷が奇数のセクターは状態が構成できない** — 超選択則が
try_new の Err として実装に現れる。ゲージ対称性はパラメータではなく Hilbert 空間の形である。

**トレイト層 (lib.rs)**: `QrnStateV2` (rdm / density / amp) + `QrnDynamicsV2` (step / conserved) +
トレイト越しの共通読み出し `v2_entropy` / `v2_mi_matrix` / `v2_density_profile`。
陰性対照 (ランダム状態) も**同じトレイト・同じ読み出し関数**を通る。

| h=0 = 自由フェルミオン (磁束 2 セクター、JW 巻き付き符号 (−1)^{N_f−1} 込み) | E₀ 一致 1e-12 |

最初の設計ミスも記録する: 素朴な期待「w=0 の弦コスト = 2h·r」は**偽**だった。

min(2h·r, 2m) に直した。バグではなく物理 (string breaking) が検査を訂正した。

| 読み出し | 結果 |
|---|---|
| **幾何** | MI→MDS 隣接復元 100% (自由対照 m=0.2 も 100%)。対照: シャッフル 29% / ラベル置換 0% / ランダム 14% |
| **エントロピー** | S(ℓ) = 0.655 → 0.677 → 0.679 (ℓ=2,4,7) — 閉じ込め相の面積則的飽和 |

**握手 (読み出し間の整合)**: h を上げると σ が伸び (静的閉じ込め)、v が落ち (動的伝播)、
ギャップが開き (Δ: 0.08 → 2.13)、S 成長が止まる — 4 つの独立な読み出しが同じ「閉じ込め」を指す。
v6.7 の握手 (前線 = 2 sin k_F) の相互作用版である。

## 4. 方法と正直な注記

- 前線推定器 (サイトごとのピーク 50% 初到達) が測るのは前線**ピーク**の速度で、
  自由双極子では群速度の重み平均 4w/π ≈ 1.27 に対応する。LR 光円錐速度 2w 自体は
  L=14 では前線幅 ~t^{1/3} に埋もれて局所推定できない — 検査は (i) LR 上界 (厳密不等式)、
  (ii) 4w/π 帯、(iii) 単調減少で行った。絶対閾値は裾を拾い v=3.2 を出す (開発記録)。
- 厳密半充填・質量ゼロの自由鎖は 2k_F = π 振動で偶数距離の相関が消え、サイト単位 MI が
  偶奇縞になる (v0.7 が 2 サイトブロックで読んでいた理由)。自由対照は m=0.2 で取った。
- ゲージ理論の部分系エントロピーは中心選択に依存する。本実装の rdm は (占有, ε) 基底で
  ε を環境に置く選択 — ASM-Z2-MINIMAL に登録し、選択依存性の定量化は将来課題。
- MDS の副次規準 (半径ばらつき・λ2/λ1) は L≈200 較正で、L=14 の急峻な MI 減衰では
  歪む (相互作用 GS: rsd 26%)。判定は一次規準 (隣接復元) + 対照との弁別で行った。

## 5. 残高への効き

- **ASM-GAUSS (影響閉包 27) の返済が始まった**: core テーゼは「ガウス系の成功例」から
  「拘束つき相互作用系でも成立」へ。ただし本版は Z₂・L=14 の一点 — 適用域の地図は
  連続極限監査 (v15.4, FAL-CONTINUUM) と群の拡大が描く。
- 閉じ込めの 4 面 (σ・v・Δ・S 成長) が同じ core の読み出しとして整合したことは、
  「力」と「因果」の読み出しが独立でない (同じ網の異なる顔) という中心テーゼの
  相互作用系での最初の実演。
