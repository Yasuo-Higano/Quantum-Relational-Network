# 量子情報網理論 v22.6 — K_boost 離散化の変種比較: 重み割り当ては原因でない

**第二十三期 (PROMPT/4 並列研究 1)。事前登録 (b)** — ブースト重みの単純な離散化
変種はどれも R₂ 不足を改善しない。v22.5 で唯一残った説明系「K_boost 離散化」の
うち、**重み割り当ての自由度が消えた**。残るのは演算子形式そのもの
(rank-2 波束回転と真の modular flow の重なり = v19.6 の壁近傍格子非局所性系) だけ。

## 1. 事前登録と結果

生存窓 4 点 (N=12 × d ∈ {0.5, 1.0, 1.5} + N=14 × d=1.0)、変種:
[A] 現行 (v19.2 凍結: 右楔・x ボンド中点 ξ・y/z サイト ξ) / [B] 両側反対称
(左楔を負重みで含める) / [C] セル平均 (y/z の ξ を staggered 2 セル中心へ)。
登録: (a) ある変種が 4 点全て |R₂−1| ≤ 0.05 / (a′) N 間差が [A] の半分以下 /
(b) 改善なし。**結果: (b)**。

| 変種 | max\|R₂−1\| | N 間差 (d=1.0) |
|---|---|---|
| A (現行) | 0.207 | 0.098 |
| B (両側) | 1344.6 | 1361.5 |
| C (セル平均) | 2.49 | 2.86 |

- [B] の発散は設計的教訓: 左楔の負重みが δ⟨K⟩ をほぼ相殺し分母が消える —
  **楔の第一法則は右楔生成子の専有事項** (全空間ブーストでは測れない)。
- [C] の悪化: 横ボンドの ξ をセル中心にスナップすると壁からの距離情報が粗くなり
  一様に劣化 — 現行のサイト ξ が 3 案中最良。
- 副次確認: 浅域の δS, δ⟨K⟩ は α に線形 (v22.7 run1 の診断) — **浅域 R₂ 不足
  (0.79–0.90) は α 汚染ではなく本物**。

## 2. 帰結

R₂ 不足の説明系の消去は: 横質量 (v22.2) → d/ξ 崩壊・スラブ (v22.5) →
**重み割り当て (本版)**。残存仮説は「Gauss 対 rank-2 回転が真の modular flow を
O(1) で取り逃す」(演算子レベル、v19.6 の三角測量の第 1 候補) — これは重みの
再調整では直らず、流れの生成子そのものの改良 (modular 応答の直接測定など) が要る。
