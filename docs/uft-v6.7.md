## 0. 動機 — toy model の集まりから、一つの core へ

これまでの 32 本は「個別の toy model の集まり」だった。統一理論としての説得力は
新しいシミュレーションを増やすことではなく、**既存のシミュレーションを同じ core から
出すこと**で上がる (改良方針 §7)。v6.7 は最小の第一歩として、共通の状態空間と
動力学を `lib.rs` に定義した:

```
QrnState  = ガウスフェルミオン網 (相関行列 C_ij = ⟨c†_i c_j⟩)
QrnModel  = { assumptions(), claims(), init(), evolve(t) }   ← 仮定と台帳 id を型で明示
読み出し   = readout_entropy / readout_mi_blocks / readout_ring_geometry / readout_fermi_momentum
```

## 1. 実験 — RingChain の同一状態からの 4 読み出し

具体模型 `RingChain` (円環自由フェルミオン・半充填基底状態、N=202) の**一つの状態**から:

| 読み出し | 結果 | 照合 |
|---|---|---|
| [G] 幾何 (MI→MDS) | 円環: 隣接復元 100%・半径ばらつき 0.0% | v0.7 の再現 (v6.4 の判定規準) |
| [S] エントロピー | S(ℓ) 対数則 → **c = 1.001** | Dirac CFT の理論値 1 |
| [M] 物質 (密度→k_F) | k_F = π⟨n⟩ = π/2 (厳密) → v_F = 2 sin k_F = 2 | ラッティンジャーの定理の顔 |
| [C] 因果 (局所クエンチ) | ボンドエネルギー前線の速度 **2.10** | 理論値 v_max = 2 (v1.1 と同型) |

**読み出し間の握手**: 因果の前線速度 (2.10) ≈ 物質読み出しの v_F = 2 sin(k_F) = 2.00。
幾何・物質・因果のどれも公理には入っていない — 全てが一つの相関行列と一つの
ユニタリー発展の**異なる読み出し**であり、しかも読み出し同士が整合する。
これが QRN の中心テーゼ「時空・力・物質は同じ網の異なる読み出し」の最小の実演である。

## 2. core が早速バグを 2 つ発見した (正誤記録)

共有 core には健全性検査 ‖C²−C‖ = 0 (純粋状態は射影子) を置いた。これが即座に:

1. **v6.3 の占有数の 1 ずれ** (nocc = N/2+1、正しくは閉殻の N/2) を発見。
   v6.3 の数値を修正・更新した (基本ギャップは全 N で正になり、結論はむしろ強化。
   docs/uft-v6.3.md に正誤記録)。
2. **「密度は伝播しない」という物理**を思い出させた: 二部格子+半充填のカイラル対称性
   により局所密度はクエンチで厳密に不変 (Δρ ≡ 0 を数値確認)。因果の読み出しは
   保護されないボンドエネルギーで行う必要がある — 対称性で保護された観測量は
   光円錐を運ばない、という教訓ごと記録する。

「個別バイナリのコピペ実装では見つからなかったものが、共有 core の 1 行の検査で
見つかる」— これが §7 の主張 (core 化の価値) の即物的な実証になった。
