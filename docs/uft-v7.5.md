## 1. 定理

v6.8 は v3.1 域 (小表現) を形式化した。本バージョンは領域を v5.2/v6.2-R3 に拡大する:
SU(3) 表現 ≤ 8 (六重項・随伴込み)、SU(2) ≤ 3 (三重項込み)、|6Y| ≤ 9、多重項 ≤ 5、
成分 ≤ 16。群論係数 A(6)=7, A(8)=0, 2T(6)=5, 2T(8)=6, 2T(3_w)=4 (Slansky) を
整数データとして形式化に追加。

| 定理 | 内容 | 検証手段 |
|---|---|---|
| `sm_is_solution` | SM は拡張域でも全条件を満たす | `decide` (核のみ) |
| `sextet_pair_fails` | 六重項を含む素朴な置換は SU(3)³ で落ちる | `decide` (核のみ) |
| `sm_unique_with_big_reps` | **拡張域の全列挙で解は SM の同値軌道 4 通りのみ** | `native_decide` |

毒値設計 (軌道外の解 +10⁶・燃料切れ +10⁹) は v6.8 と同一 — 定理 `scanResult = 4` の
成立が列挙の完全性まで含意する。

## 2. 意味

「SM の孤立性は表現の拡大に頑健」(v5.2) が、Rust の三重実装 (v6.2) に加えて
**定理証明器の検証済み定理**になった。これで形式化済みの領域は:

| 領域 | Rust | Lean |
|---|---|---|
| v3.1 域 (小表現, ≤15 成分) | 三重実装 (v3.1/v6.2 MITM/DFS) | ✓ v6.8 |
| v5.2 域 (大表現, ≤16 成分) | 二重実装 (v5.2/v6.2 R3) | **✓ 本版** |
| v4.3 域 (≤6 多重項, ≤24 成分) | 二重実装 | 未 (列挙が大きい — 残高) |
