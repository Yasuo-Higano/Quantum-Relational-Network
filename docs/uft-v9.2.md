## 1. 発見の経緯 — 独立実装が仕事をした

論文図 (v9.3 予定) の作成にあたり、CLAUDE.md の新規則「人間向け資料は python 可」に
従って湯川パイプラインを numpy に独立再実装し、**図中の全数値を一次ソースと照合する**
検証を図スクリプトに内蔵した。その照合が鳴った:

- 単一トーラスの証拠 lnZ = −53.77、v7.2 の MAP 予測 (3 セクター全て) — **厳密一致**
- 積模型 T²×T² の証拠 — **−18.76 vs 公表 −20.41 で不一致** (1.65 nats)

部品が全て一致するのに合成だけずれる。切り分けの結果、原因は物理でも実装バグでも
なく、**世代ラベルの規約**だった。

## 2. 機構 — wrap 境界の ±ε で世代ラベルが巡回する

磁束トーラスの局在ゼロモードの中心は厳密に 0.5 サイト格子上にあり (最大偏差 4×10⁻¹²)、
**k=0 の 1 モードは中心が格子座標 0 ≡ N の wrap 境界上に厳密に乗る** (計算値の境界距離
1.2×10⁻¹²)。v7.2 以来の「中心の昇順ソート」はこの ±ε の丸めの符号で順序が巡回する:

- 丸めが 0⁺ 側 → 境界モードは**先頭** (安定側と同じ順序)
- 丸めが N⁻ 側 → 境界モードは**末尾** (本リポジトリの Rust が偶々落ちていた側)

単一トーラスの特異値はラベル置換に厳密不変なので v7.2 [2][3] は影響を受けない。
しかし**積模型の対角世代対 (i,i) は 2 枚のトーラスのラベルを跨いで対を組む**ため、
どちらの ε 側かで異なる行列集合が生成され、lnZ が変わる。

## 3. 証明 — 公表値 10 個の厳密再現

「境界モードを末尾へ回す」ラベルを明示的に構成して評価すると:

| 検査 | 結果 |
|---|---|
| 質量のみ lnZ 6 幾何 (v8.1: −53.77/−53.76/−20.41/−20.42/−17.41/−18.36) | 全て ±0.02 で再現 [PASS×6] |
| 全 9 量 lnZ₉ 4 幾何 (v9.1: −63.57/−27.13/−27.35/−25.56) | 全て ±0.02 で再現 [PASS×4] |

numpy 側でも同じ置換で v7.2 の MAP 予測 (u: 2.91e-5/5.30e-2, d: 1.20e-3/2.62e-2,
e: 3.21e-4/5.49e-2) が 3 桁一致した。**公表値は「ある特定のラベル規約の値」として
完全に再現可能**であり、乱れた計算ではない — ただし規約が暗黙かつ不安定だった。

## 5. 判定 — 結論は頑健、絶対値には正誤表

- **勝者は両規約で T³×Z₆** (質量のみ・全 9 量とも)。**順位全体も不変** [PASS]。
  - T³ の T² への差: 質量のみ +3.00 → +2.27、全 9 量 +1.57 → **+1.77** (むしろ拡大)。
- v7.2 の「アナーキー上界 +15.0」は安定側では +16.6 に**強まる** (−18.76 vs ≤−35.4)。
- 安定側 T³ の MAP は公表側より観測に近い: m_u/m_t=1.58e-5 (実測 1.3e-5)、
  |V_us|=0.120 (実測 0.225; 公表側 MAP は 0.052 だった)。
- **正誤表**: v7.2/v8.1/v9.1 の積模型 lnZ の絶対値は「境界モード末尾」規約の条件付き値
  である (claims.yml の QRN-YUK-003/004/005 の limitations に追記済み)。選択の結論
  (T³ 勝ち・細分化は Occam 罰負け・緊張の解消) は全て両規約で成立。

## 6. 教訓と新規約

> **「固定シードで再現可能」は「規約に依らず再現可能」を意味しない。**
> 厳密縮退帯のソート順は下流で物理量になり得る。

- 以後の湯川系バイナリは**安定ラベル** (中心を 0.5 サイト格子にスナップ → 剰余 N で
  0 に正規化 → 昇順) を規約とする (CLAUDE.md の落とし穴にも追記)。
- 改良方針 (PROMPT/1) の「独立実装によるクロスチェック」がまさに設計どおりに機能した
  実例: 三重実装で守られていたアノマリー探索と違い、湯川側は単独実装だった。
  図の照合検査という「軽い独立実装」でも装置バグ級の発見に足りた。

## 7. 深い含意 — 対角世代対は規約ではなく模型の内容

ラベル 2 規約 = 対角対の組み方 2 通り = **わずかに異なる 2 つの模型**であり、その差
(1〜4 nats) は幾何選択の証拠差と同じ桁である。v9.0 残高 2 の「対角世代対の導出」は
この発見で先鋭化した: 対の組み方は幾何 (どのモードがどのモードと同じ場所に住むか) から
導出されるべき量で、次の的である。選択肢: (a) 対の全置換を marginalize した証拠、
(b) 磁束の整合条件から対を導く M3 級の構成。

## 8. 限界

- 安定ラベルも「一つの規約」である (§7)。ただし ±ε に不変で、プラットフォーム・
  コンパイラに依らず再現する — 公表値の脆さは除去された。
- 走査は {T¹,T²,T³}×{Z₆,Z₁₂}・N=18・Q=3 の窓のまま。全 9 量の T³×Z₁₂ は引き続き対象外。
