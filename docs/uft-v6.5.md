## 0. 問題 — v3.2 は M1 級だった

v3.2 (QRN-YUK-001, C4) は文献の FN 電荷で 9 量を実測の 5 倍以内に再現した。しかし
それは「合う電荷は稀なのか、この装置は何でも合わせられるのか」に答えていない。
アドバイス §6 の分類でいえば v3.2 は M1 (電荷は選択されたもの)。本バージョンは
模型比較を実装する:

- **M0**: アナーキー — 全湯川成分が O(1) (|c|∈[1/3,3] 対数一様・位相一様)、構造なし
- **M1**: FN — 電荷は自由 (各セクター q1≥q2≥0≤4・第 3 世代 0 の **15⁵ = 759,375 通り**に一様事前)
- **M2**: FN — v3.2 の固定電荷 (文献の標準割当て)

## 1. 方法

- **尤度**: 学習量 = 6 つの質量比の対数に幅 σ=ln2 (理論許容 ×2) の正規分布。
  **CKM 3 量は学習に使わず**、事後の予測として検証する (学習/予測の分離)。
- **周辺化**: O(1) 係数はモンテカルロ (n=20000, 固定シード, 共通乱数) で積分。
- **鍵となる構造**: 学習量はセクター (u,d,e) ごとに独立な (電荷, 係数) にしか依存しない
  ので、**証拠はセクター証拠表 (15×15) の積に厳密に因子化**する。全 759,375 割当ての
  証拠が約 1.5 秒で求まり、シード間の再現性は |ΔlnZ(M1)| = 0.00。
- **3×3 エルミート固有値は閉形式** (Jacobi との最大相対差 1.6e-15 を毎回検査)。

### M0 の証拠は素朴 MC では推定できない (正直な方法論)

アナーキーが m_u/m_t = 1.3e-5 を当てる確率は事前分布の裾の彼方にあり、素朴な MC は
**下方バイアスが強い** (実測: シード間で lnZ が −93.7 / −78.3 と 15 もずれる)。
そこで点推定を捨て、**保守的な上界**で置き換えた:
Z ≤ peak·E[N₁(ln r₁)] を大標本 (2×10⁷) の分位点で分割し、統計のある領域は MC+3σ、
未踏の裾は二項 +3σ で押さえる。得られた上界 lnZ(M0) ≤ −35.4 により、
ベイズ因子は**下界**として報告できる。

## 2. 結果

| 模型 | lnZ | 備考 |
|---|---|---|
| M0 アナーキー | **≤ −35.4** (上界) | 素朴 MC: −94/−78 (下方バイアスの実例として記録) |
| M1 FN 電荷自由 | **−12.19** | 2 シードで完全一致 |
| M2 v3.2 電荷 | **−7.61** | |
| M1 の MAP 電荷 | −6.42 | q_Q=(4,2) q_u=(4,2) q_d=(0,0) q_L=(3,0) q_e=(2,2) |

**ベイズ因子 (保守的下界)**: ln B(M1/M0) ≥ **+23.2**、ln B(M2/M0) ≥ **+27.8**。
M1 は 759,375 通りの事前希釈 (Occam 罰 −13.5 nats) を払ってなおアナーキーに圧勝する。

**q\* (v3.2 電荷) の位置**: 全 759,375 割当て中、上位 **0.11%**。
「選んだ電荷で合った」のではなく「合う電荷は千に一つで、文献の電荷はその中にいる」。

## 3. 較正 — この装置は何でも FN と言わない

| 較正 | 結果 |
|---|---|
| (a) M0 から生成した合成データ | ln B(M1/M0) = **−9.5** — 階層のないデータでは FN は Occam 罰で負ける |
| (b) ランダム電荷 20 組 | q\* が **20/20** に勝つ |
| (c) ε=1 の退化性 | 全 225 電荷対の証拠が**厳密に一致** (幅 0.0) — FN はアナーキーに正しく退化 |

## 4. 学習/予測の分離 — CKM は out-of-sample で 3/3

質量比 6 つだけで評価した M2 が、学習に使っていない CKM 3 量を予測:
|V_us| 比 1.41、|V_cb| 比 1.51、|V_ub| 比 4.31 — **全て実測の 5 倍以内** (9 量では 9/9、
v3.2 の結果を学習/予測分離の下で再確認)。

## 5. AIC/BIC (参考値)

係数周辺化済みの擬似プロファイル (n=6, M1 の電荷 10 個を自由パラメータと数える):
AIC は M0 ≥ 70.8 / M1 32.8 / M2 15.2 — どの罰則の流儀でも FN が勝つ。

## 6. 限界 (claims.yml: QRN-YUK-002)

- M2 は「文献の標準電荷」であり **QRN の幾何から導いた電荷ではない** — アドバイスの
  M2/M3 (QRN 由来の電荷) への格上げは残高のまま (v6.0 残高 2)。
- σ=ln2 は理論許容の設計選択。電荷空間の上限 (q≤4, 第 3 世代 0) も設計選択であり、
  certificates 級の機械可読化は将来課題。
- ε=0.22 は外部入力 (カビボ角)。ε 自体の導出は未解決。
