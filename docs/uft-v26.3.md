## 3. 発見 — χ_00 の q⁴ 保護と、その機構

事前登録の予言「保存則ゆえ χ_00 ∝ q²」は**外れた**: 実測スロープ p = 3.990 (≈ 4)。
c₁^00 ≈ 0 と独立に整合する。機構も特定できた:

> T_00(q) ≈ H + iq X_E + O(q²) (X_E = エネルギー双極子)。
> ⟨m|X_E|n⟩ = −i⟨m|J^E_tot|n⟩/(E_m−E_n)。**自由場では全エネルギー流 J^E_tot が
> 運動量だけの関数 = 保存量**なので占有→非占有の行列要素が消え、M は O(q²) から
> 始まる → χ_00 ∝ q⁴。

帰結 (反証可能な登録予言): **相互作用を入れると J^E_tot は保存でなくなり、χ_00 は
q² に戻る (c₁^00 ≠ 0 が生成される) はず** — 相互作用系へ進むときの最初の検査項目。
ゲートは保存則が保証する下界 (p > 1.8) に貼り直した (開発記録)。

## 5. 限界と次 (v26.4+ の登録課題)

- c₁^xx は q ∥ x̂ の 1 方向・T_xx 1 チャネル — **方向依存性** (q∥ŷ; モデル再ラベルで

  log 項の係数を連続極限 (2 taste Dirac) と照合するには N をさらに要する。
