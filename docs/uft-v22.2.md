# 量子情報網理論 v22.2 — G の窓: 横質量仮説の反証と f64 床の d-局所性

**第二十三期 課題 2。事前登録 (b) 外れ — だが 2 つの確定情報を残した**:
(i) 横質量仮説の反証、(ii) f64 κ 床のカナリア死が d-局所 (N 非依存) であること。
本版で見えた「R₂ の d·gap スケーリング崩壊」の兆候 (3 点) は、
続く v22.4/v22.5 の事前登録検定で**反証**された — 顛末はそれぞれの文書に。

## 1. 位置づけ

v19.6 は G の係数窓を三角測量し、2 つの凍結量 (横質量積 m_T·ξ / 核の f64 床) を
容疑者として登録した。本版は第一容疑者 m_T を外す実験: y/z を反周期から周期に変え、
m_T = √2 sin(π/N) → 0 の massless 線を復活させる (開 x 鎖が E=0 を避けるので
閉殻は保たれる)。合わせて核の DD (double-double) 精密化を試みた。

## 2. 器械と事前登録

- 3D staggered 格子 (x 開放, y/z 周期)、N ∈ {12, 16}、A = 半空間。
- v19.4 型 Gauss 波束対の rank-2 回転 (wid=1.0, 深さ d 掃引)。
  R₁ = δS/δ⟨K_A⟩ (f64+クランプ核) をカナリア (|R₁−1| < 1% で有効)、
  R₂ = δS/δ⟨K_boost⟩ (核なしブースト重み) を本命。
- 事前登録: (a) R₂(1.5; N=12) > 0.87 ∧ d 単調 ∧ R₂(2.5; N=16) > R₂(2.5; N=12) ∧
  最終 > 0.93 = 窓開通 / (a′) 単調未達 / (b) 外れ。

## 4. 発見 1 — 横質量仮説の反証

m_T = 0 でも R₂ = 0.79255 / 0.86452 — v19.4 の反周期構成 (m_T 最大) の
0.7932 / 0.8696 と 0.1–0.6% で一致。**R₂ の 1 からの不足は横質量が原因ではない。**

## 5. 発見 2 — f64 κ 床のカナリア死は d-局所

d=2.5 でカナリアは両 N とも死ぬ: R₁ = 0.94666 (N=12) / 0.94440 (N=16) —
**N 非依存** (差 0.002)。κ = ln((1−c)/c) の f64 床 (κ ≤ 32.2 飽和) は波束の
絶対深さ d だけで噛む局所現象という機構と整合。境界の精密化は v22.4
(d=2.0 でも死: R₁ = 0.980/0.980 — やはり N 非依存)。生存窓は d ≤ 1.5。

## 6. 見かけの崩壊とその運命 (v22.4/v22.5 への導線)

本版の有効 3 点 (d·gap, R₂) = (0.121, 0.793), (0.362, 0.865), (0.277, 0.821) は
d·gap = d/ξ の一変数関数に乗るように見えた (N=16 点は N=12 補間から −2.2%)。
この崩壊仮説を v22.4 (点予言 — 床の前進で採点不能) と v22.5 (生存窓 6 点の
合併単調性 — **判定 (b) 反証**: N=12 は d 非単調 [d=1.0 で 0.898]、N=14 は
ξ=1.5 と短いのに R₂ ≈ 0.80 平坦) で検定した。**幾何仮説は 3 本とも死んだ**
(横質量 / ξ/L_A スラブ / d/ξ 崩壊) — 詳細は v22.5。

## 7. 開発記録 (v1→v3)

- **DD-Rayleigh 核の棄却** (v1): f64 固有ベクトルは準ヌル縮退クラスタ内で任意混合され、
  ベクトル毎の Rayleigh 商と κ の対応が壊れる (R = −327 で捕捉)。正しい形は
  クラスタ部分空間の DD-Gram 対角化 (未実装 — named limit)。
- **v19.1 錨の構造的不定性** (v2→v3): 拡張モード対バイアス (+5.4%) を器械同一性の
  錨にしたが、周期 y/z では Fermi 面直下/直上モードが横運動量で縮退し「拡張モード対」
  の同定自体が不定 (R_拡張 = −319.7 — DD の −327 と同値域 = 核精度によらない)。
  錨は半空間幾何専用と判明、記録に降格。有効性判定は各点 R₁ カナリアに一本化。
