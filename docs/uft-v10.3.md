# 統一場理論 v10.3 — 世代対はどんな幾何変形にも還元されない: 内部整列の発見

**Version**: v10.3
**Date**: 2026-07-04
**シミュレーション**: `sim/src/bin/v103_pairgeom.rs` → `results/v103_pairgeom.txt` (機械可読: `results/v103_pairgeom.json`)
**性格**: 分類定理 (窓内) + v10.1 §4 への正誤表

---

## 1. 問題

v10.1 の世代対 σ ∈ S₃ は抽象的な置換で、幾何的な意味が不明だった。作業仮説は
「S₃ ≅ D₃ (三角形の合同変換) = 並進 Z₃ ⋊ 反転 Z₂ なので、並進部分はトーラス 2 の
Wilson 線の細分化 (Z₆→Z₁₈)、反転部分はトーラス 2 の向きの反転 (鏡映 or 磁束 −Q =
共役) に還元されるだろう」— これを総当たりで検定した。

## 2. 方法

対状態 (kb ∈ Z₆, σ ∈ S₃) の 36 通りそれぞれについて、幾何候補
**(k′ ∈ Z₁₈) × {通常 / 鏡映 (x→−x) / 共役 (磁束 −Q)}** の 54 状態と照合する。
一致判定は「プローブ 3 種の Y¹ との Hadamard 積の特異値が、全ての行 Wilson
ka ∈ Z₆ で max|Δ| < 10⁻⁹」— 位相の任意性は落ち、実質的な物理の一致だけを見る。
Z₁₈ の 18 本の対角化 + 鏡映・共役はモード変換で構成 (対角化不要)。

## 3. 結果 — 自明な恒等以外、何も一致しない

| 対状態 | 一致した幾何候補 |
|---|---|
| σ = e (36 状態中 6) | 通常・k′ = kb のみ (自明) [PASS] |
| **σ ≠ e の 30 状態** | **なし** — Wilson 細分化・鏡映・共役の全てと不一致 [PASS] |

さらに鏡映モードと共役モードも互いに不一致 (max|Δ指紋| = 8.7×10⁻⁴) — 4 つの族
{対, Wilson 細分, 鏡映, 共役} は全て異なる物理を与える。

**物理的理由**: ゼロモードの y 方向運動量 (Landau タワー番号) は世代ラベルに固定
されている。Wilson 線は位置を平行移動しても (位置↔タワー) の結びつきを保つが、
対の置換は「どのタワー同士が束ねられるか」を変える。Higgs の有限幅 (σ_H) が y 構造を
感じるため、**同じ位置でもタワーが違えば湯川は別物**になる。

## 4. 正誤表 (v10.1 §4)

v10.1 §4 の「巡回置換はトーラス 2 の 6 サイト平行移動と等価であり Wilson 格子の
Z₆→Z₁₈ 細分化に相当する」は**誤り**。位置の一致だけから等価を推測したが、湯川は
y 構造も感じるため成立しない (本バイナリが反証)。「互換は剛体並進では実現できない」
の部分はより強い形 (どの候補でも実現できない) で成立。
**推測は検証されるまで主張ではない** — claims.yml の QRN-YUK-007 も修正した。

## 5. 解釈 — 対は「第 4 の離散量子数」

世代対 σ は位置 (Wilson) にも向き (鏡映・磁束符号) にも還元されない、
**内部整列の離散自由度**である。その正体は「磁束 (3,3) の 9 個のゼロモードから
3 つの 4D 世代をどう束ねるか」という**射影の選択**そのもの — v10.1 でデータが
この自由度を +3.7 nats で買ったことは、射影機構が観測にかかる物理であることを
意味する。M3 (完全導出) が導くべき対象がまた一つ明確になった:
(i) 磁束・Wilson 線 (位置)、(ii) 向き、に加えて (iii) **タワー整列** σ。

## 6. 限界

- 分類の候補は {Z₁₈ Wilson, x 鏡映, 複素共役} — この窓の外の実現 (例: 傾いた磁束
  T⁴ の非因子化計量、オービフォールド射影) は未検定。「既約」はこの候補窓内の主張。
- プローブは σ_H = 1 の 3 種 (十分条件: 1 つでも不一致なら不一致が確定する方向の
  検定なので、既約性の結論には影響しない)。
