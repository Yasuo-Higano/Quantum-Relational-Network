# 統一場理論への道 v0.1 — 二本柱の数値検証

**Version**: v0.1
**Date**: 2026-07-03
**Sim**: `sim/src/bin/v01_qm.rs`, `sim/src/bin/v01_gr.rs`(結果: `results/v01_*.txt`)
**位置づけ**: 統一の対象である二大理論を自前の数値エンジンで再現し、以後の「創発」検証の物差しを作る。

---

## 1. 目的

統一場理論を目指す前に、統一されるべき二本柱 — 一般相対性理論と量子力学 — を
**それぞれの言語のまま**数値的に再現する。これには二つの意味がある:

1. **検証可能性の確保**: 以後のバージョンで「時空や場が創発する」と主張するとき、創発した振る舞いを比較する基準(正解)を自分のコードで持っておく。
2. **非両立性の体感**: 二つのコードは構造がまったく違う。この違いこそが統一問題の核心である(§4)。

## 2. 量子力学(v01_qm)— Crank–Nicolson 法による 1D シュレーディンガー方程式

CN 法 `(I + iΔtH/2)ψ' = (I − iΔtH/2)ψ` は厳密にユニタリー。ノルム保存は機械精度(|1−norm| ~ 10⁻¹²)で成立。

| 実験 | 理論値 | 数値結果 | 一致 |
|---|---|---|---|
| A. 自由波束の拡散 σ(t) = σ₀√(1+(t/2σ₀²)²) | t=40 で σ=10.198 | σ=10.195 | 相対誤差 3×10⁻⁴ [PASS] |
| B. Eckart 障壁 V=1/cosh²x のトンネル(E≈0.5、古典的には全反射) | 厳密透過率の波束平均 0.1197 | 0.1194 | 相対差 0.2% [PASS] |
| C. 調和振動子コヒーレント状態 ⟨x⟩(t)=3cos t | 2周期の追跡 | 最大誤差 6×10⁻³ | [PASS] |

**確認された QM の核心**: 状態の重ね合わせ、ユニタリー時間発展、古典禁止過程(トンネル)の実在、
エーレンフェスト定理(⟨x⟩ が古典軌道に従う = 古典力学は期待値の理論として創発する)。

## 3. 一般相対性理論(v01_gr)— シュヴァルツシルト測地線

軌道方程式 `d²u/dφ² + u = 1/p + 3u²`(質点)、`d²u/dφ² + u = 3u²`(光)を RK4 積分。
単位系 G=c=1、長さ単位 GM☉/c² = 1476.6 m。

| 実験 | 数値結果 | 観測値/理論値 | 一致 |
|---|---|---|---|
| A. 水星近日点移動 | **42.979″/世紀** | 観測 42.98±0.04″/世紀 | 相対差 0.003% [PASS] |
| B. 太陽縁の光偏向 | **1.7512″** | 観測 1.75″(ニュートン論は 0.876″ で棄却) | [PASS] |
| C. 光子捕獲の臨界衝突径数 | b=5.196 (=3√3) で捕獲/脱出が分岐 | 光子球 r=3GM/c² の理論値どおり | [PASS] |

**確認された GR の核心**: 重力は力ではなく計量の幾何である。ニュートン論との差
(近日点の 43″、光偏向の 2 倍)が定量的に再現された。強重力側(光子球)も確認。

**数値実験からの教訓**(それ自体が物理): 光偏向の初回計算は理論より 5.5% 小さかった。
原因は開始点を u=10⁻¹²(≠0)に置いたことによる軌道の欠け Δφ = b·10⁻¹² × …ではなく、
b·u₀ = 4.7×10⁻⁷ rad がちょうど欠損分だった。**「無限遠」の扱いという境界条件が答えを変える**
— 重力では大域構造が局所の答えに効く。この教訓は後のホログラフィー(境界が中身を決める)の伏線となる。

## 4. 考察 — 二つのコードはなぜ統一できないか

同じ「物理の数値解」でも、二つのプログラムは根本的に異なる構造を持つ:

| | v01_qm(量子力学) | v01_gr(一般相対論) |
|---|---|---|
| 状態 | 複素ベクトル ψ(ヒルベルト空間) | 実数の軌道 (u, u')(時空上の点) |
| 時間 | 外部パラメータ t(全空間で共通) | 座標の一つ(φ で軌道を径数化; 固有時は観測者ごと) |
| 発展則 | 線形・ユニタリー | 非線形(u² 項 = 重力の自己相互作用) |
| 背景 | 固定格子(絶対空間) | 幾何そのものが動的(本来は) |
| 決定論 | 振幅は決定論的、観測は確率的 | 完全に決定論的 |
| 重ね合わせ | 本質(トンネルは重ね合わせの帰結) | 概念が存在しない(計量の重ね合わせとは?) |

**統一問題の正体**: 「時空の計量」を ψ のように重ね合わせようとすると、
(a) 時間発展の「時間」自体が重ね合わさってしまう(時間問題)、
(b) 摂動論の無限大が消せない(非繰り込み性)、
(c) 何がユニタリーに発展するのか不明になる(情報問題)。

**本プロジェクトの方針**: 二つを対等に「合体」させるのではなく、
**QM を基礎に置き、GR(時空・重力)を創発現象として導出する**方向に賭ける。
根拠は survey §4 のとおり、独立な複数のアプローチ(BH 熱力学、RT 公式、Jacobson 導出)が
すべて「幾何=情報」を指しているからである。
