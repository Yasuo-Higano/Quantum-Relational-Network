## 設計判断

1. **図も検証を持つ**: 図中の全数値は results/ の機械可読 JSON から読むか、再計算の場合は
   一次ソースの数値と照合し、不一致なら exit 1。今回の実行は 20 照合全て PASS。
   この設計自体が v9.2 の発見 (世代ラベルの綱渡り) を生んだ — 図の照合は「軽い独立実装」
   として装置バグ級の問題を検出できる。
2. **正直な可視化**: fig_attainable_ratios では観測の up 対 (m_u/m_t, m_c/m_t) が積模型の
   雲の**下に外れる**ことがそのまま見える — m_c/m_t の既知の弱点 (比 14、v7.2 から継続) を
   隠さない。fig_geometry_lnz は公表側規約の値を菱形マーカーで並置し、v9.2 の正誤表を
   図にも輸出した。
3. **ラベルは英語** (投稿用)。世代ラベルは v9.2 の安定規約。
4. 依存は matplotlib/numpy のみ (再現手順は figures/README.md)。リポジトリの Rust 部分は
   引き続き外部依存ゼロ。
