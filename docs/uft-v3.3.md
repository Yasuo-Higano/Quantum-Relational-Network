## 1. 主張

Zamolodchikov の c 定理(1986): 2d QFT の RG 流には単調減少する関数 c がある
(UV 固定点の中心電荷 → IR 固定点の中心電荷)。Casini–Huerta(2004)はこれを
**c(ℓ) = 3ℓ·dS/dℓ が強劣加法性(SSA)+ローレンツ対称性だけから単調減少**として再導出した。
RG の不可逆性は場の理論固有の性質ではなく、**量子情報の一般定理(データ処理不等式)の帰結**。

## 2. 結果(交替質量フェルミオン鎖、厳密対角化)

| 検証 | 結果 | |
|---|---|---|
| UV 固定点(m=0) | c(ℓ) = 1.000〜1.004 のプラトー(ディラック CFT の c=1) | [PASS] |
| RG 単調性(m>0 全曲線) | c(ℓ) は 1 → 0 へ全域で非増加 | [PASS] |
| 普遍スケーリング | 異なる m の曲線が x=ℓm の単一関数に収束(最大差 0.019) | [PASS] |

c(ℓ) は「スケール ℓ で見たときの有効自由度の数」— 粗視化するほど単調に失われ、
その途中経過すらミクロの詳細(m の値)によらない普遍関数に乗る。

## 3. 考察 — 三つの不可逆性はひとつ

| 不可逆性 | 検証 | 背後の定理 |
|---|---|---|
| 時間の矢(もつれの拡散) | v1.1 | ユニタリー性+局所性(再帰の実効的不可能) |
| 状態の区別可能性の減少 | v2.1 | 相対エントロピーの単調性(データ処理) |
| **RG 流(自由度の消失)** | **v3.3** | **強劣加法性 → c 定理** |

物理に現れる「一方通行」はすべて、**情報は捨てられるが作れない**という
単一の量子情報原理の顔である。QRN 公理系ではこれを「単調性原理」として v4.0 で統合する。
(3+1 次元では a 定理・F 定理が対応物 — エンタングルメント証明は部分的に完成、研究前線。)

## 4. 確立事項と仮説

c 定理・CH 証明・本計算はすべて確立/厳密。「すべての不可逆性が情報単調性に還元される」は
本プロジェクトの統合仮説(反例候補: 重力系での例外の有無が試金石)。
