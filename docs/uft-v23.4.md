# 量子情報網理論 v23.4 — λ(N) 系列: 温度繰り込みは 1/N⁴ で λ∞ ≈ 32/27 に収束する

**第二十四期。形式判定 (a) — ただしその解釈文は登録設計の不足による誤りで、
データの実体は λ∞ = 1.1852 ± 0.001 (≈ 32/27) への収束 = 有限繰り込み**。
判定の精密化 (1/N⁴ 外挿の登録) は v23.5 が行う。

## 1. 測定

DD 全経路 (部分空間 Newton 残差 < 1e-24 を全 N で検証) で K_A の x-NN 要素
プロファイル T(ξ) を測り、λ(ξ) = πξ/T(ξ) のバルク窓 (ξ ≤ 3) 平均を
N ∈ {8, 10, 12, 14, 16} で取得:

| N | λ_bulk | バルク線形性 | 端曲がり | λ_y |
|---|---|---|---|---|
| 8 | 1.19187 | 0.48% | −0.02% | 1.23026 |
| 10 | 1.19020 | 0.80% | −0.20% | 1.22914 |
| 12 | 1.18671 | 0.08% | +0.69% | 1.22949 |
| 14 | 1.18614 | 0.11% | +2.96% | 1.22938 |
| 16 | 1.18589 | **0.03%** | +10.18% | 1.22944 |

- **バルクの λ は ξ に対して定数** (N=16 で偏差 0.03% — 線形 BW 形が精密に成立)。
- **端曲がり** (開放端近傍 2–3 ボンドの λ 上昇) は N とともに成長する物理端の
  境界変形 — v23.1 で切断面に対して反証された CT 型変形は、物理端には実在する。
- **λ_y = 1.2294 が N 非依存** (横成分の規格化は別定数 — 異方性 λ_y/λ_x ≈ 1.037)。

## 2. 外挿 — 1/N⁴ 収束と 32/27

mod-4 族内の差分比: (λ(8)−λ(12))/(λ(12)−λ(16)) = 6.3 — 1/N⁴ 則の理論値 5.95 と
一致 (1/N² なら 2.86)。族別 Richardson 外挿:
- mod-0 族 (8, 12, 16): λ∞ = 1.18551
- mod-2 族 (10, 14): λ∞ = 1.1847
- **λ∞ = 1.1852 ± 0.001 vs 32/27 = 1.18519 — 0.03% 一致** (数値照合。
  2⁵/3³ という有理形は staggered 種構造由来の可能性 — 導出は理論課題・Lean 化候補)。

## 3. 開発記録 — 二つの器械教訓

- run1: 端ボンド 1 本の除外では N=16 の端曲がりが平均を汚染し [G5] 誤発報 →
  バルク窓推定器 (ξ ≤ 3) に精細化 (run1 保存)。
- **登録設計の不足 (本版の教訓)**: branch (a) の基準「|λ−1| の族内単調減少」は
  λ → 1 と λ → λ* ≠ 1 を区別できない — 単調減少は λ* > 1 への上からの収束でも
  発火する。本データは後者。**(a) の解釈文「連続極限で BW 回復」は誤り** —
  正しい言明は「BW は 2π/λ∞ の繰り込まれた温度で回復する」。判別基準
  (1/N⁴ 外挿 λ∞ の 1 との比較) は v23.5 に登録。

## 4. 帰結

格子のエンタングルメント温度は連続極限でも 2π ではなく **2π/λ∞ (λ∞ ≈ 32/27)**。
G_lattice の読み出し (v23.6) はこの繰り込みを組み込む:
G = a²·λ∞/(4·s_area·(2π 規格)) 型の条件付き読み出しが次版の本丸。
