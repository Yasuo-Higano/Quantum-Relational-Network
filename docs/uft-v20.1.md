## 1. 結果 — 判定 (a′): 質量は的中、遮蔽は器械の窓の途中

| 2 | 0.7179 |

- 連続外挿 M/g = **0.5605** + 0.2239/√x — 厳密値 1/√π = 0.5642 との差 **−0.0037 (0.7%)**、
  帯 (±0.05) の一桁内。N 依存は x≤4 で 4 桁目 (有限体積収束済み)。
- 拘束を解いた基底: (占有 bitmask, 巻き付き電場 ε ∈ ℤ)・ボンド電場 E_n = ε + 累積電荷・
  継ぎ目ホップが ε → ε∓1。**Z₂ core と同型の構造で U(1) が動いた**。
- Λ 収束は劇的: |E_n| ≤ 3 → 4 で ΔM/g = 9.6e-14 — 電場切断は実質厳密。

**遮蔽判別 (不成立 — 窓の記録)**: ±½ 外部電荷対の弦エネルギー (N=16, x=2, r ∈ {2,4,6}):

- μ=1 側 (分数電荷の線形閉じ込め) は成立。μ=0 側は登録した飽和基準に届かず — 機構は
  器械の窓: 遮蔽長 ℓ ≈ √x/(M/g) ≈ 2 サイトに対し、漸近飽和には r ≳ 5ℓ ≈ 10 が要るが

- 教訓 (v19.4 の「窓」と同族): 弦の漸近形は ED サイズの外 — 判別子は**局所観測量**
  (遮蔽の雲 = プローブ周りの誘導電荷・θ 階段 = σ(q) の周期性) で書くべき。次版 v20.2。

## 2. 装置

- 基底列挙は Gosper hack + (mask, ε) の 64-bit キー二分探索。N=18 中性セクターで
  ~34 万状態、Lanczos (完全再直交, m=150) 残差 ≤ 1e-5。全 12 質量点 108 秒。
- **[A] の設計が効いた**: リンクを陽に持つ 1296 次元表現との突き合わせが、継ぎ目
  ホップの ε 規則と電場更新符号 (∂E/∂t = −j — 電荷が右へ動くと E は −1) を
  機械精度で固定した。書いている途中で逆符号のバグを 1 つこの検証設計が捕らえている。
- 電荷超選択則 (Q ≠ 0 は probe 無しで構成不能 — dim = 0) は Z₂ core の偶数電荷則の
  U(1) 版。
