## 1. BMV 反証条件の現代化 — 2025 年の地殻変動

v1.0 の反証条件 1 は「重力誘起もつれ C>0 ⇔ 重力は量子的」という二分法だった。
これは 2017 年の提案 (Bose et al. / Marletto–Vedral) の標準的な読みであり、
「古典場は LOCC なのでもつれを媒介できない」という定理に基づく。

しかし **Aziz & Howl, "Classical theories of gravity produce entanglement",
Nature 646, 813–817 (2025)** は、物質側を場の量子論として扱うと、
**量子化されていない古典重力でも仮想物質過程を介して量子情報が伝送され、
もつれが生成されうる**ことを示した。「C>0 ⇒ 重力量子化」は言い過ぎになった。

### 何が生き残るか (再定式化)

BMV 型実験の C>0 が排除するのは「重力が**非相対論的な古典チャネル** (LOCC) である」
クラス (例: 有効な半古典フィードバック模型・一部の collapse 模型) であって、
古典重力の全てではない。したがって QRN の反証条件は次のように精密化する:

1. **QRN (A2/A3) の予言**: C > 0、かつ位相が Newton 型スケーリング
   **Δφ = G m² τ Δx²/(ħ d³)** (質量の 2 乗・時間に線形・距離の逆 3 乗) に従い、
   もつれ生成に**付随する異常デコヒーレンスがない** (ユニタリーなチャネル)。
2. **QRN が死ぬ条件**: 予測位相 (実現可能パラメータで Δφ ~ 0.05–1 rad, v2.5) において
   C ≡ 0 が確立されること。これは「幾何=もつれ」(A2/A3) の直接棄却。
3. **C > 0 でも決着しないこと**: 量子化の証明には (i) スケーリング指数の精密検証、
   (ii) Aziz–Howl 型の古典 QFT 寄与の理論的分離 (寄与の大きさはパラメータ領域依存)、
   (iii) 古典チャネル模型固有の付随ノイズの排除、が別途必要。

### 正直な自己評価

この表が示す弱点: **「QRN 固有」の数値予言はまだ乏しい**。ほとんどの行は競合理論と
共有される。固有性が最も高いのは (a) BMV の位相スケーリング+コヒーレンス保持
(A2/A3 に直結)、(b) Λ〜1/√V₄ 系の w(z) 揺らぎ帯 (ただし C3 級)。
「QRN 固有の定量予言の開発」を残高に明示的に追加する (v7.0 §残高)。

## 3. 研究ソフトウェアとしての体裁

FAIR 原則 (データだけでなくアルゴリズム・ワークフローにも適用) に沿って追加:

| ファイル | 内容 |
|---|---|
| `LICENSE` | MIT |
| `CITATION.cff` | 引用メタデータ (GitHub の Cite this repository に対応) |
| `REPRODUCIBILITY.md` | 環境・決定性 (固定シード)・成果物の種類・検証手順・誤差の考え方 |
| `CONTRIBUTING.md` | 外部クレート禁止・PASS/FAIL 内蔵・台帳追記・陰性対照の文化・定理証明器の規則 |
| `.github/workflows/ci.yml` | fmt / clippy (correctness ゲート) / ビルド / 台帳検証 / 監査スイート / 証明書の同一性 / 代表シミュレーション |
| `figures/v62_landscape.svg` | 理論空間スペクトルの図 (アドバイス §13-4) |

CI の設計判断 (正直に): `clippy -D warnings` は旧バイナリの可読性優先の記法
(`0*5+k` の行×列インデックス等) と衝突するため、**correctness 系のみをゲート**にし、
該当箇所には意図を明記した allow を付けた。フォーマットは全ファイル `cargo fmt --check`
準拠。監査系バイナリ (v6.2+) は FAIL で非ゼロ終了するため CI がそのまま検出する。
探索証明書は再実行して `git diff --exit-code certificates/` で同一性を検査する
(解集合が変わればコミットなしに CI が落ちる)。
