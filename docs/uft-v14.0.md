## 2. 第十四期の物語 — 対の物語の完結

第十一期に始まった問い「世代対 σ とは何か」が、5 期をまたいで閉じた:

1. σ は仮定だった (v7.2) → データに棄却された (v10.1) → 主結論は生き残った (v10.2)
   → どの幾何変形にも還元されない (v10.3) → 機構のはしごで測られた (v11.2–11.3)
2. 最後の対抗仮説「σ は因子化近似の影 — 指数が世代を直接数える傾き幾何が真実」は、
   道具 (Lanczos) を作って適正スケールで検定され (v12.1–v13.2)、**構造的な理由**
   (素数 3 が 2 タワーの掛け算を禁じる) で棄却された。
3. 生き残った答え: **σ はコンパクト化の既約な離散データである** —
   「9 個のゼロモードからどの 3 つを 4D 世代に束ねるか」は、位置にも向きにも傾きにも
   還元されない、模型の独立した入力であり、データはそれを感じる (+3.7 nats)。

離散データとしての σ は弦理論の離散トーションや Wilson 線のモジュライと同格の
対象である。「なぜこの σ か」は「なぜこの磁束か」と同じ種類の問いになった —
これは問いの敗北ではなく、問いの**正しい住所**が確定したということである。
