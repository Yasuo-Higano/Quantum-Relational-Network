# 統一場理論 v11.1 — 例外群走査の完結: カイラル物質の入口は E₆ だけ

**Version**: v11.1
**Date**: 2026-07-04

**対応残高**: v11.0 §3 残高 4 (理論空間の地図 — 例外群の直接走査)

## 1. 問題

v8.2 は E₆ の 27 のカイラル芯が SM 1 世代であることを厳密分解で示した。
例外群は他に G₂, F₄, E₇, E₈ がある — これらからカイラル物質は出るか。

## 2. 方法 — 引用でなく構成で

鍵は表現論の確立された事実 (C0):「G₂, F₄, E₇, E₈ の既約表現は全て自己共役
(実または擬実) であり、複素表現を持つ例外群は E₆ だけ」。自己共役表現の
フェルミオンは vectorlike に対にできるので、カイラル芯 (共役対と完全中性を
剥がした残り) は空になる。

本バージョンはこれを引用で済ませず、**物質が入り得る最低次表現と随伴のウェイト系を
整数厳密に構成して W = −W (多重集合) を機械検証**した:

| 群 | 表現 | 構成 (整数座標) |
|---|---|---|
| G₂ | 7, 14 | Σ=0 の 3 次元格子: 短ルート 6 (ノルム² 2) + 長ルート 6 (ノルム² 6) |
| F₄ | 26, 52 | 2 倍格子: 長 ±2eᵢ±2eⱼ (24) + 短 ±2eᵢ, (±1)⁴ (24) |
| E₈ | 248 | 2 倍格子: ±2eᵢ±2eⱼ (112) + 偶符号 (±1)⁸ (128) = 240 ルート + 零 8 |
| E₇ | 56, 133 | E₈ 内の実現: α=2e₇+2e₈ に直交 (126 ルート)、56 = {2β−α : β·α=4} |

構成自体の正しさは (i) 次元、(ii) ルートのノルム² (G₂ の長短比 3 を含む)、
(iii) **Weyl 鏡映での閉性** (E₈ は全 240 ルートの鏡映で 248 が閉じることまで検査) で
担保する。陰性対照は SU(3) の 3 — 同じ判定器が複素表現を正しく「非自己共役」と
判定する。

## 3. 結果 — 17 検査全て PASS

- G₂: 7 = 7̄, 14 = 14̄ ✓ / F₄: 26 = 26̄, 52 = 52̄ ✓ / E₇: 56 = 56̄ (擬実), 133 = 133̄ ✓ /
  E₈: 248 = 248̄ ✓
- 対照: SU(3) の 3 ≠ 3̄ ✓ (判定器は盲目でない)

**結論: E₆ 以外の例外群はカイラル物質を作れない。** v7.1 (2 本目の U(1) = B−L)・
v8.2 (3 本目なし、E₆ の 27 の芯 = SM 1 世代) と合わせて:

> 例外群の走査は閉じた。カイラル物質の入口は E₆ の 27 であり、その芯は SM 1 世代。
> 無矛盾性の階段 (Y → B−L → なし) と例外群の頂点 (E₆) が同じ答えを指している。

## 4. 限界

- 検査したのは各群の最低次表現と随伴 (物質の標準的な入口)。「全ての既約表現が
  自己共役」の全称部分は C0 (教科書的事実 — 実際、G₂/F₄/E₈ の基本群は自明で
  Dynkin 図に非自明な自己同型がなく、E₇ は擬実) の引用であり、機械検証したのは
  代表 7 表現。
- 「vectorlike → カイラル芯空」も C0 (質量項が書ける)。ここに独自の主張はない —
  本バージョンの寄与は走査の**完結の宣言**を検証可能な形で置くこと。
