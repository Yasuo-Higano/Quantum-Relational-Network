## 2. 第二十期の物語 — 梯子を一段ずつ

第二十期はただ一つの問いに使われた: **「エンタングルメントの熱力学から Einstein
方程式へ渡る橋は、3+1D の格子で実際に架かるか」** (PROMPT/3 の v20 課題)。
梯子は 5 段、判定は事前登録で凍結して 7 版を走らせた。

```
[✓] 第一段 δS = δ⟨K_A⟩          v19.1 (a): 4 点全て 0.1% 級
[▣] 第二段 K_A = 2πΣξT₀₀        v19.2/19.4/19.6 (b): 限界の定量化 (下記)
[✓] 第三段 Fisher 計量           v19.3 (a): 三性質 (正値・二次・単調)
[✓] 第四段 3+1D QNEC            v19.5 (a): Q_Δ = +2.08/+3.12, ε なし
[✓] 第五段 有限振幅の非線形       v19.7 (b): 不等式層は普遍・構造層は族依存
```

**立った三本の柱は全て情報側の量で書かれている** — 第一法則 (モジュラー応答)、
Fisher (S_rel の二次形式)、QNEC (エントロピー曲率 vs エネルギー流)。QNEC は
モジュラー核を一切使わない器械で立ち、核の数値的困難と独立であることを示した。

**立たなかったのは幾何の名札だけ**である。第二段 (K_A をブースト形 2πΣξT₀₀ と
読み替える = Newton 定数の読み出し口) は 3 器械の三角測量で「測定可能窓 (壁距離
d ≲ 2) では逸脱 13〜23%・窓の外はどの器械も信号を失う」と定量化された。決定的
だったのは v19.2 の N=16: 両波束対の R₂ が 5 桁一致 (0.75168/0.75167) し、逸脱が
**配置非依存のスケール不変量**であること — 固定物理形状で凍結される有限スラブ比
ξ/L_A と横質量積 m_T·ξ = √2π/16 が主因候補で、格子精細化では消えない。

第五段は結論が二層に割れた。**不等式層** (S_rel ≥ 0・入れ子単調性) は α = π/2 の
完全回転まで両摂動族で厳密に成立する — 定理の層は格子でも定理である。**構造層**
(二次近似からのずれの形) は族依存 (壁波束対: 超二次 +13%・粒子正孔: 亜二次 −52%)
で、「非線形性」に普遍な重力的意味を与えるには状態族を物理で固定する必要がある。
副産物として **1 量子の完全励起の S_rel = 5.04 nats** — v0.7 (1+1D) の「1 量子は
無限小でない」が 14 版を経て 3+1D の測定量になった。
