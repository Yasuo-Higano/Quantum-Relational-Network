## 1. 結果 — 判定 (b): 壁際は平坦 0.80、深部は比が退化

| d | R₂ | \|δS\| (α=0.02) | 読み |
|---|---|---|---|
| 0.5 | 0.8015 | 1.3e-2 | 健全 |
| 1.5 | 0.8088 | 2.7e-3 | 健全 |
| 2.5 | 3.0032 | 1.2e-4 | 形式上有効・実態は比の退化 |
| 3.5 | −0.0398 | 1.5e-5 | 同上 |
| 4.5 | −0.0010 | 2.2e-6 | 同上 |

- **壁際 (d ≤ 1.5) の R₂ は 0.801 → 0.809 とほぼ平坦** — v19.4 の Gauss 対
  (0.793 → 0.870 上昇) と**族依存の違い**。局所ブースト形の逸脱は摂動族に依存する
  = K_A − K_boost はスカラー繰り込みでなく演算子レベルの差 (非局所構造)。
- **深部の信号崩壊 (本版の主発見)**: δS は d あたり ×1/5〜1/7 で指数的に消える。
  機構は v19.4 と**別**: u (非占有内容) と w (占有内容) は大域的に直交
  (⟨u\|h\|w⟩ = 0 厳密) するため、コヒーレンス uw† のボンド応答は Fermi 波長で
  振動し相殺する。壁際では ξ 重みの勾配が相殺を破る (だから健全)。深部では
  ξ ≈ 一定 → δK_boost も δS も同時に消え、**比 R₂ は退化** (+3.0, −0.04 の散乱)。
- **事前登録の有効性判定の不備も記録**: 絶対信号床 (1e-9) と Richardson 線形性は
  この退化を捕らえない (両辺とも厳密に線形なまま小さくなる)。正しい判定は摂動の
  エネルギースケールに対する相対信号量 — 次に深部を掃く者への装置教訓。
  (分岐は登録どおり 5 点で判定 → 非単調 (b)。除外し直しても (b) は不変。)

## 2. 第二段の総括 — 3 器械の三角測量

```
v19.2 (壁密着・物理幅固定): R₂ 0.80 → 0.77 (N=8→12) — N を上げても消えない
v19.4 (Gauss 対・格子距離): 0.79 → 0.87 上昇、d ≥ 2.5 は核カナリアが除外
v19.6 (粒子・正孔・格子距離): 0.80 平坦、d ≥ 2.5 はコヒーレンス相殺で比退化
```

**測定可能窓 (d ≲ 2) では R₂ = 0.77〜0.87 (逸脱 13〜23%)、窓の外はどの器械も
届かない** — これが N ≤ 16・f64 での第二段の到達点。機構候補は 3 つ (いずれも
データと整合する仮説として記録):
1. **壁近傍の格子非局所性** (Eisler–Peschel 系) — v19.4 の上昇傾向と整合。
2. **有限スラブ幾何**: A は半無限でなく深さ L_A の板 (x=0 に物理壁) — BW 定理の
   前提 (無限半空間) が破れ、補正 O(ξ/L_A) ~ 10-25%。物理形状固定なら N 非依存の
   プラトーを予言 — **v19.2 の N=16 実測 (0.774→0.769→0.752, 減速する微減) は
   近似プラトー**: 壁近傍格子効果単独なら格子距離の増加で改善するはずが、しない —
   スケール不変成分 (本候補と候補 3) が支配する読みと整合。
3. **横方向の有効質量**: y/z 反周期境界で m_T = √2 sin(π/N) (N=12 で 0.37,
   相関長 ~2.7) — 質量そのものは BW を破らないが格子補正の窓 a ≪ ξ ≪ 1/m_T を
   狭める。
脱出路: N ≥ 24 (窓 ~5.5)・横質量ゼロの構成・多倍長核。

## 3. 残高への効き

第二段は「同定成立」でも「反証」でもなく**限界の定量化**で閉じる: 第一法則
(第一段・核ベース) は 0.1% で立つが、その K_A を局所ブースト形で読み替える
連続極限主張は N ≤ 16 では 13-23% の逸脱内に留まり、深部検定の器械は 2 系統
とも信号を失う。G の読み出し口はこの窓が開くまで保留。
