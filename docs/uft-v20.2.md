## 1. 結果 — 判定 (b): ただしコントラストは全観測量で同じ向き

| 観測量 (N=16, x=2, ±½ プローブ, r=6) | μ=0 (m=0) | μ=1 | コントラスト |
|---|---|---|---|
| 弦エネルギーの曲率 (v20.1, r≤6) | −0.061 | −0.006 | **10×** |
| 中点電場 E_mid (本版の判別子) | +0.218 | +0.419 | **1.9×** (登録 2× に 4% 届かず) |
| 誘導電荷雲 I(w=1) | −0.260 | −0.084 | 3.1× |
| 誘導電荷雲 I(w=3) | −0.332 | −0.100 | 3.3× |

- **m=0 では磁束が溶け、μ=1 では弦が磁束を運ぶ** — 向きは全観測量で一致。だが
  E_mid 比の実測 0.521 は登録 0.5 を 4% 外れ、**事前登録どおり (b)**。
- 結論: **m=0 遮蔽の量的判別は N=16 の ED 窓では閾値科学にならない**。遮蔽長
  ℓ ≈ 2 サイトに対し、漸近的コントラスト (≫2×) には中点距離 ≫ ℓ が要る —
  質的署名 (2〜10×) の記録が本版の到達点で、量的完成は連続外挿つき大規模法
  (DMRG 等 — 探索層で可) の課題として限界を明示する。
- θ 階段 σ(¼,½,¾,1) = 0.054, 0.212, 0.462, 0.652 (単調増加 — 折り返し・弦切れは
  r ≤ 6 のクロスオーバー手前): **記録**。判定は v20.3 (N=18, r≤8, 順序命題) へ。

## 2. 開発記録 — 3 走の器械較正 (各走のゲートが実際の欠陥を捕捉)

1. **run1 (r=8)**: プローブを対蹠点に置き、内外二本の弦 E = ±½ が**厳密縮退** —
   基底状態が不定になり雲の対称性が壊れた (I₀ = −0.38 vs I₈ = +0.19)。和則ゲートが
   捕捉。→ r=6 (縮退が 0.5 開く)。
2. **run2/3 (鏡映-C)**: 反対称ゲート max\|δq(n)+δq(6−n)\| < 1e-8 を「厳密」として
   登録したが、**staggered の C 変換は 1 サイト並進を伴う**ため偶サイト固定プローブ
   とは非整合 — μ=0 で 3.1e-1、μ=1 で 8.2e-2 の破れは物理 (器械バグでなく
   私の対称性の誤解をゲートが暴いた)。→ 記録に降格。
3. **窓積分 I(w) は CDW のノードに乗る** (I(1)=−0.26, I(2)=−0.16, I(3)=−0.33 の振動)
   — 窓閾値は脆い。E_mid (E = ε + 累積電荷 — CDW が累積で平滑化) へ乗り換えたが、
   それでも閾値推定が 4% 外れた。**教訓: メソスコピック窓での漸近閾値の推定は
   構造的に脆い — 判定は順序命題で書く** (v20.3 が実装)。
