## 1. 方法 — 装置を信じない三段構え

1. **fit 装置の自己検証** (合成データ): B=0 の回復 (B̂=2e-5)・B=0.5 の検出 (B̂=0.5000)・
   ノイズ 1e-4 での偽 B なし (B̂=3e-5)。
2. **厳密系列での較正**: Z2 core の h=0 は磁束 2 セクターの厳密和 — その e(L) 級数
   (L=10..46) の連続極限は解析値 −2/π。fit は B̂ = −0.636634 (|Δ|=1.4e-5)、
   p̂ = 2.02 (CFT の Casimir 補正 1/L² と整合) を返した。**装置は正しい答えを知らずに
   正しい答えを出した。**
3. **適用**: 相互作用 core (v15.3) とガウス core (v6.7) の readout 群へ。

## 3. 陰性発見 — 前線速度推定器は未収束

v̂(h=0, L) = 0.894 → 0.983 → 1.171 → 1.502 (L=10,12,14,16)。ピーク 50% 到着の推定器は
この窓で連続極限に達しておらず、LR 上界 2w に向かって上方ドリフトする (前線幅 ~t^{1/3}
と系サイズの競合)。**これは物理の破れではなく推定器の限界**であり、台帳には次の形で残る:

- 厳密に立つもの: v̂ ≤ 2w (Lieb–Robinson 上界) が全 L で成立。
- L に頑健な物理: 閉じ込め抑制比 v(1.2)/v(0) < 0.6 が全 L で成立。
- 開いたまま: v̂(L→∞) が 2w に収束するか (より大きな L は ED の外 — 将来 MPS 級の道具)。

「監査が最初に捕まえたのは物理ではなく自分の推定器だった」— これは v6.3 (QNEC の
誤差予算) や v9.2 (ラベルの綱渡り) と同じ型の、装置の正直な記録である。

## 4. 残高への効き

- **ASM-LATTICE (閉包 51) が「監査済み」になった** — 少なくとも本版が走査した
  readout 群 (エネルギー・凝縮・ギャップ・エントロピー・弦張力・幾何・中心電荷) では、
  格子を細かくしても消える残差しかない。FAL-CONTINUUM は発火せず。
- 監査の適用範囲の限界も明示: L ≤ 16 (ED)、gapped 相のみ、単一冪 fit。
  臨界点近傍 (相関長 ≫ 1) の監査は道具 (MPS/DMRG 級) を要する — 将来の残高。
