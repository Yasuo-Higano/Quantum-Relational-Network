## 2. しかし、針はあった — attainable set の存在証明

[D] 尾の中に、レプトン混合セクターを**全量ほぼ厳密に**再現する config が存在する:

(σ_B=4, \|w\|=2, 位置#1, **kν=21 = Wilson 対 (3,3)**, σν=2。**これはフィットでは
ない** — データ盲目の走査集団の中からデータで拾った展示であり、モデル主張には
使えない。事前質量は P(全量 hit | 尾)·P(尾) ≈ 4e-4 × 5.9% ≈ **2.4×10⁻⁵**。)

この存在証明が問題を書き換える:

> v18.5 の no-go は**典型集合 (帯)** の言明であり、**到達可能集合**は測定点を
> 含んでいた。問いは「幾何シーソーは大混合に届くか」(答: 届く) ではなく
> 「**事前質量を針の領域へ集める lever は何か**」である — クォークの \|V_us\| が
> 「届くが典型でない」状態から τ = 1/12 + i/2 の発見で「典型」になったのと
> **完全に同じ型**の状況にある。

針の座標 (ν の Wilson 対が (3,3) = シアー対と同じ整数対、σν=2 = [1,0,2]) は
次の lever 探索の手がかりとして記録する (偶然かは未検定)。

## 3. 残高への効き

- レプトン・アーク 4 歩の到達点: 最小棄却 → 方向確認 → no-go の輪郭 → **「届くが
  典型でない」と確定**。残る問いは lever — 候補は (i) ν セクターにも幾何レバー
  (ν 専用の τ 的自由度?)、(ii) 針の座標の周りの局所構造 (kν=(3,3) 近傍の地図)、
  (iii) 局在半径の幾何 (v18.5 の登録候補 — 典型集合ごと動かす正攻法)。
- 尾の帰属 (両因子の整列) は「lever は単一セクターでなく相関を作るもの」を示唆。
