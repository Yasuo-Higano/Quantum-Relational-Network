# 統一場理論 v11.4 — 動的テンソル分解の第一級化: 分解は入力ではなく読み出し

**Version**: v11.4
**Date**: 2026-07-04
**シミュレーション**: `sim/src/bin/v114_tensorcore.rs` → `results/v114_tensorcore.txt` (機械可読: `results/v114_tensorcore.json`)
**対応残高**: v11.0 §3 残高 7 (core 移行 — 最終項目)

---

## 1. 何をしたか

v1.3/v2.0 の追補「テンソル分解は任意ではない」を、core の読み出し
**`readout_natural_partition`** として第一級化した: サイト間相互情報量 I(i:j) を
全対で計算し、貪欲最大マッチングでサイトを対に組む。「どのテンソル分解が自然か」を
状態自身に語らせる読み出しである。

## 2. 検証 — 同じ読み出しが状態ごとに違う分解を発見する

| 状態 | 発見された分解 | 判定 |
|---|---|---|
| RingChain (n=42) 基底状態 | **空間的な隣接対** 21 組 (100%) — 局所性の再発見 (v6.4 と整合) | [PASS] |
| TfdPair (n=21×2, β=1) | **L–R の鏡像対** 21 組 (100%) — 空間でなく「橋」が分解の単位 (ER=EPR の分解版)。捕捉 MI 0.680 | [PASS] |
| GrowingChain 到着直後 | **到着ボンド対** (6,7)(8,9)(10,11) がそのまま現れる (捕捉 0.882) | [PASS] |
| 同・発展後 (t=30) | 到着対は **0/3 残存** (捕捉 0.469) — 対が溶けて組み変わった | [PASS] |
| 健全性 | 3 状態とも purity_defect < 1.4e-15 | [PASS] |

## 3. 意味

- **分解は状態が決める**: 円環では空間、熱場二重では橋、成長宇宙では到着の単位。
  そして発展はそれを組み替える — 「動的テンソル分解」の実演。
- core 移行地図はこれで**残高なく閉じた**: 4 模型 (RingChain / TfdPair /
  GrowingChain / PacketRing) + 読み出し群 (エントロピー / MI / 幾何 / 零エネルギー /
  カイラリティ / **自然な分解**)。
- v1.3 の「時空 = 量子誤り訂正」との関係: あちらは所与の符号構造の検証、こちらは
  構造を**発見する**読み出し。ガウス系での最初の一歩 (対マッチング) であり、
  ブロック (>2 サイト) への一般化は将来の拡張。

## 4. 限界

- マッチングは貪欲法 (大域最適の保証なし) で、対 (2 サイト) に限定。多体ブロックの
  分解 (QEC 符号の発見) はガウス系の外を含み将来課題。
- 「自然」の基準は MI — 他の基準 (条件付き MI・純度) との比較は未実施。
