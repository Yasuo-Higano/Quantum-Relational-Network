## 1. 確定した解析構造 (v241_dispersion, 17 検査全 PASS)

v22.2 以来の 3+1D staggered 格子 (x 開放・y/z 周期, ホップ ±1/2) について:

1. **反交換** {D_i, D_j} = 0 と **H² = ΣD_i²** — 厳密 (半整数演算で逸脱 0.0)。
2. **バルク分散 E(k) = ±√(cos²kx + cos²ky + cos²kz)** — 全周期 N=8 の dense
   スペクトルと多重集合一致 (Δ < 2e-12)。
3. **零集合 = 8 点ノード (余次元 3)** — E = 0 ⟺ cos kx = cos ky = cos kz = 0
   (平方和の厳密代数)。零モード数 8 (N=8)、ノード球外の min E = 0.31。
   **taste 計数: 8 成分 = 2 taste × 4 成分 Dirac** (3+1D staggered の標準結果)。
4. **Fermi 速度 v = 1、等方** (軸 3 方向 + 対角で 1.00000000)。
5. **x 開放でも閉形式**: H = D_x + (−1)^x M と厳密恒等式
   (−1)^x φ_n = φ_{N+1−n} (開放鎖 sine モード) により
   E = ±√(cos²(πn/(N+1)) + cos²ky + cos²kz) — dense と Δ < 2e-12 (N=8,10,12)。
6. **シェル gap 公式**: gap(N) = 2·min E。mod-4 シェル構造 (v23.x の観察) の起源
   = 横運動量 π/2 (μ=0 系列) の有無。gap(4|N) ≈ 3.1/N, gap(N≡2) ≈ 9.2/N。

## 2. 帰結 — v24.0 の主張への含意

- **v23.6 の「対数 vs 定数」二択は前提不成立**: Gioev–Klich の対数面積則
  (S ~ N²·lnN) は余次元 1 の Fermi 面の結果。本模型は点ノード (余次元 3) なので、
  期待すべきは S = αN² + 抑制された補正。正しい有限サイズ族での再裁定 = v24.6。
- **半空間 C_A は (ky ペア, kz) の N²/2 個の N×N 実対称ブロックに厳密分解** —
  dense N³ 対角化 (v23.4 の限界 N=16) を経ずに N=128 級へ (v24.2 が照合・認証)。
- 離散化間比較 (v24.5) の種数規格化: staggered = 2 Dirac / Wilson = 1 Dirac。
