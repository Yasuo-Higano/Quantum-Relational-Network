## 1. 結果 — 判定 (a)、ただし副検定は素朴機構を棄却

三つ巴 (Δ = lnZ − 一様):

| 事前 | Δ (bw=0.25 / 0.5 / 1.0) |
|---|---|
| β=1 傾き (v16.12 回帰✓) | **+3.34** |
| flat1D (深さ平坦化) | +3.13 / **+3.29** / +3.23 |
| flat2D (対数一様化) | +2.50 / +2.91 / +3.06 |

- **主判定 (bw=0.5): (a) 成立** — flat1D は β=1 と **0.05 nats** で並ぶ。全ビン幅で
  ±0.5 内、flat2D も +2.9〜+3.1 の同じ帯。**性能のレベルでは、生き残った原理の
  働きは測度補正 (浅い比の過剰代表の除去) と同一**である。
- **副検定は不整合**: クォーク対の配位密度 ln n(d) の勾配は
  **中央値 −0.19** (10–90% 帯 [−0.24, −0.005]) — β=1 = e^{−d} 密度の厳密な相殺、
  という素朴な機構は**棄却**。密度は指数型ではない (浅い側に厚い塊 + 薄い深い尾)。
- 統合読み: 傾き (非有界)・1D 平坦化 (有界)・2D 対数一様化はどれも「浅い塊の
  過剰重みを取り除く」働きを持ち、その後の形の違いに**証拠は鈍感** (+2.9〜+3.3 の
  帯)。尤度が深すぎる裾を切るので、塊さえ除けばどの形でも典型集合が観測の深さ窓に
  乗る — **生き残った内容は「測度補正」クラスであり、単一の β や単一の形ではない**。

## 2. 平坦化は対称幾何を殺さない — それでも (1,3) が首位

β=1 (非有界重み) は対称対角を −2038 級で壊滅させたが、平坦化の重みは有界
(ln(1/ビン数)) なので対角が戦線に戻る:

| 幾何 | β=1 | flat1D(.5) | flat2D(.5) |
|---|---|---|---|
| (1,3) | −18.26 | −18.43 | **−18.80 (全体首位)** |
| (2,4) | **−18.23** | **−18.25** | −19.24 |
| (2,3) | −18.67 | −18.59 | −19.07 |
| (3,3) | −2037.6 | −20.70 | **−19.71 (復帰)** |
| (2,2) | −2038.2 | −22.13 | −20.97 |

- **(1,3) は全 8 変種で谷底 4 幾何の首位** — v16.11 の作業仮説は「測度補正」クラス
  全体に対して頑健。全 21 幾何でも首位は (1,3) か (2,4) (\|Δs\|=2 帯) で不変。
- 対角の「死刑」(v16.11) は β 傾きの非有界性の産物だった — 平坦化では (3,3) は
  首位から 0.9〜2.3 nats 差の次点圏。**幾何差別の強さは原理の形に依る**が、
  向き (非対称 > 対称) はクラス全体で共通。

## 4. 残高への効き

- **QRN-SEL-003 の解釈が確定した**: 「β=1 = スケール不変測度」は性能レベルで正しい
  (事前登録 (a)) が、機構は「指数密度の相殺」ではない — 正確な言明は
  「**シアー族の配位測度は浅い質量比を過剰代表しており、それを除く任意の測度補正
  (傾き・平坦化・対数一様化) が +3 nats 級で生存する**」。原理の座は単一の
  Depth(β) から**測度補正クラス**に置き換わった。
- 選択原理アーク (v15.5 → v16.11 → v16.12 → 本版) はこれで一巡: 全滅 → Depth 生存
  → 有限 β の物語と判明 → 正体は測度補正。**「宇宙は何を選ぶか」への現在の答え:
  「比のスケールに関して素朴な一様測度は間違っており、補正はデータを見ずに
  構成できる」**。
- (1,3) 作業仮説はクラス全体で安定 — 下流 (事後帯・τ_im) の基準幾何として使える。
