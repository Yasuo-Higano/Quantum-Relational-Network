## 2. 執筆上の判断

- **独立短報** (companion 方式)。測度アークは自立した物語 (生存 → 解剖 → 正体 →
  登録 → 判定 → 溶解 → 残響) を持ち、[2] (CP 論文) の §10 とは重複より発展の関係。
- 中心命題は結語に固定: 「**測度は一様でよい。問われるべきは幾何だった。残るのは
  熱力学の弱い声**」— ただし §8 で「本計画の模型空間内の言明」と明確に限定。
- S3 の教訓を独立の一文として立てた: *a measure that raises the evidence and a
  measure that improves predictions are different objects* — この арк の最重要の
  方法論的輸出物。
