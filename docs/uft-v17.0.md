## 2. 第十七期の物語 — 裁かれる機械が、選ぶ機械を裁いた

第十六期の末尾で「非因子化窓の CP が次期の本丸」と書いた。その問いは 4 版で
答えられた (傾きでは足りない → シアーが作る → 値段の勘定 → 10 量で反転) だけで
なく、**外れの連鎖が地図を描いた**: \|V_td\| の miss (v15.7) が hit に転じ (v16.4)、
\|V_us\| の 2.5 倍過大は σ・Wilson・N・τ_re の全レバーを無罪にした果てに
「対称性の病気」と診断され (v16.9)、その診断すら証拠の分解能以下と裁かれた (v16.10)。

そこで初めて選択原理が仕事をした。証拠が決められないものを、データを見ない原理が
破る — v15.5 で全滅した候補の中から、壊滅していた Depth が新しい模型族で生存し
(v16.11)、しかしその生存は「最深の選好」ではなく (v16.12 で崩壊点を確認)、
**配位測度の補正**だった (v16.13)。期の初めに「CP を作る幾何はどれか」と問い、
期の終わりに「模型族の測度は何であるべきか」という一段深い問いに到達した —
公理の言葉で言えば、**離散データの台帳 (σ・Wilson・磁束・トーラス数) に複素構造 τ が
加わり、その上の測度が初めて経験的な対象になった**。
