## 2. 公理系への反映(v2.0 からの差分)

**A3(重力)追補**: 線形化(v0.7)に加え、非線形化に必要な二次構造 —
S_rel の正値性(エネルギー条件)・Fisher 計量の正値性(正準エネルギー=二次摂動の安定性)・
単調性(ウェッジの入れ子=バルクの半順序)— がすべて成立(v2.1)。
残る未検証はモジュラー非可換性(QNEC)と三次以上。

**A4(物質と力)大幅改訂**: 「網の励起」という一般論から、選択原理へ:
1. 内部量子数は隠れたテンソル因子の**幾何**(電荷=巻き数、量子化=トポロジー: v2.2)
2. 世代の複製は隠れ構造の**位相不変量**(縮退度=磁束数: v2.3B)
3. 電荷の値は**量子的無矛盾性**(アノマリー消去)が一意に決める(v2.3C)
4. カイラリティは離散構造への制約であり、高次元境界モードを要求(v2.3A)
> **物質の内容は恣意的ではない。「無矛盾に符号化できるものだけが存在できる」** —
> v2.0 の「冗長な記録が実在をつくる」と対をなす、v3.0 の中心テーゼ。

**A6(宇宙論)追補**: 構造の起源を接続 — 真空のもつれ(A2)→ 引き伸ばしとスクイージング(v2.4)
→ デコヒーレンス(v1.6)→ 古典密度場 → 重力崩壊(v0.1)。観測値 n_s=0.9649 は
この機構の「わずかな非 dS 性」の検出である。
