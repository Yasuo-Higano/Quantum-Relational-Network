   反転 C̃′ = diag(occ) + ũ*ũᵀ − w̃*w̃ᵀ は、固有基底の位相回転だけで発展する
   (U(t) = V e^{−iεt}ũ — O(ns²))。ns³ の行列積が消え、N=12 の 5 時刻掃引が 27 分。
2. **T_tx = エネルギー流 −dE_A/dt (中心差分)**: 運動量密度の格子規約リスクを
   エネルギー保存 (厳密・機械精度でゲート) に還元。T_tt/T_xx は面のボンドエネルギー。
3. **純粋性の補集合ゲート**: S(A) = S(Aᶜ) を λ=0 で両側計算して一致確認 (≤1e-9) —
   λ≥1 は縮む補集合側だけ評価 (計算量 1/5〜1/50)。
4. **S″ は真空差引形**: S_vac(λ) 単独は薄板・staggered 偶奇の格子系統 (N=8 で
   S″_vac = +0.76) を持つが、差分 ΔS(λ) では消える (run1 で発見)。フィットは
   a + bλ + cλ² + d(−1)^λ。
