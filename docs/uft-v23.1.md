# 量子情報網理論 v23.1 — ブースト生成子の格子形 I: NN プロファイルは線形 BW と 1% 一致

**第二十四期 開幕。事前登録 (b) — 二境界仮説の反証**。だが [P] 直測が決定的な
測定を残した: **K_A の x-NN ボンドプロファイルは線形 (Bisognano–Wichmann) と
1% 一致** — ブースト重みの「形」は正しく、R₂ 不足は K_A の**非近接構造**が担う。

## 1. 問いと方法

v22.7 で G 窓の残高は「深部まで K_A と整合するブースト生成子の格子形」1 点に
なった。本版は two-pronged で攻めた:
- **[P] 直測**: DD で厳密構成した K_A = Σ κ_k v_k v_kᵀ の x ボンド要素 (横平均)
  プロファイル T(ξ)。
- **[R] 検定**: R₂ を重み 3 種 (線形 ξ / CT 放物 ξ(2ℓ−ξ)/(2ℓ) / 実測形) で
  d ∈ {0.5..2.5} (DD カナリア有効域) を測定。
- 理論動機: 深部破綻域は波束が開放端に近づく領域と一致 → BCFT (Cardy–Tonni) の
  二境界重みなら端で重みが有限に留まる。浅域の CT 予測 (−8%/−12.5% @ d=1.0/1.5)
  は v22.5 実測 (−10%/−13.5%) と整合していた。

## 2. 結果

**[P] プロファイル (ξ = 1..5, T/T_ref):**

| ξ | 実測 | 線形形 | CT 形 |
|---|---|---|---|
| 1 | 1.0000 | 1.00 | 1.00 |
| 2 | 1.9977 | 2.00 | 1.82 |
| 3 | 2.9959 | 3.00 | 2.45 |
| 4 | 3.9382 | 4.00 | 2.91 |
| 5 | 4.9615 | 5.00 | 3.18 |

**実測は線形と 0.1–1.5% 一致・CT (放物) は棄却** — 深部で最大 56% ずれる。
浅域予測の「一致」は偶然だった (事前登録が思い込みを殺した好例)。

**[R] R₂ 検定 (カナリア R₁ は全点 1±0.02%):** CT 重みは改善せず
(d=1.5 で 0.31 に悪化、d=2.5 の符号反転も残存) → **(b)**。
実測形変種 [P] の R₂ 破綻 (−0.25 等) は重み補間の端クランプによる実装
アーティファクト (プロファイル測定自体は健全 — 開発記録)。

## 3. 帰結 — 問いの精密化

K_A の NN 成分は線形ブーストそのもの (1%)。それでも R₂ (NN 重みの応答比) は
0.79–0.90 + 深部符号反転。**よって不足は K_A の非近接要素 (Δx ≥ 2・対角・
長距離) が δ⟨K_A⟩ に寄与する分** — これは重みの再調整では直せない構造。
次版 v23.2 = δ⟨K_A⟩ の距離分解 (バケット予算測定) で担い手を直接定量化する。

## 4. 開発記録

- 実測形変種の重み関数は範囲外 ξ の端クランプで横ボンド (ξ_site = 0.5) を
  2 倍過重み付けした — 変種 [P] の R₂ は無効 (プロファイル測定は独立に有効)。
- CT 重みの浅域「一致」(§1) は 2 点の偶然 — 5 点の直測が形を裁いた。
