## 1. パズル

ホログラフィー(v0.5, v0.7)で時空内部(バルク)の 1 点の情報は境界の**どこ**にあるのか。
素朴な答えはどれも矛盾する(特定の場所にあるなら他の場所からは復元できないはずだが、
AdS/CFT では十分大きいどの境界領域からも復元できる)。
Almheiri–Dong–Harlow (2015) の答え: **辞書は量子誤り訂正符号である**。その最小模型を厳密検証した。

## 2. 結果 — 3 クォートリット完全符号

論理クォートリット(=バルクの 1 点)を 3 物理クォートリット(=境界の 3 領域)に符号化し、
参照系 R(情報の原本)と最大もつれさせる:

| 領域 | I(R:領域) | 意味 |
|---|---|---|
| 単一クォートリット ×3 | **0(−4×10⁻¹⁶)** | どの単一領域もバルクを全く知らない [PASS] |
| ペア ×3 | **2 ln 3(最大)** | どの 2 領域からも完全復元可能 [PASS] |

消失誤り耐性: 論理状態を変えても単一サイトの密度行列は**トレース距離 0** — 1 領域を
失っても情報は漏れも失われもしない [PASS ×3]。

## 3. 考察

1. **「バルクの点」は場所ではなくパターンである。** 情報はどの単一領域にもなく、
   相関(もつれ)の中に遍在する。バルク局所性は近似的・創発的な概念で、
   誤り訂正の「論理演算子の可換性」として実現される。
2. **ブラックホール地平線との対応**: 1 領域の消失 = その領域がエンタングルメント・
   ウェッジから外れること。v0.8 の島公式は「どの領域の符号に情報が入っているか」の
   計算そのものだった。
3. **QRN 公理 A0 の精密化**: 基本ヒルベルト空間のテンソル分解は任意ではなく、
   **誤り訂正符号の構造**(冗長性+復元可能性)を持つ。「なぜ時空は安定か」への答えの候補:
   誤り訂正が UV の雑音から幾何(論理自由度)を守っているから。
4. 単一領域が最大混合(S=ln3)= RT 公式の面積項。**面積=エントロピー(v0.5)と
   誤り訂正は同じ構造の二つの顔**(Harlow の再構成定理)。

## 4. 確立事項と仮説

符号の性質は厳密(有限次元線形代数)。「現実の時空がこの構造を持つ」は AdS/CFT では
定理級に確立しつつある(HaPPY 符号、Harlow 2016)が、我々の宇宙(dS)への拡張は仮説。
