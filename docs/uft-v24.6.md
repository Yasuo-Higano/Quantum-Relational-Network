## 3. G^bare の再分類 (v23.6 の再解釈)

G = a²/(4·λ·s∞) は定義式で、λ の方向依存 (v24.4) がそのまま定義依存になる:

- x スキーム (λ_x = 1.185468): **G/a² = 1.0725**
- y,z スキーム (λ_y = 1.229430): **G/a² = 1.0341**

**正名: G_entropy^bare(a; regulator, 方向スキーム)** — 方向スキームだけで
3.7% 動く bare 面積係数であり、物理的 Newton 定数ではない。v23.6 の
「G∞ ≈ 1.06a²」は λ = 32/27 (棄却済み)・x スキーム相当の一点だった。
「G ∝ a²」は s_area ≈ const からの代数的必然 (発見ではない — PROMPT/6 §1.4)。
