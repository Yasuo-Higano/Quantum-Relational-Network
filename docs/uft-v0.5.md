**Version**: v0.5

**位置づけ**: ここから「統一仮説の検証」に入る。仮説の第一の柱
**「面積とはエンタングルメント・エントロピーである」**の微視的証拠を厳密計算で得る。

---

## 1. 方法

調和格子(自由スカラー場の格子化)の基底状態はガウス状態なので、部分領域 A の
エンタングルメント・エントロピーは相関行列 X=⟨xx⟩, P=⟨pp⟩ の制限から**厳密に**計算できる
(シンプレクティック固有値 ν = √eig(X_A P_A)、モンテカルロ誤差なし)。

## 2. 結果

### A. 1D 臨界系(m=0): 対数則と中心電荷の復元

S(ℓ) = (c/6)·ln[共形長] + 定数(端に接する区間、開境界)に対しフィット:

**c = 0.974(理論値: 自由ボソン CFT の c = 1)[PASS]**

もつれのスケーリングだけから、その系を支配する連続場理論(CFT)の中心電荷が読み取れる。
**もつれは幾何(スケール)の情報を持っている。**

### B. 1D ギャップ系(m=0.5): 飽和

S(ℓ) は ℓ=8 以降 10⁻⁵ の精度で一定(0.27095)。相関長 1/m を超えるともつれは増えない。
**もつれは境界近傍にのみ宿る。**

### C. 2D: 面積則 — 本バージョンの主結果

| ブロック | 境界長 4ℓ | S |
|---|---|---|
| 2×2 | 8 | 0.414 |
| 4×4 | 16 | 0.905 |
| 8×8 | 32 | 1.900 |

**S = 0.062×(境界長) − 0.085、決定係数 R² = 0.99999 [PASS]**

エントロピーは領域の**体積 ℓ² ではなく境界 4ℓ に比例**する。
(熱力学エントロピーなら体積比例になるはず — 基底状態のもつれは根本的に「表面的」である)

## 3. 考察 — ブラックホールはなぜ「面積」なのか

1972 年 Bekenstein、1974 年 Hawking はブラックホールのエントロピーが
**S = A/4ℓ_p²(体積でなく面積!)**であることを示した。これは 50 年来
「何の自由度を数えているのか」が謎だった。

本バージョンの計算は、その最も単純な答えを支持する:

> **量子場の基底状態では、任意の領域のもつれエントロピーが自動的に面積則に従う。
> 地平線のエントロピーとは、地平線を横切る量子相関(もつれ)の量である。**

- 発散の構造も一致する: 格子間隔 a → 0 で S ~ A/a²(2+1D)。重力がこれを A/4G で
  有限化するなら、**ニュートン定数 G はもつれの UV 構造で決まる**(Sakharov の誘導重力、
  Jacobson 1995 の出発点)。
- Ryu–Takayanagi 公式(2006, AdS/CFT)は逆向きの同じ等式: 境界理論のもつれ = バルクの極小面の面積。
- 「もつれを切ると空間が千切れる」(Van Raamsdonk 2010): S→0 にすると領域間の相関が消え、
  接続された時空として記述できなくなる。

**統一仮説の用語で**: 幾何の基本量「面積」は、情報の基本量「もつれ」の別名である。
v0.7 でこれを動的に(「もつれの変化 = 幾何の変化 = 重力」)昇格させる。

## 4. 確立事項と仮説
