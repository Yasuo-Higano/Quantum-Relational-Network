## 1. 方法 — トーラスの形を格子で傾ける

シアー境界条件: y-wrap を (x, N−1) → (x+s mod N, 0) に張り替える。これは格子ベクトル
e₂ = (s, N)、すなわち**複素構造 τ = (s + iN)/N** のトーラスの離散実装である。
v12–v13 の「傾き」は**磁束** (場) の傾きだったのに対し、これは**幾何そのもの**の形 —
連続体では τ_re ≠ 0 が湯川の theta 関数に本物のモジュライ位相を入れる、CP 破れの
標準的な源である。

構成は v10.1 の Dirac 型 (2 成分・局在・安定ラベル) — その s=0 は J = 0 **厳密**
(v15.7 の構造零) なので、**検出床が最深** (10⁻¹⁶) の装置になる。装置検査:
全プラケット磁束の一様性 (張り替えが磁束を保つこと) を機械検査 — 偏差 3.6×10⁻¹⁵。

## 2. 結果 — J が 11 桁立ち上がり、世代数は保たれた

(参考: 測定 J = 3.08×10⁻⁵)

三つの事実が同時に成立した:

1. **複素構造は CP を作る。** J は s=0 の構造零から s≠0 で 11 桁立ち上がる —
   プログラム全体 (長方形 v15.7・傾き磁束 v16.1) で初の J ≠ 0 の窓。
2. **世代数 3 と両立する。** 厳密 3 重縮退が全シアーで保たれる (幅 ~2×10⁻¹³、
   ギャップ 0.1147 不変)。傾き磁束では格子の数論が縮退を割った (v13.2) のと対照的に、
   シアーは指数を壊さない — 幾何の形は世代を数える構造と直交している。
3. **大きさが観測と同じ桁。** 最小シアー s=1 で max\|J\| = 2.9×10⁻⁵ ≈ 測定 3.08×10⁻⁵。
   これは Wilson 窓の包絡 (上限) であって fit でも予言でもない — だが、長方形が 0、
   傾き素数点が < 3×10⁻⁸ である中で、複素構造だけが観測の桁に届く。

## 3. 「CP を作る窓」の地図 (v15.7 → v16.1 → 本版)

| 窓 | CP (J) | 世代数 3 | 判定 |
|---|---|---|---|
| 長方形 T²×T² + 実 Wilson (v15.7) | 0 (構造零) | ✓ | CP 失格 |
| 傾き磁束 T⁴, Pf=3 素数 (v16.1) | ≤ 2.6×10⁻⁸ (格子スケールの 1/6000) | ✓ (3 点のみ) | CP 失格 |
| **シアー (複素構造 τ) T² (本版)** | **10⁻⁵〜10⁻⁴ — 観測の桁** | **✓ (全 s)** | **生きている** |

v15.7 の holdout の外れが指定した探索は、二つの棄却を経て三点目で生きた窓に着地した。
CP の源は「モードの非因子化」(v16.1 で棄却) ではなく**幾何の形の位相**である。

## 4. 限界 (正直に)

- max\|J\| は Wilson 窓の包絡であり予言ではない。**次版の本丸**: T²×T² 積模型に
  シアーを入れ、9 量証拠 (深さ)・Occam 罰 (s ∈ Z_N の追加離散パラメータ)・CP (J の
  事後分布) の三つ巴を v6.5 の枠で検定する — 「深さと CP を同時に出す窓」の存否。
- 単一トーラス水準の検定 (質量比の階層は単一 T² では浅い — v7.2 の no-go)。
- シアーの離散化は一実装 (連続 τ 極限・スピン構造の検証は未実施)。
