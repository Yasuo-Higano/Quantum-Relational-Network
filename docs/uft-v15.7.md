## 2. holdout 検定の結果 — 外れが最も雄弁だった

**方法**: v10.1 の T²×T² エンジン (v15.5 で回帰済み: lnZ = −19.863, MAP 全 9 量 lnL = −5.874
が記録と一致) の事後分布の下で、**一度も尤度に入れていない** |V_td| と J を計算した
(v9.1 が「CP 位相・|V_td| は未使用」と明記 — 事後的 fit ではない)。

**結果 1 — |V_td| は外れた**: 事後 95% 帯 [0.0002, 0.0070] に対し測定 0.0086 (上端の 1.23 倍)。
模型は |V_td| を系統的に小さく出す。

**結果 2 — J は構造的に零だった (発見)**: MAP の J = 8×10⁻¹⁷ (機械零)、事後質量の
P(|J| > 10⁻⁶) = 0。理由は構造的: **長方形トーラス + 実 Wilson 格子の湯川位相は
行×列に因子化し、CKM は再位相化で実になる** — この幾何窓は CP 破れを原理的に作れない。
測定 J = 3.08×10⁻⁵ ≠ 0 により、窓は登録と同時に反証された (status: window-refuted)。

**含意 — 外れの中身が地図になる**:
- v12.1–v13.2 の傾き磁束 (非因子化 T⁴) は、証拠比較では 15 nats 負けた「美しい構成」
  だった。しかし **CP 破れは非因子化を要求する** — 傾き/複素構造の方向は美的選択では
  なく、観測が強制する拡張であることを holdout が独立に指した。
- |V_td| の系統的過小も同根の可能性がある (実 CKM は位相の自由度を持たないため
  第三世代混合の大きさが拘束される)。
- 残高 2/9 (σ・磁束・Wilson 線の上位原理) に新しい制約が加わった:
  **正しい窓は CP を作る窓でなければならない** — 選択原理の探索空間 (v15.5) は
  次期、非因子化幾何込みで再設定される。

## 3. 方法の正直な注記

- 「pre-register」の意味: 機構 (v10.1) とその全パラメータは本版より前にコミット済みで、
  holdout 量は構造的に尤度へ入っていない。採点は登録と同じバイナリ・同じ実行で行われ、
  外れも predictions.yml に status として残る (削除しない)。
- J の符号は模型空間の複素共役対称性 (Wilson 反転) で予言されない — 予言されるのは
  |J| (今回は 0)。
- PRED-004〜006 は既存の計算証拠 (v4.3/v6.2 の地形、v0.6 のブースト不変、v7.1 の B−L
  一意性) に基づく登録であり、新しい計算は含まない。

## 4. 残高 6 の更新

「重力以外の QRN 固有予言」は**存在する形式になった**: 台帳 6 件、うち 2 件は即座に
採点され (1 miss / 1 window-refuted)、その外れ方が次の研究 (非因子化幾何での CP) を
指定した。予言は「作る」ものであると同時に「外れて地図を描く」ものである。
