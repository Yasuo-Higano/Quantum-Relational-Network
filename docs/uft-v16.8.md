- **事前登録の答えは (c)**: どの τ_re も \|V_us\| factor を 1.8 以内に置けない (最良は
  s=5 の 1.90 — だが証拠を −4.7 nats 払う)。Cabibbo 過大は τ_re で直せない —
  対称シアー族の一次元断面上では**構造**である。
- **発見: 証拠の谷は τ_re = 1/12** (lnZ₁₀ = −21.76 — v16.4 以来の勝者 1/18 を
  +0.51 nats 上回る)。1/12 は N=18 格子では踏めない点 (s/18 は 1/12 を含まない) —
  **倍格子は τ_re の走査分解能を上げ、間にあった谷を見つけた**。N→∞ で τ_re が
  連続モジュライに近づく描像の、最初の実地である。
- 新しい谷の MAP は全 12 量 factor 5 以内を維持し、質量比が際立つ:
  m_d/m_b **1.08**・m_s/m_b **1.01**・\|V_cb\| 1.10・\|V_ts\| 1.10。holdout も
  \|V_td\| 1.46→**1.23**・\|V_ts\| 1.52→**1.10** と証拠と同じ向きに動いた
  (尤度に入れていない量が谷で良くなる — 模型が過適合でない兆候)。
- **J は s≥2 で解けている** (factor 1.06–1.17)。v16.3 の単調性 (J ∝ s) の頭打ち —
  J が要求するのは「非零の最小シアー」であり、それ以上は要らない。

## 2. 地形は滑らかな坂ではない — s=4 の峰

τ_re=1/9 (s=4) で lnZ₁₀ が −25.9 に沈み、m_u/m_t が factor 7.15 と**走査で初めて
factor 5 を破った**。これは v16.3 の N=18 粗格子で (2,2) (同じ τ_re=1/9) が −24.69 と
沈んでいたのと整合する — **1/9 は格子分解能によらず悪い点**であり、1/12 はその手前の
谷である。|V_us| も 2.20→3.04 と跳ね返っており、地形には谷と峰の構造がある
(単調な綱引き [v16.3] は粗い格子の見え方だった)。峰の起源 (磁束 Q=3 との整合性?
s|N の約数構造?) は未解明 — 記録に留める。
