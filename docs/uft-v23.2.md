# 量子情報網理論 v23.2 — δ⟨K_A⟩ の距離分解: 応答は相殺の微小残差だった

**第二十四期。事前登録 (a)** — だが発見は登録の枠を超えた:
**modular 応答 δ⟨K_A⟩ はバケット間の巨大な相殺の微小残差**であり、R₂ の O(1) 逸脱は
「sub-% の演算子差 × 相殺増幅」である可能性が浮上した。

## 1. 方法

真の modular 応答 δ⟨K_A⟩ = Σ_{ab} K_A[a,b]·δC[b,a] (K_A は DD 厳密構成、δC は
Gauss 対 rank-2 の中心差分) をペア種別に分解:
対角 / x-NN / y-NN / z-NN / x-NNN / 横 NNN / 長距離。
[G4] バケット総和 = 固有基底計算の δ⟨K⟩ (恒等式) を全 d で機械検証 (8 桁一致)。

## 2. 結果 — 相殺の構造

| d | δ⟨K_A⟩ | x-NN | y-NN | z-NN | 長距離 | 増幅率 A |
|---|---|---|---|---|---|---|
| 0.5 | +4.15e-4 | −4.6e-4 | −3.1e-4 | +1.19e-3 | +9.7e-7 | ~4.7 |
| 1.5 | −5.60e-5 | −9.69e-3 | +6.60e-3 | +2.70e-3 | +3.4e-4 | ~340 |
| 2.5 | −1.48e-6 | +5.60e-3 | −5.69e-3 | +3.7e-4 | −2.9e-4 | ~8000 |

- 個別バケットは総和の数倍〜数千倍 — **総和は相殺残差** (d=2.5 で個別項の 0.03%)。
- 非近接 (NNN + 長距離) は d=1.5 で総和の ~6 倍、d=2.5 で ~200 倍の大きさを持ち
  NN 群と相殺している — 登録 (a) (非近接 ≥ 10%) は大差で成立。
- 対角と NNN は小さい (長距離が非近接の本体)。

## 3. 帰結 — R₂ の再解釈 (仮説)

K_boost の NN 形は K_A と 1% 一致 (v23.1)。だが応答の総和が個別項の 0.03–20% の
残差である以上、**sub-% の演算子差が相殺増幅で O(1) の比の逸脱になる** —
R₂ は悪条件比であり、Bisognano–Wichmann は演算子レベルで成立している可能性。
検定は v23.3 (バケット別 K_boost vs K_A 応答比 ρ_b): 全 NN バケットで
ρ_b = 1 ± 3% なら増幅説が確定し、R₂ は器械として退役、G 読み出しは
プロファイル一致 (v23.1) を正当化ゲートとして前進する。
