## 0. 問い — v7.0 残高 6 (QRN 固有の定量予言)

v6.6 は Aziz–Howl (2025) を踏まえ「C>0 ⇔ 量子重力」の二分法を捨てた。残った宿題は
「では QRN (重力 = ユニタリーな量子チャネル) の**固有**シグネチャは何か」。本バージョンは
古典チャネル模型との判別式を密度行列の第一原理計算で導出する (乱数不使用・決定論)。

## 1. 設定と導出

BMV の 2 枝重ね合わせ × 2 質量は 2 量子ビットに厳密に写像される (枝 L/R = |0⟩/|1⟩、
Newton 位相の枝依存性 = σz⊗σz 結合、λτ = Δφ)。

- **量子チャネル (QRN)**: H = (ħλ/4)σzσz のユニタリー発展。
- **古典チャネル (KTM 型)**: 片方を連続測定 (率 Γ) → 記録を古典信号として他方へ
  フィードバック。Wiseman–Milburn 平均化で「同じ実効 σzσz + 測定バックアクション Γ +
  フィードバック雑音 λ²/16Γ」の Lindblad 方程式になる — **力は完全に再現される**。

数値検証済みの帰結:

| | 量子チャネル | 古典チャネル (最適化後) |
|---|---|---|
| もつれ (負性 = 分離可能性と同値) | C = \|sin(Δφ/2)\| | **厳密に 0** |
| 単粒子可視度 | V = \|cos(Δφ/2)\| | V₁V₂ ≤ **e^(−Δφ/2)** (相反定理) |
| 相補性 | **C² + V² = 1** (純粋・可逆) | 純粋度 < 1 (不可逆) |
| 可視度損失の次数 | Δφ² / 8 (2 次) | ≥ Δφ/2 − O(Δφ²) (1 次) |

**相反定理** (数値導出): 同じ力を伝える古典チャネルの総位相緩和は
Γ + λ²/16Γ ≥ λ/2 (等号 Γ = λ/4) — つまり**最適な古典チャネルでも、可視度の損失は
「量子チャネルなら作ったはずのもつれ位相の半分」を下回れない**。
Γ 走査 (0.05λ〜2λ) で上界と最適点を数値確認した。

## 2. 何が更新されたか

1. **量子側の予言の修正 (数値が教えた物理)**: 素朴な期待「量子なら V=1」は誤り。
   もつれが育つと単粒子可視度は V = \|cos(Δφ/2)\| に**下がる** — ただしそれは
   C² + V² = 1 の**可逆な変換** (大域状態は純粋、条件付き測定でフリンジ復活)。
   古典チャネルの損失は同じ V でも**不可逆** (純粋度が落ちる)。
2. **判別は (C, V) 平面**: QRN は円周 C²+V²=1 上、古典チャネル族は C=0 の軸上で
   V ≤ e^(−Δφ/2)。もつれ検出と可視度測定は独立のテスト (後者は単粒子干渉のみで
   測れ、二体相関が不要)。

## 4. 反証条件の最終形 (v6.6 の表を精密化)

- **(C>0, V ≈ cos(Δφ/2), 復活あり)** → 古典チャネル族は死ぬ。QRN と整合。
- **(C=0, V ≤ e^(−Δφ/2), 復活なし)** → QRN (A2/A3: 重力=量子チャネル) が死ぬ。
- **中間 (C>0 だが円周から外れる)** → Aziz–Howl 型の古典 QFT 効果や環境デコヒーレンスを
  含む精密比較の領域 — 単純な二分法では読めない (v6.6 の警告の定量版)。

## 5. 限界 (claims.yml: QRN-EXP-003)

- 古典チャネルは KTM 型 (測定+フィードバック) の族。相反定理はこの族に対する数値的
  厳密結果であり、あらゆる「古典重力」への一般定理ではない (Aziz–Howl 型は別枠)。
- 環境デコヒーレンス (Casimir・ガス衝突) は含めていない — 実験は V の背景損失を
  別途較正する必要がある。
- 2 枝 × 2 質量の最小模型。多枝・連続変数への拡張は未実施。
