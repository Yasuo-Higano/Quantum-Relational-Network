## 1. 定式化 — Fierz 恒等式が Clebsch を消す

```
Σ_a (λ^a)_{γδ}(λ^a)_{αβ} = 2δ_{γβ}δ_{αδ} − (2/3)δ_{γδ}δ_{αβ}
⇒ 2w Q_k·Q_l = w Σ_{αβ} (c†_α c_β)_k (c†_β c_α)_l − (w/3) n_k n_l
```

- サイト = 3 色 8 状態。サイト Casimir は占有だけで決まる: {0,1,2,3} 粒子 →
  {0, 4/3, 4/3, 0} (2 粒子 = 3̄、3 粒子 = **バリオン一重項** — SU(3) 固有)。
- プローブ: 基本 3 は色レジスタとの交換 +w、反基本 3̄ は同方向回転 −w (Fierz の別枝)。
- 真空 (x=0, μ>0) = 0/3 交互の**バリオン結晶**。大域一重項は罰則 λC₂ (λ=6)。

## 3. 開発記録 — run1 の 2 つの教訓

1. **端のプローブは電場から切断される**: (1,3,5) 配置ではサイト 5 の w(5,·) = 0 —
   最終サイトのプローブは弦の物理を持たない (配置バグ)。run1/2 の実測 14/3 < 16/3 は
   この切断配置に固有の遮蔽セクター。内部 (0,2,4) へ移すと **16/3 が機械精度で成立**
   (非破断) — 配置修正が同時に「セクター遮蔽は幾何に依る」ことの実地確認になった。
2. **[A] の再設計 (v20.5 の教訓の適用)**: 「厳密値」はセクター仮定を含むため、
   ゲートは「メソン等式 (機構検証) + バリオン変分不等式 ≤ 16/3」とし、等式成立か
   遮蔽かは診断出力で記録する形に。結果は等式成立側に落ちた。
