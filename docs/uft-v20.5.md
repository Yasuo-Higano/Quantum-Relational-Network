## 1. 定式化 — 開鎖では非可換ゲージ場が厳密に消える

ゲージ固定 U_n = 1 の後、電場項は蓄積色電荷の Casimir になる:

```
H* = −x Σ (c†_{n,α} c_{n+1,α} + h.c.) + μ Σ (−1)^n (n_n−1) + Σ_links L_n²
L_n = Σ_{k≤n} Q_k  ⇒  H_elec = Σ w(k,k) Q_k² + 2Σ_{k<l} w(k,l) Q_k·Q_l,
w(k,l) = N−1−max(k,l)   (長距離非可換 Coulomb)
```

- **行列要素は Fock 基底のスピン演算子だけ** (Clebsch–Gordan 不要)・**ゲージ切断誤差ゼロ**
  (U(1) core の Λ に相当するものが無い)。同サイト双線形 Q± は JW 弦が閉じ符号 +1。
- 外部プローブ (表現 j) は静的スピンとして L_n に合流 (不純物 Hilbert 因子 (2j+1))。
- 大域一重項は罰則 λS²_tot (λ=4) で選択 — ⟨S²⟩ ゲートが 1e-13 で検証。
