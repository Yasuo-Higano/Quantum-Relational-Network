# 量子情報網理論 v22.4 — f64 κ 床の境界測定: 点予言は床の前進で採点不能

**第二十三期。事前登録 (a′)** — 崩壊仮説の点予言 R₂(N=16, d=2.0) = 0.8667 ± 0.025 は
**両点カナリア死で採点不能**。代わりに床境界の精密測定が確定した。

## 1. 事前登録と結果

- 登録: (a) R₁(16, 2.0) 有効 ∧ R₂(16, 2.0) = 0.8667 ± 0.025 = 崩壊確認 /
  (a′) 両点カナリア死 = 床の前進を記録 / (b) 有効なのに外れ = 崩壊反証。
- 結果: **判定 (a′)**。

| N | d | d·gap | R₁ | R₂ |
|---|---|---|---|---|
| 12 | 2.0 | 0.4821 | 0.97966 [除外] | (1.798) |
| 16 | 2.0 | 0.3691 | 0.98006 [除外] | (0.819) |

## 2. 確定情報 — 床境界は d ∈ (1.5, 2.0)、劣化プロファイルは N 非依存

R₁ の 1 からの偏差: d=1.5 で ~1e-4 → **d=2.0 で 2.0%** (0.97966 / 0.98006 —
N=12/16 の差わずか 4e-4) → d=2.5 で 5.5% (v22.2) → d≥3.5 で崩壊。
**f64 κ 床は波束の絶対深さ d だけで噛む局所現象** (v22.2 発見 2 の境界精密化)。
生存窓 d ≤ 1.5 が全 N 共通の器械限界として確定 — この窓内での崩壊仮説の
検定が v22.5 (判定 b — 反証)。

## 3. 正直な注記

- N=16, d=2.0 の R₂ = 0.819 は見かけ上予言帯の近くだが、カナリア死 (核が 2% 狂う
  領域) のため採点しない — カナリアを設けた理由そのもの。
- N=12, d=2.0 の R₂ = 1.798 は床汚染域の値が任意方向に暴れる実例 (記録)。
