## 1. 方法 — 原理を事前分布にする

v10.1 の T²×T² 証拠エンジン (五重和・σ marginalize・安定ラベル) を再利用し、
**データを見ない**スコア s_P(config) を持つ原理候補 P を正規化事前 π_P ∝ e^{s_P} として
同じ和に注入する:

```text
lnZ_P = ln Σ_c π_P(c) L(9 観測量 | c)     (π_P 正規化 — 次元罰なし、集中の当否だけ)
```

**エンジンの較正 (一次ソースとの回帰)**: lnZ(一様) = −19.863 (v10.1 公表 −19.86)、
lnZ(M_diag) = −23.612 (公表 −23.61)、MDL 正規化子 = 4e-11 (解析値 0)。全て [PASS]。

**候補の operationalization** (入力・自由パラメータ・棄却条件つき — PROMPT/2 柱 4 の要求):

## 3. 読み方 — 三つの発見

1. **宇宙は「最深」を選ばない。** 階層最大化事前は、観測 (m_u/m_t ≈ 1.3×10⁻⁵) より
   はるかに深い config に重みを集中させ、−494 nats で自滅した。階層の深さは
   最大化ではなく**特定の値** — 「深いほど良い」型の物語 (動的選択・人択的偏りの
   素朴版) は、この窓では定量的に死んでいる。
2. **答えは台地ではなく崖に住む。** 高尤度 config は Wilson 空間の急峻な領域にあり、
   感度最小 (自己安定) 事前は λ を上げるほど単調に悪化する。「摂動に強い真空が
   選ばれる」は逆向きだった。ゼロモードの精密な配置が階層を作る以上、これは
   機構的にも整合的な棄却である。
3. **弱い正信号は「結合の強い真空」方向にだけある。** P4 (+0.32) は生存閾値に
   届かないが、唯一符号が正で γ について単調増加。真の一ループ自由エネルギーへの
   置き換えが次の的として残る。

**残高 2/9 の更新**: σ・Wilson 線は、幾何への還元 (位置 v10.3・向き v10.3・傾き
v13.2) に続き、**選択原理のクラス (単純さ・安定性・極値化・熱力学代理) にも還元
されなかった**。既約な離散データという住所が、より広い対抗仮説空間で確認された。
同時に C5 の在庫が一つ減った (P5 は空)。
