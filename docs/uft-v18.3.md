## 2. 読み方 — 情報量のある外れ

**最小幾何シーソー (M_R ∝ I) は、クォークを成功させた「重なりの階層性」を
そのままレプトンに運ぶ — だが実際のニュートリノは大混合・弱い階層であり、
これを 68% レベルで棄却する。** J=0 構造零が複素構造を指したのと同型の外れ方で、
指す先は明確である:

> シーソー m_ν = Y_ν M_R⁻¹ Y_νᵀ は **M_R の構造次第**で整列した階層から大混合を
> 作れる。外れの住所は Dirac 側 (幾何は正しい可能性が高い — θ13 は当たった) では
> なく **Majorana 側 = B−L 破れの幾何** (M_R のテクスチャ) にある。

これは偶然ではない符合を持つ: anomaly-search 論文の結果「第二の力の最小の
足し方は B−L で、それは ν_R を強制する」— **B−L 破れ場のプロファイルが M_R の
幾何を与える**のは、このプログラムの台帳が既に持っている構造である。次の的は
「M_R = B−L 破れ場との重なり積分」の実装 (新しい離散データ: B−L 場の住所)。

## 3. 装置

- 収束点 (36×18; 3,3) 固定・キャッシュ命中。σ_H 事後重み (G0, 10 量) を先に計算し
  正しく折り込み (peak は物理幅 ~1/18 で整合)。e 配位の事後カット we<1e-8
  (帯に影響しない高速化)。全実行 75 秒。
- 帯の"広さ"自体が正直な出力 — ν セクターは尤度に触れないので事後で絞られない。

## 4. 次の的

- **M_R の幾何化** (B−L 破れプロファイル → M_R テクスチャ) — 大混合が出るかが
  最初の検定。出れば PMNS が holdout から証拠へ昇格する道が開く (J の前例)。
- sign(J_lep) の予言には ν 住所の絞り込み (= M_R 構造) が前提。
- 0νββ 有効質量 m_ββ の帯は M₀ が入る — M_R 幾何化とセットで。
