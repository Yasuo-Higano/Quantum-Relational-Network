# 量子情報網理論 v22.3 — SM 直積 core: 単一状態から 3 因子ゲージと重力を同時に読む

**第二十三期 課題 1。事前登録 (a) 成立** — 単一の力学 (クォーク二重項 (3,2,1/6) の拘束 core) と
その単一の基底状態から、[A] 代数・[G] 3 因子ゲージの弦・[g] 重力第一法則・[M] 幾何相関減衰の
4 読み出しが同時に成立した。v21.7 (最小 5 条件監査) が条件 1 の欠落として名指しした
「同一状態からの同時実演」がこれで埋まる。

## 1. 位置づけ

- v20.5/20.6 (SU(3) core)・v20.9 (SU(2))・v21.0 (SU 階段) は**因子ごと**の toy だった。
  PROMPT/3 §9 条件 1 は「単一の基礎自由度・単一の力学から時空/重力/ゲージ/物質が
  **同時に**読み出される」ことを要求する — 本版はその 1+1D 実演。
- 重力側の語彙 (関係エントロピー・modular 応答) とゲージ側の語彙 (拘束 core の弦) が
  **同じ一つの波動関数**の上で両立することが要点。

## 2. 模型と方法

- 物質 = クォーク二重項 Q = (3, 2, +1/6) の 6 成分/サイト (色 α × 弱 a)。開鎖 N=4
  (フル 24 モード)。ゲージ消去後の Coulomb 形式:
  H = −x Σ hop(種対角) + Σ_links [C₃(L) + C₂(L) + Y_L²] + λ(C₃tot + C₂tot), λ = 3。
- 色 Fierz: 2w Q_k·Q_l = w Σ E^{αβ}_k E^{βα}_l − (w/3) n_k n_l (E^{αβ} = Σ_a c†_{αa}c_{βa})。
  弱も同型 (F^{ab} = Σ_α c†_{αa}c_{αb})。Clebsch 不要の交換演算子形式。
  サイト対角自己項は 64×64 の on-site Casimir 行列を起動時に数値構成。
- セクター = 等マージナル (cc₀=cc₁=cc₂ ∧ ww₀=ww₁), dim = 103,704。罰則により
  基底状態は大域 SU(3)×SU(2) 一重項。
- 器械: 再開始 Lanczos (温間スタート対応)・1 サイト RDM (64×64)・
  modular 演算子 K₀ = −ln ρ_A (jacobi 対角化)。

## 4. 開発記録 — 多体第一法則の構造ゼロ分類学 (run1–5 保存)

[g] の器械は 6 構成が連続で失敗した。全て δS ~ 1e-15 の厳密ゼロで、診断の系譜が
そのまま**多体 RDM 第一法則の検証装置に対する no-go の分類学**になった:

1. **転置マスターゼロ** (回転型の普遍 no-go): 実反対称生成子 Y と実基底状態では
   ρ_A(−ε) = ρ_A(ε)ᵀ が厳密。S は転置不変・K₀ は対称行列なので、**中心差分の奇部は
   どの回転 Y でも恒等ゼロ** — 状態回転には検証可能な 1 次第一法則が存在しない (0=0)。
   run1–4 の個別診断 — (i) 実対称生成子 → δρ_A の 1 次が純虚反対称、(ii) 境界跨ぎ輸送
   → n_A 超選択、(iii) A 内ユニタリ → 等スペクトル、(iv) 大域一重項 × カレント回転 →
   δρ_A が随伴方向のみで Tr(随伴×不変) = 0 — は、この普遍ゼロの部分症状だった。
2. **PH ゼロ** (H 摂動の半分を殺す): 半充填の粒子正孔対称の下で V = n₁ は PH-奇
   (n → 6−n)。ψ(−λ) = U_PH ψ(+λ) となり、S と ⟨K₀⟩ は PH 不変汎関数なので中心差分が
   全 λ で厳密ゼロ、真の dS/dλ も 0 (これも 0=0)。
3. **正解**: 落とし穴欄の確立手法「無限小のハミルトニアン摂動」+ **V を PH-偶に取る**。
   V = (n₁−3)² (隣接サイトの電荷揺らぎ・一重項・実対称) で初めて δS ~ λ × O(1) の
   健全な信号が出た。v19.1 (自由場 C 行列) が回転で検証できたのは one-sided
   O(ε²) 応答を見る別形式だったため — 多体 RDM 形式には移植できない。
