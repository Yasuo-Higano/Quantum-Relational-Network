## 1. 設定と器械

模型は v22–v25 系と同一規約の 3D staggered (η_x = 1, η_y = (−1)^x, η_z = (−1)^{x+y},
staggered 質量 m(−1)^{x+y+z})。境界は **x のみ反周期 twist** — ノードの kx = π/2 を
格子から外し E_min = sin(π/N) > 0 (N=12 で 0.2588)、y/z は周期のまま人工 IR ギャップを
最小化する。[S0] 解析分散 ±√(Σcos²k + m²) の多重集合と dense jacobi が 4 系
(N ∈ {8,12} × m ∈ {0, 0.5}) で max|Δ| ~ 1e-12 — 模型実装の同一性証明。

器械の定義:
- **T_00(x)** = 各ボンドのエネルギーを半分ずつ両端に配る局所密度 + 質量項 (on-site)。
  [S1] Σ_x T_00(x) = H が **bit 厳密** (max|Δ| = 0.0)。
- **T_00(q)** = Σ_x e^{iq·x} T_00(x) (ボンドは中点位相 — 対称な密度)。
- **vierbein 結合** = x ボンド変調 t → (1 + ε f(x)) t。δH/δε が T_xx 型 stress を定義。

## 2. 三要件の厳密検査

1. **保存 (Ward の時間成分)** [S2]: C(q) := i[H, T_00(q)] について C(0) = 0 が厳密
   (エネルギー保存)、‖C(q)‖/q は q → 0 で有界安定 (N=12, r(2q₁)/r(q₁) = 1.06) —
   「T_00 の流れ J^E が局所的に存在する」の格子版。
2. **結合 (Feynman–Hellmann)** [S3]: 一様歪みで dE₀/dε = ⟨T_xx(0)⟩ が 9 桁一致
   (−102.71472139 vs −102.71472148)。q≠0 変調では一次応答が運動量保存で零 (4.7e-11)、
   **二次応答 d²E₀/dε² = −28.823286 が Lehmann 和 −χ_O = −28.822524 と独立照合で一致**
   (相対 3e-5) — 「エネルギーの ε 応答」と「行列要素の和」という 2 経路の相互証明。
3. **応答の対称性と収束** [S4]: χ_00 の **y↔z 対称が機械精度 (~1e-15)** — v24.x の
   W 対称性 (λ_y = λ_z の起源) の応答関数版の実測証明。共通運動量 q = π/2 の χ_00/V は
   N=8 → 12 で 1.5% (m=0) / 0.5% (m=0.5) 一致。

副産物の測定: **x/y 方向非対称** (twist-x 境界 + η 構造) は最小 q で
m=0: 0.446 (N=8) → 0.344 (N=12)、m=0.5: 0.179 → 0.037 と N とともに減少 [S4a'] —
gapped 系は速く、gapless 系は遅い (有限サイズの普遍的な影)。[S5] ギャップ対照:
χ_00(q_min) は m=0.5 で massless の 0.884 倍に抑制 — 器械は最も粗い物理を正しく見る。
[S6] 変異検出: η_z 落とし (S0 が検出, 逸脱 0.75)・質量二重計上 (S1 が検出, 0.50)。

## 3. 開発記録 (run1 → run2 のゲート較正 — v24.3 W1/W2 と同型)

run1 の FAIL 2 件はいずれも**ゲートの前提誤り**で、物理数値は run1/run2 で同一:
1. S4a 初版は「χ_00 の x↔y 立方対称 (1e-10)」を課した — 前提誤り。twist は x のみ、
   η 位相構造も x を特別視する (v24.x の λ_x ≠ λ_y = λ_z と同根)。厳密なのは y↔z
   (W 対称性) で、x/y 非対称は測定すべき有限サイズ量 (最大 45% @N=8 massless)。

   q² 補正で r は成長し得る (線形消滅と矛盾しない)。N=12 の小 q 側 2 点・窓 [0.8, 1.25]
   に較正。
