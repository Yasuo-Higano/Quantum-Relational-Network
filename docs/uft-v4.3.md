## 2. 隣人はなぜ現実でないか(追加の物理フィルタ)

16 成分の解(例: (3,1)₋₁/₆ を含む)はアノマリー的には無矛盾だが:
1. **分数電荷ハドロンを予言する**: クォークの超電荷が n ≡ ±2 (mod 6) を破るため、
   3 クォークの色一重項(バリオン)が半整数電荷を持つ — 観測的に強く排除
   (分数電荷の安定粒子は精密探索で否定されている)。
2. **単純群に埋め込めない**: v3.1B の 5̄⊕10 パターンに合わない
   (電荷量子化=GUT 埋め込み可能性を課すと、≤24 成分では SM のみが生き残る)。
3. 24 成分のファミリーは電荷が ±9/6 に張り付いた高電荷構成で、同様に排除される。

> **無矛盾性(アノマリー)+電荷量子化(ハドロンの整数電荷)を課すと、
> ≤24 成分の理論空間で標準模型は文字どおり一人である。**

## 3. 考察

- 「最小性の選択理由」への視界が変わった: 選択は不要かもしれない —
  観測と整合する候補が(この範囲には)**他に無い**。
  恣意性は「なぜ ≤24 成分の世界か」(= なぜ低エネルギーに少数の自由度か)に押し込まれ、
  それは RG(v0.3, v3.3)が答える種類の問いである。
- 17〜23 成分の空白は印象的: 理論空間は連続的な landscape ではなく、
  疎な離散点の集合(quantized landscape)。
- 限界(正直に): 探索範囲は表現 ≤(3,2)・|Y|≤3/2・≤6 多重項。より大きい表現
  ((8,1), (6,1) 等)や 3 因子以外の群は未走査 — 拡張は将来課題。
