# 量子情報網理論 v24.3 — BW スケーリング窓監査: 32/27 の棄却と、消えない異方的 λ

**第二十五期の本丸 I。判定 (a′) — 窓でも非 1 の λ が生存 (ただし 32/27 ではない)**。
認証済みブロック器械 (v24.2) で λ の全面 λ(N, ξ, 方向) を N ≤ 96 まで測り、
PROMPT/6 §1.1–1.2 の two つの問いに答えた: 32/27 は正しいか (**否 — 棄却**)、
固定 ξ の値はスケーリング窓で 1 に戻るか (**戻らない — 完全に平坦**)。

## 1. 32/27 の棄却 — 外挿論争を直読みで裁く (W5)

固定 ξ ≤ 3 推定器 (v23.4/23.5 互換) の N 系列は美しく収束し、**N=96 で外挿なしの
直読み**ができる (mod-0 族; 差分比は 1/N⁴ 則と一致 — v23.4 の指数は正しかった):

| N | 16 | 24 | 32 | 48 | 64 | 96 |
|---|---|---|---|---|---|---|
| λ_bulk | 1.185893 | 1.185557 | 1.185496 | 1.185473 | 1.185469 | **1.185468** |

mod-2 族も同値に収束 (N=18: 1.185667 → N=34: 1.185481)。

**λ∞(固定 ξ) = 1.185468(2)。32/27 = 1.185185 との偏差 2.82e-4 は収束残差
1.5e-6 の 187 倍 — 32/27 は棄却**。v23.5 の「0.006% 一致」は N ≤ 16 で
mod-2 族が 2 点しかない外挿の分解能不足だった (PROMPT/6 §1.2 の指摘どおり。
族平均が偶然 32/27 に落ちていた)。**v24.0 の次期本丸「32/27 の理論導出」は
対象消滅により中止** — アーティファクトを厳密に導出しても物理は出ない。

## 2. スケーリング窓 — λ(ξ) は完全に平坦 (1 に戻らない)

測定規約 (v24.2): DD + クランプ 2 点梯子 (1e-30/1e-26)。κ 床の実地検出により
信頼域は全 N で ξ ≤ 5 (ξ=6 は Δ~1.4e-4、ξ=7 は ~1.3e-3 の床誤差付きで参考)。

N=96 のプロファイル: **λ_A(ξ) = 1.18547 が ξ = 1..6 で 5 桁不変**。
窓フィット (事前登録 4 族, holdout 選抜): λ∞ = 1.18550、族間スプレッド 3e-5。
1 へのドリフトは検出されない — «λ(ξ→∞) → 1 で BW 素朴回復» の分岐 (a) は不成立。

横方向はさらに強固: **λ_y(ξ) = λ_z(ξ) = 1.22943 が全 ξ・全 N で不変**
(y と z が 7 桁一致するのは新事実 — η の位相構造は y/z 非対称なのに、モジュラー
応答は y/z 対称)。異方性 λ_y/λ_x = 1.03708 は UV でも窓でも変わらない。

## 3. 器械知識 (開発記録)

- 対称性不変量 B_x ≡ 0・A_z ≡ 0 は**信頼域内で** 1e-10 以下で成立。初版ゲートは
  全 ξ 域で max を取り誤発報 — 窓外 (クランプ域) の破れ ~1e-5 は κ 床雑音の
  診断値であって器械故障ではない (mod-0 の N ≥ 24 のみに出る — μ=0 横モードの
  深部縮退とクランプの相互作用。mod-2 は窓外でも 1e-25)。
- 固定 ξ のクランプ梯子差は N 非依存 (深部 κ 尾の飽和) — 信頼域 ξ*=5 は
  サイズを上げても広がらない。より深い窓は四倍精度 (qd) の課題。

## 4. 帰結 — λ の解釈の転回

λ は「格子エンタングルメント温度の有理繰り込み」(v24.0) ではない:
(i) 32/27 は棄却、(ii) 値は方向依存 (λ_x ≠ λ_y = λ_z) のまま窓で不変 —
物理温度なら等方化するはず。残る解釈候補は「NN チャネルの方向別演算子規格化」
(v23.2 の長距離重み ~16% の裏面) — v24.4 (演算子整合) が Z_T の言語で確定させる。
