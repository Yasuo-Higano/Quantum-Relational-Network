## 1. 結果 — 判定 (b)、そして点言明より厳しい分布言明

10 量事後の 68% 帯 (16/50/84%) と MAP、測定との被覆:

| 量 | (3,3) 帯 | (1,3) 帯 | 測定 | 被覆 (3,3)/(1,3) |
|---|---|---|---|---|
| \|V_us\| | [0.422, 0.516] | [0.250, 0.490] | 0.225 | ✗ / ✗ |
| \|J\| | [1.13, 2.65]×10⁻⁵ | [0.82, 2.17]×10⁻⁵ | 3.08×10⁻⁵ | ✗ / ✗ |
| \|V_cb\| | [0.0277, 0.0403] | [0.0291, 0.0586] | 0.041 | ✗ (紙一重) / ✓ |
| \|V_td\| (holdout) | [0.0065, 0.0115] | [0.0052, 0.0104] | 0.0086 | ✓ / ✓ |
| \|V_ts\| (holdout) | [0.0270, 0.0413] | [0.0291, 0.0571] | 0.0405 | ✓ / ✓ |

- **事前登録の答えは (b)**: \|V_us\| の 68% 帯は重なる ([0.42, 0.52] vs [0.25, 0.49]) —
  縮退は証拠 (lnZ) だけでなく事後分布の置き場所でも縮退している。
- **ただし測定の位置は非対称**: \|V_us\| = 0.225 は (1,3) 帯の下端 0.2497 の**すぐ外**
  (分布の ~10–16% 裾)、(3,3) の床 0.422 の**遥か下**。幾何同士の帯は重なるが、
  測定は (1,3) の側に立つ。
- **分布言明は点言明より厳しい**: MAP スコアカードの「12/12 factor 5 以内」(v16.5〜)
  に対し、68% 帯は両幾何で \|V_us\| と \|J\| を外す — **模型族の張力は \|V_us\|
  (過大方向) と J (過小方向) の 2 箇所に集中**しており、これは v16.9 の綱引き
  (\|V_us\| は非対称を・J は対称を要求) の分布版である。
- **holdout は両幾何とも 68% で被覆** (\|V_td\|, \|V_ts\|) — out-of-sample の健全性は
  分布レベルでも維持。被覆数は (1,3) 3/5 vs (3,3) 2/5 (\|V_cb\| の差 — ただし
  (3,3) は 0.0403 vs 0.041 の紙一重で外れており、過大評価しないこと)。
