# 統一場理論 v12.2 — 傾き T⁴ の証拠検定: アンザッツなしの 3 世代を秤にかける

**Version**: v12.2
**Date**: 2026-07-04
**シミュレーション**: `sim/src/bin/v122_tiltwilson.rs` → `results/v122_tiltwilson.txt` (機械可読: `results/v122_tiltwilson.json`)
**対応残高**: 残高 2 — v12.1 (構造編) の証拠編

---

## 1. 位置づけ

v12.1 は傾き磁束 T⁴ が指数 = Q₁Q₂ + ts = 3 の厳密 3 世代を持つことを実証した (構造)。
本バージョンはその 9 量証拠を計算し、機構のはしごと比較する (秤)。

**この構成の原理的な利点**: 3 つのゼロモードがそのまま 3 世代なので、
- 「9 個から 3 個を選ぶ射影」が存在しない → v10.1 の σ が存在しない
- トーラス間の対がない → v9.2 のラベル綱渡りが原理的に発生しない
- 特異値と |CKM| はモードの順序・位相に完全不変 → 規約が入り込む場所がない

## 2. 方法

- Wilson 線: セクターごとに定数ホロノミー (w_{y1}, w_{y2}) = 2π(k₁,k₂)/6 ∈ Z₆²
  (36 配置/場 — はしごの模型と同じ事前体積 5·ln36 + σ_H 4 点)。
- 傾きがあると並進では Wilson を代用できない (v12.1 開発記録) ので、
  **36 配置を個別に対角化** (2592², std::thread で 12 並列 — 外部クレートなし)。
- 検証: 全 36 配置で縮退幅 < 10⁻⁸ (**指数 3 は Wilson 線に依らない**)、(0,0) 配置の
  ギャップが v12.1 と一致 (回帰)。

## 3. 結果

| 検査 | 結果 |
|---|---|
| 全 36 配置の縮退幅 max | 1.4×10⁻¹² (< 10⁻⁸) [PASS] |
| ギャップ min | 0.3195 (全配置で一様 — スペクトルは Wilson 不変) |
| (0,0) 回帰 (v12.1 の 0.3195) | [PASS] |

**証拠**: lnZ₉(傾き T⁴) = **−94.51**

| 機構のはしご | lnZ₉ |
|---|---|
| 対角対 | −23.61 |
| 一様オービフォールド | −21.83 |
| 向き + 並進 | −20.86 |
| S₃ 対 | −19.86 |
| **傾き T⁴ (本結果)** | **−94.51** |

**S₃ 対に −74.65 nats 及ばない — 正直な陰性結果。** 指数 3 の構成は Wilson 線に
完全に頑健だが、磁束 (2,2) の N=6 点では到達可能な階層が浅すぎる (m_u/m_t = 1.3×10⁻⁵
に遠く届かない)。構造 (v12.1) と証拠 (本結果) の距離が 75 nats と測れたことが収穫で、
**同じ格子で指数 3 を保つ深い磁束 (例: (3,3,2,−3) は 9−6=3)** の系統走査が次の的である。

## 4. 限界

- 磁束 (2,2,1,−1)・N=6 の 1 点。指数 3 を保つ他の磁束 (より深い階層を持ち得る組合せ)、
  格子の細分化、σ_H 事前の拡張は未走査。
- ヒッグスは T⁴ 原点中心の等方ガウス 1 種。
