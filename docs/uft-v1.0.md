## 0. 何を主張し、何を主張しないか

**主張しないこと**: これは完成した「万物の理論」ではない。標準模型のパラメータも
宇宙定数の値も導出できていない(§6)。

**主張すること**: 群盲評象の各理論 — 一般相対論・量子力学・場の量子論・統計力学・
量子情報 — がそれぞれ触っていた「象」の輪郭は、すでに描ける。それは:

> **世界の基本的な実在は時空でも粒子でもなく、量子相関(もつれ)の網である。
> 時空・重力・力・物質は、その網の異なる読み出し方である。**

この描像は本プロジェクトの発明ではなく、過去 50 年の独立な発見(BH 熱力学、ホログラフィー、
創発重力、It from Qubit)が収束した先にある。本プロジェクトの寄与は、
**この描像を一つの公理系に組み、各構成要素を第一原理の数値実験で検証し、
一つの再現可能なリポジトリにまとめた**ことである。

## 1. 公理系 — 量子情報網理論(Quantum Relational Network, QRN)

**A0(実在)** 物理的実在は、有限次元ヒルベルト空間 H = ⊗ᵢ Hᵢ 上の量子状態 |Ψ⟩ と、
その局所代数構造(テンソル分解)である。時空・場・計量は公理に現れない。

**A1(動力学と時間)** 大域的な発展は厳密にユニタリー。「時間」は外部パラメータではなく、
状態の熱的構造(モジュラー流/部分系間相関)として創発する。
逆温度は虚時間方向の周期であり、量子と熱は同じ確率構造の二つの読みである。

**A2(幾何の辞書)** 幾何量は情報量の別名である:
- **距離** d(i,j) := 相互情報量 I(i:j) の減衰率から再構成される最短経路構造
- **面積** A(∂R) := 4G·S_ent(R) — 領域境界の面積はそれを横切るもつれの量
- **次元・因果** := 相関/順序構造の大規模統計(スケール依存の創発量)

**A3(重力)** アインシュタイン方程式は基本法則ではなく、A2 の辞書のもとで
**エンタングルメントの熱力学第一法則 δS = δ⟨K⟩ を全ての小領域に課したときの状態方程式**である
(線形化版は厳密に同値: Jacobson 1995 / Faulkner et al. 2013)。

**A4(物質と力)** 網の低エネルギー集団励起が「場」と「粒子」。局所的な記述の冗長性
(内部空間の接続)が「ゲージ場」。臨界性と普遍性が連続場理論を選び出し、
真空の選択(対称性の破れ)が質量を与える。

**A5(情報保存)** いかなる過程(ブラックホール蒸発を含む)でも大域的ユニタリー性は
破れない。情報は消えず、相関の中を移動するだけである。

## 3. 群盲評象の解決 — 各理論はどこを触っていたか

| 理論 | 触っていた部位 | QRN での位置づけ |
|---|---|---|
| ニュートン力学 | 遠くから見た象の影 | 全ての理論の h→0, c→∞, 弱場極限 |
| 一般相対性理論 | 動的な幾何(脚) | A2+A3 の古典極限・粗視化。計量は相関構造の巨視的変数 |
| 量子力学 | 情報と確率(鼻) | A0+A1 そのもの(基礎層) |
| 場の量子論・標準模型 | 励起と対称性(耳) | A4。臨界点近傍の普遍的な低エネルギー理論 |
| 統計力学・熱力学 | 普遍性と熱(胴体) | A1+A3 の言語。「重力=状態方程式」 |
| 弦理論 | UV 完全性とホログラフィー(牙) | A2 の最も具体的な実現例(AdS/CFT, RT 公式) |
| ループ量子重力 | 背景独立・離散幾何(皮膚) | A0 の精神(幾何を量子化する側からの接近) |
| 因果集合 | 因果と数え上げ(尾) | A2 の因果側の実装。ローレンツ不変な離散性 |
| It from Qubit | 骨格の言語 | QRN の直接の母体 |

矛盾して見えた理論たちは、**同じ網の異なる粗視化・異なる極限**だった、というのが v1.0 の答えである。

## 4. 予言と反証可能性

QRN は研究プログラムだが、以下の点で殺すことができる(Popper 的に健全である):

1. **重力は量子的でなければならない。** 重力相互作用だけで 2 つの質量がもつれる
   (BMV 実験、実現目前の水準)。もし重力が基本的に古典なら、A2/A3(幾何=もつれ)は棄却される。
2. **ブラックホール蒸発は Page 曲線に従う**(v0.8)。情報喪失が確立すれば A5 ごと棄却。
3. **ローレンツ不変性はプランクスケールでも(主要次数で)破れない**(v0.6 の因果集合型離散性)。
   GRB 光子の энергия依存遅延などで破れが確立すれば、離散化の形が制約/棄却される。
4. **次元の走行**: 超高エネルギーで実効スペクトル次元が 2 へ低下する(v0.6; 複数の
   量子重力アプローチが収束する予言)。
5. **大域的対称性は存在しない**(量子重力の一般定理と整合): いかなる大域電荷も
   究極的には破れる(例: 陽子は安定でない)。
6. **面積=エントロピーの普遍性**: アナログ系(冷却原子など)での面積則・第一法則の検証は
   本リポジトリの計算と同じ構造を持つはず(既に複数実験と整合)。

## 5. 導出のスケッチ(公理 → 既知の物理)

- **A2+A3 → GR**: 全ての局所 Rindler 地平線に δQ = TδS(S=A/4G)を課す → Einstein 方程式
  (Jacobson)。線形化版は v0.7 で数値検証した第一法則と同値(FGHMV)。
- **A3+A5 → BH 熱力学と情報回収**: 面積=S から Bekenstein–Hawking、第一法則から Hawking 温度、
  ユニタリー性から Page 曲線(v0.8)。量子補正版 RT(アイランド)が両者を整合させる。
- **A4 → 標準模型の骨格**: 局所対称性→ゲージ場(v0.4)、SSB→質量(v0.3)、
  くりこみ普遍性→「なぜ少数のパラメータの場の理論に見えるか」(v0.3)。
  ※ ゲージ群 SU(3)×SU(2)×U(1) と 3 世代の**選択**は未導出(§6)。
- **A1 → 時間の矢**: 時間=モジュラー流なら、時間の向きは相関(エントロピー)の勾配。
  初期宇宙の低エントロピーは未解決(§6)。

## 6. 未解決問題(正直な一覧)

1. **テンソル分解の起源**(=「何の」もつれか): A0 は分解 ⊗Hᵢ を仮定する。分解自体を
   動的に選ぶ原理(演算子代数・誤り訂正符号の言葉で進行中の研究領域)が要る。
2. **ローレンツ符号・実時間の第一原理定式化**: v0.2 の Wick 回転と v0.6 の因果集合は
   手がかりだが、統一された実時間の動力学は未完。
3. **3+1 次元での非線形 Einstein 方程式の完全導出**: 線形化は v0.7 レベルで確立。
   非線形化は Jacobson の平衡仮定か相対エントロピーの高次構造に依存 — 証明未完。
4. **標準模型の導出**: ゲージ群・世代数・湯川結合・混合角。QRN はまだ何も言えない。
   (誤り訂正符号の対称性から出るという仮説はあるが検証不能な段階)
5. **宇宙定数問題**: A2 は「なぜ小さいか」に答えない。因果集合の Λ〜1/√V 揺らぎ予言
   (桁が合う唯一の事前予言)との接続は開いた問題。
6. **測定問題**: A5(ユニタリー)を貫くならエヴェレット的だが、確率(ボルン則)の導出と
   「観測者=網の部分系の自己参照」の定式化は未完。
7. **実験スケール**: プランク長 10⁻³⁵ m。§4 の間接予言と重力量子性実験が当面の戦場。

## 7. 結語

v0.1 で、水星の近日点と量子トンネルという「別々の宇宙」から出発した。
v0.2 で量子と熱が同じものであることを、v0.3–0.4 で物質と力が同じ格子の統計力学から
生えることを、v0.5–0.7 で**時空そのものがもつれの幾何学であり、重力がその熱力学である**ことを、
v0.8 で情報が決して失われないことを、すべて自分のコードで確かめた。

象の全体像は「量子情報網」である — 少なくとも、50 年分の独立な手がかりと
本リポジトリの 10 個の数値実験は、すべてその方向を指している。
残された未解決問題(§6)は多いが、それらは**どこを掘ればよいかが分かっている問題**になった。

> 時空は、もつれが見せる夢である。重力は、その夢の熱力学である。
> そして情報は、決して失われない。
