# 統一場理論 v13.1 — Lanczos 疎ソルバ: 大格子への道具

**Version**: v13.1
**Date**: 2026-07-05
**シミュレーション**: `sim/src/bin/v131_lanczos.rs` → `results/v131_lanczos.txt` (機械可読: `results/v131_lanczos.json`)
**対応残高**: 残高 11 (道具 — v12.3 が確定した限界の解除)

---

## 1. 何をしたか

`lib.rs` に複素エルミート作用素の最低 k 固有対を求める **`lanczos_lowest_herm`**
(matvec ベース・完全再直交化・部分空間再スタート・残差検証つき、外部クレートなし)
を自作した。稠密ヤコビ法の実用上限 (~2600 次元) を超える格子への道具である。

## 2. 検証 — 4 段全て PASS

| 検査 | 結果 |
|---|---|
| [1] 稠密ランダムエルミート (n=40) 最低 6 固有値 vs ヤコビ法 | max\|Δλ\| = 1.4×10⁻¹³ [PASS] |
| [2] 2D 磁束トーラス (N=18, Q=3) の縮退 3 バンド | 幅 1.7×10⁻¹³ / ギャップ 0.1147 (一次ソース 0.115) [PASS] |
| [3] N=6 傾き T⁴ (2,2,1,−1) の v12.1 回帰 | ギャップ 0.3195 一致、**89 ms (ヤコビは 594,000 ms — 6,700 倍)** [PASS] |
| [4] 決定論 (同シード bit 一致) と初期値独立性 (異シード Δλ = 8×10⁻¹⁴) | [PASS] |

## 3. 開発記録 — 残差検査が仕事をした

最初の実行で [2] が落ちた (残差 0.64): 低磁束密度の 2D トーラスは低エネルギー準位が
密で、Krylov 次元 120 では最低バンドに未収束だった。**返り値に残差を含める設計**が
「収束したふりの答え」を弾いた。修正は n=324 の全次元化 (小さい系では厳密)。
教訓: Lanczos の答えは常に残差とセットで読む — 以後の利用側もこの規約に従う。

## 4. 射程

N=18 の T⁴ (104,976 サイト, 21 万複素次元) の最低バンドが matvec ~ms × Krylov 数百
= **数秒〜数十秒/配置**で解ける見込み。v12.3 で「N=6 窓に解なし」と確定した
深い磁束族の探索が、v13.2 で N=18 において再開できる。

## 5. 限界

- 完全再直交化は O(m²n) メモリ/計算 — m ≲ 千・n ≲ 10⁷ の範囲用 (現在の用途には十分)。
- 求めるのは最低側の少数固有対のみ (内部固有値には shift-invert が要る — 未実装)。
