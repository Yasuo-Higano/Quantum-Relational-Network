## 1. g(μ) の厳密閉形式 — 既知厳密解の同定 (コミット I)

### 1.1 先行研究 (C0)

V. Eisler, "On the Bisognano-Wichmann entanglement Hamiltonian of nonrelativistic
fermions", **J. Stat. Mech. (2025) 013101** [arXiv:2410.16433] は、半無限 staggered 鎖

> Ĥ = −(1/2)Σ(c†ₙcₙ₊₁ + h.c.) + μΣ(−1)ⁿ c†ₙcₙ (同論文 式 (64))

の半無限領域 entanglement Hamiltonian を、相関行列と可換な三重対角行列
T (t_m = −m/2, d_m = (−1)^m μ(m−1/2), 式 (67)) を使い

> **H_EH = 4κ K(κ′) · T**, κ = 1/√(1+μ²), κ′ = |μ|/√(1+μ²) (式 (72))

と**厳密に**与えた (K は第一種完全楕円積分)。結論は「BW 形は gapped 鎖で、分散依存の
非普遍 prefactor を除いて厳密」— v24.4/v24.5 で QRN が実測した「BW 線形**形**は 5 桁で
生存、規格化 λ だけが regulator 物性として走る」の 1D 版と完全に同じ構図である。

本プロジェクトの鎖 (v251 の chain_k, hopping +1/2) とは局所ユニタリ cₙ → (−1)ⁿcₙ で
同値 (staggered 項は不変、ボンド符号のみ反転 — K の勾配絶対値は不変)。

### 1.2 QRN 規格化への翻訳 — g_m ≡ g は定理になる

v25.1 の定義 K_NN = g·πξ + b、(−1)^x K_xx = g_m·2πξ_s·μ + b_m に H_EH = 4κK(κ′)T を
代入すると πg = 2κK(κ′)、2π g_m = 4κK(κ′)。すなわち

> **g(μ) = g_m(μ) = (2/π) κ K(κ′)** — 運動項と質量項の規格化は同一関数 (厳密)。

v25.1 の観測「g_m ≈ g (4–5 桁)」は近似的一致ではなく厳密な同一性だった。
b ≈ 0 (純線形) も Eisler の H_EH に定数項が無いことの帰結。

### 1.3 三表示と漸近形

K(k) = π/(2 AGM(1, √(1−k²))) と AGM の同次性 AGM(ca,cb) = c·AGM(a,b) から:

| 表示 | 式 |
|---|---|
| 楕円 | g = (2/π)·K(κ′)/√(1+μ²) |
| AGM | g = 1/AGM(1, √(1+μ²)) |
| 積分 | g = (2/π)∫₀^{π/2} dθ/√(1+μ²cos²θ) (Legendre 標準形 a=√(1+μ²), b=1) |

漸近形 (v25.1 §4 の開問への解答):

- **小 μ**: g = 1 − μ²/4 + 9μ⁴/64 − 25μ⁶/256 + 1225μ⁸/16384 − … 係数は
  (−1)ⁿ[(2n−1)!!/(2n)!!]²。**μ² の解析級数 (収束半径 |μ|=1) であり、v25.1 が候補に
  挙げた μ²lnμ 型の非解析項は存在しない**。
- **大 μ**: g = (2/(π|μ|))·ln(4|μ|)·(1+O(1/μ²)) — **1/μ² 型でも純 1/μ 型でもなく、
  log 補正つき 1/μ**。
- 分散 E(q) = √(cos²q+μ²) の複素零点から ξ_corr = 1/arsinh|μ|。μ=10⁻³ では
  ξ_corr ~ 10³ ≫ N=256 — v25.1 の小 μ 域が窓に乗らなかった理由も閉じた
  (有限鎖の役割の再定義はコミット III)。

### 1.5 事前登録スクリーニングの決算

設計ノートの粗スクリーニング最良候補 **π/(2K(κ′)) は棄却が確定** (S7-m5 がドリフト
0.16% @μ=0.5 → 3.81% @√2 を再現 — ノートの記録 0.16%→3.8% と一致)。この候補が
「近いのに合わない」理由も構造的に判明した: Eisler (72) より EH スペクトル間隔は
2π K(κ′)/K(κ) = [prefactor 4κK(κ′)] × [T の固有値間隔 π/(2κK(κ))] と**因子分解**され、
スクリーニング候補は後者 (CTM 間隔関数) を測っていた。ボンド勾配 g は前者の
prefactor である。「楕円積分構造 (XY 対応 m = 1/(1+μ²)) の見込み」は的中、
関数の取り違えだけが敗因 — 数秘術ではなく構造推定が正しかった事例として記録。

### 2.2 異方性定理 (λ_⊥ > λ_x の向きは必然)

**定理**: r_⊥ < r_x (すなわち λ_⊥ > λ_x)。

**証明**: BZ 一様測度で Y = cos²ky と Z = cos²kz は独立、E[Y] = 1/2。
h(s) = g(√s) は狭義単調減少 (g′(μ) = −μ⟨cos²θ(1+μ²cos²θ)^{−3/2}⟩ < 0 — §1 S6e で
器械照合済み)。r_⊥ − r_x = ⟨2Y·h(Y+Z)⟩ − ⟨h(Y+Z)⟩ = 2Cov(Y, h(Y+Z))。Z で条件付ける
と E[Y|Z] = E[Y] (独立) より Cov(Y, h(Y+Z)) = E_Z[Cov(Y, h(Y+Z) | Z)]。各 z について
h(Y+z) は Y の狭義減少関数で Y は非退化だから Chebyshev の相関不等式より
Cov(Y, h(Y+z)) < 0。ゆえに r_⊥ − r_x < 0。∎

区間演算はこれを数値的にも**証明**する: r_x − r_⊥ ≥ 0.0301307 (> 0 厳密)、
**λ_⊥ − λ_x ≥ 0.04391**。

**物理的含意**: 異方性 λ_⊥/λ_x = 1.037084… は Lorentz 破れではなく「横チャネルの
重み 2cos²ky と質量依存 prefactor g(μ) の**負の共分散**」— 重みが大きい (cos²ky 大)
ところほど μ が大きく g が小さい — の幾何学的必然。v25.1 の「チャネル重みの幾何」が
向きの符号まで定理に昇格した。

### 2.3 区間演算の設計 (iv.rs — 仮定 ASM-IEEE754 を新規登録)

外向き丸め (1 ulp ナッジ、IEEE-754 の正しい丸めを仮定)・π は区間定数
[PI, next_up(PI)]・cos は剰余項パディングつき Taylor (K=13, 係数 1/(2k)! も区間) —
**libm を信頼しない**。セルの Y = cos²k は cos の単調性で厳密範囲化し、区間 AGM
(任意ステップで hull ⊇ AGM) → 階層和 (行 → 添字順畳み込み — 平坦和は部分和の桁上がり
でナッジが肥大する) → n² = 2³⁰ の 2 冪除算 (正確)。幅は O(h) (モノトーン包 Riemann の
宿命): r_x 7.4e-6 / r_⊥ 5.8e-5。**「証明は区間 (5–6 桁)、15 桁は独立 2 実装の相互照合」
という分業**を明示する。変異検出層: 重み sin² (逸脱 0.060)・係数 2 忘れ (0.407)・
μ² の kz 落とし (0.055) — 全て検出。
