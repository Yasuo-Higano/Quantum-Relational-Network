## 1. 結果 — 判定 (b): ただし構造は厳密に立った

| 測定 | 値 | 予言 | 比 |
|---|---|---|---|
| dQ⁵/dt* (N=12, x=2) | 6.606 | 7.639 | 0.865 |
| dQ⁵/dt* (N=14, x=2) | 7.706 | 8.913 | 0.865 |
| dQ⁵/dt* (N=14, x=4) | 7.564 | 8.913 | 0.849 (帯 15% を 0.1% 超過) |
| **勾配比 N=14/N=12** | **1.1665** | **14/12 = 1.1667** | **0.02% 一致** |
| M/g 動的 (x=2 / x=4) | 0.477 / 0.522 | 0.718 / 0.673 | 0.665 / 0.776 |

- **示量性 (∝ N = ∝ L) は 0.02% で厳密** — アノマリーが体積に比例する構造そのものは
  完璧に立った。外れは N 非依存の乗法係数 ~0.86 に集約される。
- **係数 0.85–0.87**: 素朴な staggered 軸性カレント Q⁵ = −Σ Im⟨c†_n c_{n+1}⟩ の
  格子繰り込み (Z_A 様) + フィット窓中の E の低下。ga ∈ {0.5, 0.71} での 13–15% は
  O(ga) 補正として整合的な規模。登録帯 (±15%) を x=4 の 1 点が 0.1% 超過 —
  **登録どおり (b)** (帯は動かさない)。
- **点毎の法則は非線形周期で崩れる (記録)**: dQ⁵/dt = (2N/π)Z⟨E⟩(t) の Z は
  x=2 で 0.65 ± 0.19 — 一定でない。初期 (断熱域) は 0.86、振幅 1 の周期が
  高運動量対を占有すると素朴カレントは法則から外れる — 格子軸性カレントの
  改良 (improvement) が要る領域を特定。
- **プラズマ振動は非調和に軟化 (記録)**: ω から出る動的質量は v20.1 のスペクトル
  質量の 0.665 (x=2) / 0.776 (x=4)。E₀=1 の振動は対生成閾値 (½ 弦の安定域) を
  跨ぐ大振幅で、線形応答 (ω = M) の外 — 設計時の誤解を登録が刻んだ形。
  軟化が x と共に減るのは相対振幅の減少と整合。
