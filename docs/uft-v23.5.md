# 量子情報網理論 v23.5 — λ∞ 外挿判定: 有限繰り込み 2π/λ∞ の確定、λ∞ = 32/27 (0.006%)

**第二十四期。事前登録 (a′) — 有限繰り込みの確定**。v23.4 の登録設計不足
(単調性では →1 と →λ* を区別できない) を正しい判別基準で再登録し、判定した。

## 1. 方法と結果

一次データ = v23.4 の λ(N) (N = 8..16, DD 全経路)。λ(N) = λ∞ + c/N⁴ の族別
Richardson 外挿:

| 外挿 | λ∞ |
|---|---|
| mod-0 族 (8,12) | 1.18544 |
| mod-0 族 (12,16) | 1.18552 |
| mod-2 族 (10,14) | 1.18472 |

- [J1] 1/N⁴ 自己整合: mod-0 の 2 外挿が **Δ = 0.0001** — スケーリング則が精密成立。
- [J2] 族間整合: Δ = 0.0008 (< 0.003) ✓。
- **λ∞ = 1.18512 (族平均)。32/27 = 1.18519 との比 = 0.99994 (0.006%)。**

## 2. 言明

**連続極限でも格子のエンタングルメント温度は 2π ではなく 2π/λ∞ = 2π·(27/32) —
Bisognano–Wichmann は繰り込まれた温度で回復する** (λ∞ = 1 ± 0.02 の素朴回復は
判別基準の上で棄却)。有理値 32/27 = 2⁵/3³ との 0.006% 一致は staggered
種構造由来の厳密有理因子を強く示唆する — 導出は理論課題 (Lean 化候補)。
