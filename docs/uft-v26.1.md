## 1. 問いと方法

三部作は「独立した 3 つの確認」ではなく一つのモデル開発過程である — 各段階は観測済みの
失敗 (J の構造零・|V_td| の miss・|V_us| の過大) を使って次のモデルを選んだ。段階内の
holdout では**プログラム全体の adaptive model-selection cost** を払えない (PROMPT/7 §6)。
そこで S = Σ_k ln p(D_k | M_{k−1}, D_{<k}) を、**当時凍結されていた模型の事後予測密度**
(当時と同一の evidence 機構を逐語写経し、当時の観測模型 lognormal σ=ln2 を核にした混合)
で評価した。credit の分水嶺は台帳が凍結した「初見表」:

- D1 (質量比 6 + |V_us|,|V_cb|,|V_ub|) = 開始資本 (v3.2 で既見) — 規約により Δ ≡ 0。
- **D_J, D_VTD @ v15.7** — credit は rect T²×T² M_perm (PRED-002/003 の出所) のみ。
  shear の v16.4 holdout hit は既見データの再現であり credit にならない (保守規則)。
- **D_VTS, D_UT2 (β, γ) @ v16.5** — credit は v16.4 凍結の shear (1,1) のみ。

ベースライン: アナーキー (o1 複素係数・電荷ゼロ) と FN (v3.2 文献電荷 × ε=0.22) —
いずれも MC 10⁶ 標本 (シード固定、半割差 max 0.051 nats)。

## 4. 読み方 — 何が壊れ、何が生き残るか

1. **支配項は D_J**: v15.7 までのプログラム (矩形族 + 傾き T⁴ まで含めた試行集合 23 種の
   一様混合でも上界 −45.4) は CP を**原理的に**作れなかった。複素位相を持つ両ベースラインは
   作れた — 特に **FN (v3.2) は |J| をほぼ的中させる (+8.53; 最大可能 +9.83)**。
   Wolfenstein スケーリングを自然に持つ FN に対し、実 Wilson 線の幾何は CP の分だけ
   構造的に遅れて出発していた。
2. **J を除いても対 FN は net 負 (−1.96)**: 幾何は |V_ts| で辛勝 (+0.28)、γ で負け
   (−1.56)、|V_td| でも僅差負け (−0.68)。「乱数係数を消した」ことの prequential な代金は
   まだ回収されていない。対アナーキーでは J 以外 +7.04 — 階層を作れること自体の価値は
   実在する。
3. **これは三部作の段階内証拠を無効にしない**: +306 nats (J 導入後の再戦) や 12/12
   スコアカードは「同時代データへの適合と模型比較」として正しい。本監査が言うのは、
   **プログラム横断の「予測」の台帳はまだ赤字**だということ — 各論文の holdout が
   払っていなかったコストの、これが総額である。
4. **生き残るもの**: (i) 収束点 τ = 1/12 + i/2 はまだ新鮮データで試されていない —
   将来の観測量更新 (質量比の格子 QCD 更新・PMNS 拡張) が最初の真のテストになる。
   (ii) 乱数係数ゼロという構成的性格 (FN は 18 個の O(1) 複素乱数を周辺化している)。
   (iii) predictions.yml の機構そのもの — PRED-002/003 が正直に採点されていたからこそ
   本監査の分水嶺が引けた。

## 5. 限界

- 核幅 (R1/R4) は当時規約 lognormal σ=ln2 を全観測量に一律適用 — UT 角の観測誤差
  (PDG の ±1° 級) を使えば全モデルの β/γ 密度は大きく動く。ただし**差分** (program −
  baseline) は核幅感度 [ln1.5, ln3] で符号不変 (表 1)。
- D_J の bracket は「試行集合の最良メンバー (傾き T⁴)」への一様混合 — M1 の各メンバーの
  真の |J| 事後を個別に積分してはいない (上界としてのみ有効。strict は −∞)。
- FN ベースラインの電荷は文献値 = 当時の観測を知る文献の選択 — FN 側にも隠れた
  選択コストがある (対アナーキー比較がその下限を与える)。
- 台帳の保守規則 (既見 → credit なし) の緩和側 (v16.4 hit を credit) を採ると
  D_VTD は shear の +3.92 になるが、総額の符号は変わらない (J が支配)。
