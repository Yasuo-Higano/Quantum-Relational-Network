## 2. 装置 — 二層構造 (PROMPT/3 §7 の最小セット)

- **探索層** (python/numpy, 172 行): 2 サイト DMRG。電場 MPO は走和構成
  W = [[I, q, cq²+mass],[0, I, 2cq],[0,0,I]] ⊕ ホップキャリア (D=5)。
  自己検定内蔵: (i) MPO 縮約 = 独立直接構成の H (偏差 0.0 — MPO の代数を機械検証)、
  (ii) DMRG = ED @ N=8 (1.8e-14)。
- **監査層** (Rust std): 開鎖は巻き付き ε が無く**切断なしで ED が厳密** (N=16 で
  dim 12,870) — 同一幾何アンカーを自前計算し JSON と照合 (一致 ≤ 2e-14 —
  χ=96 の DMRG は N=16 で実質厳密)。
- 開発記録: 初走は (i) 右正準初期化のボンド次元の左右取り違え、(ii) 環境テンソルの
  bra/ket スロット混同 — 前者は形状例外、後者は**自己検定 (DMRG vs ED) を最初の
  関門に置いた設計**が捕捉する構え (実際は同時修正)。探索層にも PASS/FAIL 文化を
  そのまま適用した初例。
