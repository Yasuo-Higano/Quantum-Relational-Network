   4 次元から見ると「電荷 n を持つ質量 m_n ≈ (2π/N_y)·n の粒子の梯子」。±n は粒子/反粒子。

3. **ゲージポテンシャル = 隠れ次元のねじれ**: Wilson 線 α を入れるとスペクトルが
   m²_n = 2−2cos((2πn±α)/N_y) へ移動(8×10⁻¹⁵)— 各状態は**自分の電荷 n に比例して**応答
   = ミニマル結合 qA の幾何的起源。
4. **ゲージ不変性 = 隠れ次元の座標変換**: ねじれの局所配分を乱数で変えても(Σθ 固定)
   スペクトル不変(3×10⁻¹⁵)— ホロノミーだけが物理(v0.4 の教訓が純幾何から再演)。
5. 分散関係の分離 ω² = 4sin²(k_x/2) + m²_n を直接検証(7×10⁻¹⁵)。
