## 2. Depth の反転の解剖 — 尾の切断

v15.5 で最悪だった原理が最良になった。per-geometry の内訳が理由を示す:

| 幾何型 | 一様 | Depth β=.25 | 差 |
|---|---|---|---|
| 対角 (2,2)/(3,3) | −22.3 / −21.8 | **−514.9 / −514.6** | v15.5 の rect と同じ壊滅 |
| 非対角 (1,3)/(1,2)/(2,3) | −21.98 / −22.00 / −22.26 | **−20.10 / −20.11 / −20.35** | **+1.9 nats 改善** |

- **対称幾何 (rect 含む) では深さの尾が観測を突き抜ける** — 最深 config へ集中する
  事前は自滅する (v15.5 の機構が対角でそのまま再現: −514.9 vs v15.5 の −514.4)。
- **非対称幾何では深さの尾が観測の近くで切れる** — 二つの異なるモード表の積は
  極値が揃わず、族の最深 config が測定された階層の近くに座る。そこへ集中する
  事前は、一様が浅い config に浪費する重みを回収して +1.9 nats 勝つ。
- 解釈 (機構は未証明): 「深いほど良い」は、**深さに構造的な上限を持つ模型族の中で
  だけ正しい物語**になる。原理の当否は模型族に強く条件付き — これが本版の最大の教訓。

## 3. 谷底の裁定 — 破れる、ただし向きは原理ごとに違う

事前登録の基準 (対差 ≥1 nat) は MDL (首位 (2,2))・Robustness (λ で (3,3)/(1,3) が
入れ替わる)・Depth (首位 (1,3), 対差 494) が満たした。しかし:

- **棄却された原理の裁定に資格はない** (MDL・Robustness)。Robustness は自分の
  λ 変種間で首位が割れており、裁定器として不安定。
- **生存した唯一の原理 Depth は (1,3) を選ぶ** — 対角 (2,2)/(3,3) を −494 で消し、
  (2,3) を −0.26 に置き、(1,3) を首位にする。Thermo (未決) の弱い選好は (3,3) 向き
  (対差 0.5 — 破らない)。
- **規律ノート**: 事前登録は「破る」の判定基準だけを固定し、「複数原理が違う向きに
  破ったときどれを採るか」は固定していなかった。「生存原理のみが裁定資格を持つ」は
  自然だが**事後の解釈**である — この選好を使う下流の版は、(1,3) を「Depth 生存下の
  作業仮説」と呼ぶこと (証拠単独の選択ではない)。

## 5. 残高への効き

- **上位原理の残高が動いた**: v15.5 の「生存 0」から「**生存 1 (Depth)・符号再現 1
  (Thermo)**」へ。σ・Wilson 線が既約な離散データであることは変わらないが、
  **幾何 (対称/非対称) は Depth 原理で選べる可能性**が開いた — 証拠 (v16.10) が
  縮退させたものを、データを見ない原理が破るという、選択原理に期待されていた
  仕事そのものである。
- Depth の生存は同時に**対称幾何の死刑宣告** (−494) を含む — もし将来の証拠が
  対角を強く選び直したら、Depth は棄却に戻る (反証可能性が具体的な形を持った)。
- Thermo の +0.3〜+0.5 は 2 つの窓 (rect 9 量・シアー 10 量) で符号・大きさとも
  安定 — 「弱いが消えない正信号」として、一ループ真空エネルギーへの置き換え
  (v15.5 の限界) が引き続き残る。
