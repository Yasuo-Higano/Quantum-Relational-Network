# 量子情報網理論 v25.1 — λ の起源: 1D 質量関数 g(μ) への厳密還元

**第二十六期第 1 版。判定 (a) — λ の起源が確定**。v25.0 の残高 1
(λ_x = 1.185468 / λ_⊥ = 1.229430 の解析的導出) の器械的前半が閉じた。

## 1. 厳密還元 (バンド分解 + 荷電共役)

ブロック理論 (v24.1/24.2) の帰結として、横ブロック (ky, kz) の 2 バンドは
回転基底で**質量 ±μ (μ = √(cos²ky + cos²kz)) の 1D staggered 鎖 2 本に厳密分解**
する。荷電共役 c → (−1)^x c† は μ ↔ −μ を結び K(−μ) = −ΣK(μ)Σ (numpy 独立検算
4e-16)。ゆえに 3D の λ は 1D 鎖の 2 つの regulator 関数の BZ 平均に還元される:

- **1/λ_x = ⟨g(μ)⟩_BZ** — g(μ) = 鎖のモジュラー核 NN ボンド勾配 / πξ
- **1/λ_⊥ = ⟨2cos²ky·g_m(μ)⟩_BZ** — g_m(μ) = 鎖の質量チャネル対角勾配 / 2πξμ

## 2. 検証 (v251_gmu, 7 検査 PASS)

- [S3] **厳密和則** (N=32, 認証窓 ξ≤5): 3D 半空間スキャン (A_x/K_y/B_z) =
  1D 鎖和、窓内 max|Δ| = 4.5e-7 / 1.1e-7 / 1.2e-7 (床はクランプ境界モードの
  DD 分解能 — 全域診断 ~3e-2 はクランプ雑音域)。
- [S4] **連続再構成**: μ 格子 78 点の g/g_m + BZ 求積 (M=512 収束) →
  **λ_x = 1.185472 (目標との相対差 3.7e-6) / λ_⊥ = 1.229427 (2.5e-6)**。
- g(0) = 0.9990 (v24.5 XX の λ=1 再現)、全 μ で ξ 線形性 (残差 < 7e-5) —
  massive 1+1D でも BW 線形形は成立し、規格化 g(μ) だけが走る。

g(μ) は単調減少: g(0)=1 → g(0.5)=0.945 → g(1)=0.8346 → g(√2)=0.7457。
g_m(μ) ≈ g(μ) が 4–5 桁で成立 (μ ≥ 0.12 の測定域) — **運動項と質量項の
規格化が同一関数**である示唆 (v25.2 の解析形探索の強い手がかり)。

## 3. 器械の欠陥修正 (開発記録 — 重要)

S3 初版の O(1) 破れから **modular_k の片側クランプ欠陥**を発見: 1 側の判定
`c > 1.0 − clamp` は f64 で 1.0−1e-30 が 1.0 に丸まり**永遠に発火しない** —
深く占有されたモード (1−c < DD 床) の κ が雑音化し、切断から遠い深部ボンドを
O(1) 汚染していた。**両側とも T 精度の距離判定に修正** (dd.rs)。
v24.3/v24.4 の再走で**全公表数値が 5 桁不変**を確認 (窓 ξ≤6 には指数的に
届かないため — 変わったのは雑音レベルの診断値と、両側を感知するようになった
梯子の深部感度のみ)。ゲート較正の教訓 (v24.3 W1/W2 と同型) も再度記録。

## 4. 帰結と次

- λ_x, λ_⊥ の「導出」は **1D 鎖 1 本の関数 g(μ) の解析形**に帰着した (v25.2)。
  漸近ヒント: 小 μ で g−1 ∝ μ²lnμ 型か、大 μ 減衰は 1/μ 型かのフィットが次の仕事。
- 異方性 λ_⊥/λ_x の起源も自明化: 同じ g でも**チャネル重み** (運動項は一様平均、
  質量項は 2cos²ky 重み) が違うから — Lorentz 破れではなく重みの幾何。
- g_m ≈ g が厳密なら 1/λ_⊥ = ⟨2cos²ky·g(μ)⟩ — 単一関数のモーメント 2 つに帰着。
