**Version**: v0.0(調査編)

**Status**: 基礎調査 — 以後の全バージョンの土台

## 0. 目的

インドの寓話では、盲人たちが象の脚・鼻・耳・尾をそれぞれ触り、「柱だ」「蛇だ」「団扇だ」「縄だ」と言い争う。
現代物理学もこれに似る。一般相対性理論・量子力学・統計力学・標準模型は、**それぞれの適用範囲では驚異的な精度で正しい**。にもかかわらず互いに矛盾し、全体像(象)が見えていない。

本文書では、まず各理論が「何を解決し」「何を解決できていないか」を整理する。
その上で、諸理論が**異口同音に指差している方向**を抽出し、以後のバージョンで数値実験によって検証していく作業仮説を立てる。

## 1. 確立された理論たち — それぞれが触った「象の部位」

### 1.1 ニュートン力学+万有引力(1687)

| | |
|---|---|
| **解決したこと** | 天上と地上の力学の統一(リンゴと月は同じ法則で落ちる)。惑星軌道、潮汐、彗星回帰の予言。 |
| **残した問題** | 水星近日点移動の残差 43″/世紀。遠隔作用の機構不明。光速不変(マイケルソン=モーリー)と非両立。絶対空間・絶対時間という検証不能な前提。 |
| **教訓** | 「別々に見えた現象(天と地)が一つの法則の二側面だった」— 統一の原型。 |

### 1.2 電磁気学 — マクスウェル(1865)

| | |
|---|---|
| **解決したこと** | 電気・磁気・光の**三位一体の統一**。場の概念の導入(遠隔作用の排除)。電磁波の予言。 |
| **残した問題** | 方程式がガリレイ変換で不変でない → エーテル問題 → 特殊相対論を強制。点電荷の自己エネルギー発散(のちの QFT の発散問題の萌芽)。 |
| **教訓** | 統一は新しい実在(場)と新しい対称性(ローレンツ不変性)を要求する。 |

### 1.3 特殊相対性理論(1905)

| | |
|---|---|
| **解決したこと** | 空間と時間を「時空」に統一。E=mc²(質量とエネルギーの統一)。因果構造(光円錐)の確立。 |
| **残した問題** | 重力を含められない(瞬時の遠隔作用は因果律違反)。加速系の扱い。 |

### 1.4 一般相対性理論(1915)— 象の「重い脚」

**核心**: 重力は力ではなく**時空の幾何**である。G<sub>μν</sub> = 8πG T<sub>μν</sub>。物質が時空を曲げ、曲がった時空が物質の運動を決める。

| | |
|---|---|
| **検証済み** | 水星近日点移動 43″/世紀(→ **v0.1 で数値再現**)。光の偏向 1.75″(1919 エディントン)。重力赤方偏移、Shapiro 遅延、GPS の相対論補正(±38μs/日)、連星パルサー、重力波(GW150914, 2015)、ブラックホール撮像(EHT, 2019)。 |
| **未解決** | **(a) 特異点**: Penrose–Hawking 定理により、理論自身が自らの適用限界(曲率無限大)を予言する。**(b) 量子化不能**: G の質量次元が −2 のため摂動論的に繰り込み不可能。**(c) ダークセクター**: 銀河回転曲線・宇宙加速膨張を説明するには未知の物質・エネルギーが 95% 必要。**(d) 情報パラドックス**: ブラックホール蒸発でユニタリー性が破れるように見える(→ v0.8)。 |
| **触った部位** | 「時空は動的で、物質と対話する幾何である」 |

### 1.5 量子力学(1925–27)— 象の「揺らぐ鼻」

**核心**: 状態はヒルベルト空間のベクトル。観測量は演算子。時間発展はユニタリー(シュレーディンガー方程式)。測定は確率的(ボルン則)。

| | |
|---|---|
| **解決したこと** | 原子の安定性(古典論では電子は 10⁻¹¹ 秒で墜落)。スペクトル線、周期表、化学結合、トンネル効果(→ **v0.1 で数値再現**)、超伝導・半導体(現代文明の物質的基盤)。 |
| **未解決** | **(a) 測定問題**: ユニタリー発展と波束の収縮はいつ・なぜ切り替わるのか(コペンハーゲン/多世界/GRW/QBism — 実験的に未決着)。**(b) 古典極限**: なぜマクロ世界は重ね合わせを見せないか(デコヒーレンスは部分的回答)。**(c) 重力との整合**: 「時空の重ね合わせ」を記述する枠組みがない。 |
| **触った部位** | 「実在は状態ベクトルと確率であり、情報が基本量である」 |

### 1.6 場の量子論と標準模型(1948–1975)— 象の「精密な耳」

**核心**: 粒子は場の励起である。力は**局所ゲージ対称性** SU(3)×SU(2)×U(1) の帰結である。

| | |
|---|---|
| **解決したこと** | QED: 電子異常磁気モーメント g−2 が実験と **12 桁一致**(物理学史上最高精度)。電弱統一: W/Z ボソンの質量を発見前に予言。QCD: 漸近的自由(2004 ノーベル賞)とクォーク閉じ込め(→ **v0.4 で格子ゲージ理論により再現**)。ヒッグス機構: 2012 年に 125 GeV で発見。反物質、ニュートリノ振動(拡張後)。 |
| **未解決** | **(a) 自由パラメータ 19+ 個**(質量・混合角・結合定数 — なぜこの値か説明なし)。**(b) 階層性問題**: ヒッグス質量はなぜプランク質量の 10⁻¹⁷ 倍か。**(c) 強い CP 問題**。**(d) ニュートリノ質量の起源**。**(e) 3 世代の理由**。**(f) 真空エネルギー**: QFT の見積りは観測値の **10¹²⁰ 倍**(物理学史上最悪の予言)。**(g) 重力が入らない**。**(h) ダークマターの候補なし**。**(i) バリオン非対称の起源**。 |
| **触った部位** | 「力とは内部空間の幾何(接続)であり、対称性が動力学を決める」 |

### 1.7 統計力学・熱力学 — 象の「普遍的な胴体」

| | |
|---|---|
| **解決したこと** | エントロピーの微視的定義 S = k ln W。相転移と臨界現象。**くりこみ群**(Wilson): ミクロの詳細はマクロの普遍性に洗い流される — 「連続体の場の理論は臨界点の影」(→ **v0.3 で再現**)。 |
| **深い橋** | **ブラックホール熱力学**: S = A/4ℓ<sub>p</sub>²(Bekenstein–Hawking)。エントロピーが体積でなく**面積**に比例する — 重力・量子・熱・情報の四叉路(→ v0.5)。Unruh 効果、Hawking 温度。 |
| **未解決** | 重力系の統計力学の基礎(BH エントロピーは**何の**自由度を数えているのか)。時間の矢の起源(なぜ初期宇宙は低エントロピーか)。 |
| **触った部位** | 「マクロな法則はミクロの詳細に依らない。熱・情報・幾何は換算可能である」 |

### 1.8 宇宙論 ΛCDM(1998–)

| | |
|---|---|
| **解決したこと** | CMB 音響ピークの精密フィット、ビッグバン元素合成(軽元素比)、宇宙の大規模構造形成、膨張史。わずか 6 パラメータ。 |
| **未解決** | ダークマター 27% + ダークエネルギー 68% = **宇宙の 95% の正体が不明**。インフレーションの機構と初期条件。初期特異点。ハッブル定数の測定間の緊張(H₀ tension)。 |

## 2. 未解決問題の地図 — どの理論の「境界」で起きているか

| 問題 | 衝突している理論 | 手がかり |
|---|---|---|
| 量子重力(時空の量子化) | GR ↔ QM | BH 熱力学、ホログラフィー |
| 真空エネルギー / Λ 問題(10¹²⁰) | QFT ↔ GR ↔ 宇宙論 | 因果集合論の Λ 揺らぎ予言 |
| BH 情報パラドックス | GR ↔ QM のユニタリー性 | Page 曲線、島公式(2019–) |
| 測定問題 | QM ↔ 古典世界 | デコヒーレンス、情報理論 |
| ダークマター/エネルギー | 宇宙論 ↔ 標準模型 | 未発見粒子 or 重力の修正 |
| 階層性・パラメータ問題 | 標準模型の内部 | 対称性?人間原理?力学的選択? |
| 特異点(BH・ビッグバン) | GR の内部崩壊 | 量子効果による解消? |
| 時間の矢・時間の本性 | 熱力学 ↔ GR ↔ QM | 熱時間仮説、エンタングルメント |

**観察**: ほぼすべての境界問題に「エントロピー・情報・エンタングルメント」が顔を出す。これは偶然ではない、というのが本プロジェクトの直感である。

## 4. 象の輪郭 — 諸理論が異口同音に指す方向

上の表を横断すると、**独立に発展した複数のアプローチが同じ結論に収束している**箇所が見える:

1. **幾何 = 情報**: BH エントロピー(GR+QM+熱力学)、RT 公式(弦理論)、Jacobson の導出(熱力学)、面積演算子(LQG)— 全員が「**面積はエントロピーである**」と言っている。
2. **連続時空は創発的**: くりこみ群(場の理論)、DT/CDT(格子重力)、因果集合(離散因果)、弦の最小長 — 全員が「**滑らかな時空は近似である**」と言っている。
3. **量子と熱の同型**: Wick 回転(QFT⇔統計力学)、Unruh/Hawking 温度(加速⇔熱)、KMS 状態・熱時間仮説 — 「**時間・温度・エンタングルメントは同じものの三つの顔**」。
4. **ゲージ原理の普遍性**: 標準模型は接続の幾何、GR も(ローレンツ群/微分同相の)接続の幾何 — 「**すべての力は幾何である**」。
5. **ユニタリー性は守られる**: Page 曲線の島公式による導出は「情報は失われない」側に軍配を上げつつある。

### 作業仮説(本プロジェクトが検証していく主張)

> **時空・重力・ゲージ場・物質は、単一の「量子情報網」(pre-geometric な量子多体系)の異なる読み出しである。**
> - 距離とは相関(相互情報量)の減衰である
> - 面積とはエンタングルメント・エントロピーである
> - 重力とはエンタングルメントの熱力学(第一法則)である
> - ゲージ場とは記述の局所冗長性である
> - 粒子とは網の集団励起である
> - 時間とは熱的・モジュラー流である

この仮説は一足飛びには検証できない。そこで **v0.1 から v1.0 まで、各構成要素を Rust シミュレーションで一つずつ数値的に確かめながら**、理論を段階的に組み上げる。計画は `01-roadmap.md` を参照。

## 5. 方法論についての注意(知的誠実性)

- 本プロジェクトで作るのは**トイモデル(縮小模型)**である。トイモデルは「機構が原理的に働くこと」を示すのであって、「我々の宇宙がそうであること」を証明するのではない。
- 各バージョンの文書では、**確立された物理**(教科書レベル)、**確立された研究成果**(査読論文レベル)、**本プロジェクトの仮説**(検証対象)を区別して明記する。
- シミュレーションには可能な限り**厳密解・実測値との比較**を含め、コードの正しさを検証可能にする。
