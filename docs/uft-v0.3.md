## 3. 考察 — 標準模型の設計原理はすべてここにある

1. **粒子とは場の励起である。** 格子上の φ の集団振動(相関の波)が「粒子」。
   粒子の個数は保存しない(場の量子論が粒子の生成消滅を扱える理由)。
2. **質量とは真空の選択である。** 破れ相では φ は ±v のどちらかに落ち、その周りの小振動が
   質量を持つ粒子になる。ヒッグス機構(W/Z ボソンの質量、2012 年のヒッグス発見)の原型。
3. **連続時空(場理論)は臨界現象である。** 格子間隔 a の理論で相関長 ξ/a → ∞ とする極限が
   連続場理論の定義そのもの。**「連続」は基本ではなく、臨界点近傍の見え方である。**
   → これは統一仮説の中心命題「滑らかな時空は創発的」の場の理論側からの証拠。
4. **普遍性: ミクロの詳細は消える。** φ⁴ と Ising が同じ臨界理論に流れる事実は、
   「プランクスケールのミクロ構造が何であれ、低エネルギー物理は少数の対称性と次元だけで決まる」
   ことを意味する。**だから我々は、ミクロの正体を知らないまま低エネルギー物理(=現実の全物理学)を
   もっているのだし、逆にミクロの理論(統一場理論)は低エネルギー実験から一意に逆算できない。**
   統一理論の検証がなぜ困難かの、くりこみ群による説明。
