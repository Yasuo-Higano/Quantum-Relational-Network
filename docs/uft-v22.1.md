# 量子情報網理論 v22.1 — 重電荷窓の開通: M(q)/g = q/√π の格子検証

**第二十三期 課題 3。事前登録 (a) 成立** — 電荷 q のフェルミオンを持つ massless Schwinger
模型で、ボソン質量比 **M(2)/M(1) = 2.1001** (x=16, N=64) が登録窓 2 ± 7% に着地。
v21.1 で環のフラックス状態に阻まれた重電荷窓が、開鎖 + 全電荷罰則 + 二層構造
(Rust/python) × 三推定器で開いた。ボソン化の教科書線 M(q)/g = q/√π の比が
格子第一原理から出る。

## 1. 位置づけ

v21.1 (環 ED) では巻き付き ε のフラックス状態が低エネルギー帯を占拠し q ≥ 2 の
ボソンが読めなかった。開鎖は ε を持たない (厳密) — が、**開鎖には開鎖の寄生状態が
あった** (§3)。本版はその同定と除去を含む、計器開発の完全な記録である。

## 2. 器械 (最終形)

- 模型: H = −x Σ hop + Σ_pairs w(k,l) q_k q_l, w = N−1−max(k,l) + λ_Q,
  q_k = qf·(n_k − [k odd]), λ_Q = 10。
- 計算層 `v221_dmrg.rs` (Rust, PROMPT/4): 2 サイト DMRG + MPO 走和構成 (D=5)。
  決定的 6 スレッド (アンカー 2 + 本測定 4)。numpy 版の ~20 倍。
  直交罰則法で E₁、接続電荷相関 G(r) の深プラトー有効質量 (r=20) で第三推定。
- 監査層 `v221_dmrgex.rs`: N=10 全空間 ED (dim 1024, セクター仮定なし) との照合。
- ゲート 12 本: [G1-G4] DMRG = ED (N=8/10, Δ ~ 1e-13), [G5] E₀ 交差エンジン,
  [G6] 対整合 E₁ ≥ E₀ (stuck の自己検出), [G7] 直交度, [G8] 相関質量 = gap 質量 ±20%,
  [I1-I3] 監査層 ED 照合 + 罰則の中性不変性 (厳密恒等式), [I4] q=1 E₀ 交差エンジン。

## 4. 開発記録 — 寄生状態と計器較正の系譜 (全走行保存)

1. **run1 (λ_Q = 0, 判定 b)**: E₁ が ⟨n⟩ = 4.000 の荷電状態 — **開鎖端の電荷は
   どのリンクにも寄与せず自由** (環の磁束状態の開鎖版)。中性ギャップは 3〜8 倍上。
2. **修正 = 全電荷罰則 λ_Q(Σq)²** (ペア重み +λ_Q の一行)。中性スペクトルは厳密不変 —
   この恒等式自体を監査ゲート [I3] にした。
3. **Rust 移植の較正 10 走** (PROMPT/4「重い計算は Rust」— numpy は einsum 割当てが
   律速で 20h/対): (i) θθᵀ Gram の σ 除算が正準性破壊 → 射影式 (Uᵀθ/θV)、
   (ii) 乱数初期化の盆地問題 → Néel + x アニーリング、(iii) 温間 E₁ の Krylov 崩壊
   (ψ₀ が厳密固有だと Krylov が 1 次元に潰れ E₀+λ_o を返す)、(iv) 局所 Ritz 値は
   環境破損時に**非変分の偽低値**を隠す → 大域 Rayleigh 商への誠実化 (変分下界
   違反が原理的に不可能に)、(v) Gram 打ち切りの尾部破壊 (σ < 1e-8 が固有値 2 乗の
   床で壊れる) → **Hestenes 一側 Jacobi SVD** で解決 — E₀ が python と Δ3.6e-10。
4. **q=1 の E₁ は密な多ボソン梯子で罰則法が降下しない** (Rust 変分上界どまり・
   python は 31h で収束)。→ 第三推定器 (E₁ 不要の相関質量) を [G8] で検証して併置。

## 5. 正直な限界

- 絶対質量は +15% (格子 a・箱・端の系統込み) — 主張は**比**のみ。
- q ≥ 3 とクォーク模型の共鳴窓は未着手 (named limit 継続)。
- Rust の罰則法 E₁ は q=1 の密スペクトルで実用にならない (相関法が正道)。
