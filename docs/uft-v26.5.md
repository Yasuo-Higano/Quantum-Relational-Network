# 量子情報網理論 v26.5 — spin-2 分解: graviton チャネルの c₁ も bare 量である

**第二十七期第 5 版。判定 (a) — 6 検査 PASS。**
静的応答のテンソル構造 (誘導重力 10 要件の (7)) へ: q ∥ ŷ の横方向指数 {x, z} で
**plus 偏極の graviton チャネル** D = (T_xx − T_zz)/√2 と横 spin-0 チャネル
S = (T_xx + T_zz)/√2 を組んだ。T_xy 型の奇運動量頂点 (staggered の taste 構造に絡む
point-split 問題) を回避し、**v26.4 で認証済みの演算子だけ**で spin-2 に到達する設計。
新規の器械は交差相関 χ_AB のみで、χ_D = (χ_xx + χ_zz)/2 − χ_xz、χ_S = 同 + χ_xz。

## 1. 器械

- [S0a] 交差相関 (χ_xx, χ_zz, χ_xz)(qŷ) の内蔵 dense 照合: **3.6e-14** (N=8, 折返し込み)。
- [S0b] x↔z 対称回帰: c₁[xx](ŷ) = c₁[zz](ŷ) = v26.4 の横値 (−0.01556/−0.01264) と
  4.1e-6 — 等方性 (v26.4) の第三の実証。
- [S1] 正値性 χ_D, χ_S ≥ 0 (min 3.8e-2) — 交差項の Cauchy–Schwarz 整合。
- [S4] 変異検出 (T_zz の折返し落とし → 2.5e-2 逸脱を dense 照合が検出)。
- 交差相関は χ_xz(0) = −0.0523 (m=0) — 横応力どうしは**負に相関**する。

## 2. 主結果 — c₁ の偏極分解 (N=64; 窓系統 ≤0.7%, N 移動 ≤1.4%)

| m | c₁[xx](ŷ) = c₁[zz](ŷ) | **c₁^(2) (graviton plus)** | c₁^(0) (横トレース) |
|---|---|---|---|
| 0 | −0.01556 | **−0.01935 (±0.00014)** | −0.01177 |
| 0.5 | −0.01264 | **−0.01444 (±0.00001)** | −0.01083 |

- **graviton チャネルの運動項係数は非零で精密に測れる** (|c₁^(2)| は系統の ~10³ 倍)。
- **質量走行 25.4%** (系統 0.7% の 36 倍) — **Sakharov 係数の非普遍性 (v26.3) は、
  重力を作るはずのそのチャネルで最も強く成立する**。これで応答関数三部作が閉じた:
  縦 (+0.01595, 走行 12.6%) / 横 (−0.01264) / spin-2 (−0.01444, 走行 25.4%) /
  横 spin-0 (−0.01083) — 全チャネルが O(10%) で regulator の中身 (質量) に依存する。
- チャネル空間の構造: 縦は正・横系は全て負。等方弾性の 2 独立係数 + 交差相関
  (χ_xz < 0) が spin-2/spin-0 の分解を決めている。

## 3. 解釈の限界 (正直な線引き)

c₁^(2) < 0 (χ_D が小 q で減少) を「ghost」と読むのは早い: 有効作用の運動項の符号は
静的感受率の q² 傾きだけでは決まらず、(i) 接触項 (χ(0) の regulator 部分) の分離、
(ii) ω 依存性 (動的 Ward・pole 構造)、(iii) 完全なテンソル分解 (縦・横・trace の混合)
を要する。本版の主張は「**graviton チャネルの静的 q² 係数は有限・可測・非普遍**」まで。
符号の物理は v26.6+ (ω 分解) の登録課題。
