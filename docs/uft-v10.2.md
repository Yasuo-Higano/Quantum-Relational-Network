# 統一場理論 v10.2 — 対込みの幾何選択: それでも 3 枚のトーラスが勝つ

**Version**: v10.2
**Date**: 2026-07-04
**シミュレーション**: `sim/src/bin/v102_geopair.rs` → `results/v102_geopair.txt` (機械可読: `results/v102_geopair.json`)
**対応残高**: v10.1 が幾何選択 (残高 9) に付けた条件の解除

---

## 1. 問題

v10.1 は「T²+自由対 (lnZ₉ −19.86) が T³+対角対 (−21.84) に勝つ」ことを示し、
v8.1/v9.1 の幾何選択の結論を宙吊りにした。決着には T³ にも対の自由度を与えた
公平な比較が要る。しかし T³ の全 9 量は (K_Q,K_u,K_d,σ_u,σ_d) の五重和 ~10¹³ 通りで
直接和は不可能。

## 2. 方法 — 証明付き打ち切り

(k_u,σ_u) と (k_d,σ_d) のリスト (各 216×36 = 7776 項) を質量尤度で降順に並べ、
llu+lld ≥ max−Δ (Δ=35 nats) の対だけ CKM を計算する。残りの寄与は CKM 対数尤度が
各観測で ≤ norm1 (密度の最大値) であることから

$$\sum_{\text{残り}} e^{ll_u+ll_d+ll_{CKM}} \le e^{3\,\text{norm1}}\Big(S_u S_d - \sum_{\text{計算済}} e^{ll_u+ll_d}\Big)$$

で**厳密に**上から抑えられる (S_u, S_d は全項の和 — 質量尤度は全て手元にあるので
右辺は正確に計算できる)。lnZ は区間 [下端, 上端] で報告し、幅 < 0.01 を PASS 条件と
した。実際の幅は **~10⁻¹⁰** (事実上厳密)。

## 3. 検証 (装置) — 7 回帰全て一致

| 回帰 | 結果 |
|---|---|
| T¹ 質量 / 全 9 量 (対なし; v9.2) | −53.7694 / −63.5703 厳密一致 [PASS×2] |
| T² 対角 質量, T³ 対角 質量・全 9 量 (v9.2) | 厳密一致 [PASS×3] |
| **T² 自由対 質量・全 9 量 (v10.1 の厳密五重和)** | −15.0839 / −19.8633 一致 [PASS×2] — **打ち切り実装と全和実装の相互検証** |

## 4. 結果 — 対を marginalize しても T³ が勝つ

| 幾何 | 質量のみ | 全 9 量 |
|---|---|---|
| T¹ | −53.77 | −63.57 |
| T² 対角 | −18.76 | −23.61 |
| T² 自由対 | −15.08 | −19.86 |
| T³ 対角 | −16.49 | −21.84 |
| **T³ 自由対** | **−14.11** | **−17.89** (区間幅 8×10⁻¹⁰) |

- **質量のみ: T³ が勝つ** (T³−T² = +0.98)。Occam 罰 4ln36 = 14.33 を払ってなお。
- **全 9 量: T³ が勝つ** (T³−T² = +1.97、区間で確定)。
- 対の自由度はどちらの幾何も同程度持ち上げる (T² +3.7 / T³ +3.9) — 幾何の序列は
  変わらない。

## 5. 意味 — 二重に頑健になった選択

「余剰次元の数はデータから選べ、答えは 3 枚」という v8.1 以来の結論は、これで

1. **ラベル規約に頑健** (v9.2: wrap 綱渡りの両側で勝者不変)
2. **世代対の仮定に頑健** (v10.2: 対を S₃ の全自由度で marginalize しても勝者不変)

になった。v10.1 で宙吊りになった残高 9 の条件は解除。模型族の現在の最良は
**T³ + 自由対で lnZ₉ = −17.89** — 対角対の v9.2 値 (−21.84) から 4 nats 改善した。
