## 2. 第十九期の物語 — 地図の外を指す矢印

第十八期はクォーク・セクターで「届くが典型でない」を τ = 1/12 + i/2 の発見が
「典型」に変えた。第十九期はレプトンで同じ道を歩き、**違う場所に出た**。

最小シーソーの外れは Majorana 側を正しく指し (v18.3)、プロファイルは全量を単調に
改善し (v18.4)、しかし θ23 の壁は「局在ゼロモード × 任意スカラー」の構成そのもの
に根ざすことが機構レベルで特定され (v18.5)、到達可能集合は測定点を含む (v18.6 の
針・61,900 config) のに、**どの走査座標も事前質量をそこへ集めない** (v18.7)。
クォークで τ が埋めた「届く」と「選ばれる」の隔たりが、レプトンでは**現在の
座標系の外** — ν_R の局在半径を変える幾何 (高磁束 Q_ν ≳ 2π n_x²/面積 + 射影) —
を指している。二つの独立の経路 (no-go の機構 [v18.5] と針の解剖 [v18.7]) が同じ
出口に収束したのが、この期の科学的な形である。

もう一つの収穫は誠実性の側にある: 収束点の連続極限検証 (v18.2) は 12/12 の質的
成功を保存しつつ、「−18.43」の一部が格子の追い風であり **m_c/m_t が格子依存**
であることを暴いた — 最良の結果ほど厳しく検証する規律が機能した。
