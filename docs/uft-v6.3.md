## 0. 問題 — v4.1 の PASS は査読に耐えない形だった

v4.1 は基本 QNEC の最小ギャップ +4e-5 ≥ 0 を示したが、強形 (2d CFT の
S″+6(S′)² ≤ 2πT_kk) の最小ギャップは **−3e-5** で、これを経験的な許容誤差 2e-4 で
PASS としていた。「なぜ 2e-4 なのか」に答えがない。台帳 (v6.1) はこれを
QRN-QNEC-001 の limitation として明示した。本バージョンが対処する。

主張も固定する (C1):

> 特定の自由フェルミオン toy model において、離散化・有限サイズの下で
> QNEC 型不等式を数値的に再現した。 — QNEC の新証明ではない (それは C0: 既存定理)。

## 1. 方法 — tolerance の分解と N 収束

PASS 条件を `min_gap > −tolerance`、

```
tolerance = 微分打ち切り誤差 + 有限サイズ誤差 + 丸め誤差
```

に作り直した。見積り方:

- **微分打ち切り**: S″ の中心差分を刻み Δσ=2 と Δσ=4 で計算し、Richardson の関係から
  主誤差 ≈ |S″(Δ2)−S″(Δ4)|/3 (掃引全域の最大値 = 保守的)。強形には S′ の寄与
  12|S′|·err(S′) も加える。
- **有限サイズ**: 波束のモード窓・位置・掃引を N に比例させた相似な設定を
  N = 202/302/402 で解き、最小ギャップの N 間差を取る。
- **丸め**: 相関行列に ±1e-13 の対称ノイズを注入し (固定シード 6303)、S の変動を実測。
  差分ステンシルの係数和 (=1) を掛けて S″ へ伝播。

## 2. 結果

| N | カイラル度 | 基本 min gap | 強形 min gap |
|---|---|---|---|
| 202 | +0.948 | −1.22e-4 | −2.73e-3 |
| 302 | +0.976 | **+3.97e-5** (v4.1 の +4e-5 を再現) | −1.31e-3 |
| 402 | +0.987 | +1.92e-5 | −7.92e-4 |

誤差予算 (N=402):

| 項 | 基本 | 強形 |
|---|---|---|
| 微分打ち切り (Richardson) | 4.27e-3 | 5.58e-3 |
| 有限サイズ (302→402 差) | 2.05e-5 | 5.13e-4 |
| 丸め (実測) | 3.6e-12 | 3.6e-12 |
| **tolerance** | **4.29e-3** | **6.09e-3** |

判定: 基本 +1.9e-5 > −4.3e-3 [PASS]、強形 −7.9e-4 > −6.1e-3 [PASS]。

**強形の負ギャップの正体**: −2.73e-3 → −1.31e-3 → −7.92e-4 と N に対して単調に縮む
(べき ≈ N^(−1.8))。これは**連続極限で消える離散化効果**であり、「QNEC 破れの兆候」では
ない。v4.1 の tolerance 2e-4 は結果的に妥当だったが、根拠は今回初めて与えられた。
基本ギャップも N=202 では負 (−1.2e-4) であり、「N=302 で正」は偶然ではなく
収束の途中経過であることが分かる (|gap| は N とともに単調減少)。

## 3. 陰性対照 — 非カイラル (定在波) 波束

理論の予言: QNEC は状態を選ばない定理なので**対照でも成立し続ける**。一方、
「共動する光的変形 (x−v_F t=const) で ΔS が凍結する」のはカイラル状態**固有**の
性質なので、対照では壊れるはず。

| 量 | カイラル波束 | 定在波 (対照) |
|---|---|---|
| カイラル度 Σj/(v_F Σh) | +0.976 | **+0.000** |
| 共動変形での ΔS 変動 | 6.0e-5 | **4.3e-1 (7200 倍)** |
| 基本 QNEC min gap | +4.0e-5 | +6.2e-5 (成立) |

「成立すべきものは対照でも成立し、壊れるべきものだけが壊れる」— 検証系が
何でも PASS にする装置ではないことの実証。
