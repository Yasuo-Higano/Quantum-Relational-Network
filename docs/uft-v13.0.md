## 2. 第十三期の物語 — 美しい構成が秤で負けるとき

傾き磁束 T⁴ は、この計画が積み上げてきた問い (対 σ とは何か) への最も鋭い候補解
だった: 指数が世代数を直接数え、射影もラベルも対も原理的に存在しない。構造の検証は
完璧に通った — プラケットは機械精度、縮退は 10⁻¹³ 、Wilson 36 配置で不変。

そして秤に載せたら 75 nats 負けた。

これは失敗ではなく測定である。「構成が存在すること」と「構成がデータを説明すること」
の距離が初めて数値になった。N=6 窓での境界は走査で確定し、その先 (N≥12) は
現行の稠密ヤコビ法の外にある — **限界が曖昧な壁ではなく、道具の仕様として記録された**。
