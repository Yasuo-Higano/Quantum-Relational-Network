## 2. 何が壊れ、何が残ったか

**壊れたもの (FAL-CONTINUUM 発火 — 格子アーティファクトへの降格):**

1. **λ∞ = 32/27 (v23.5)**: N=96 の外挿なし直読み λ∞ = 1.185468(2) が
   32/27 = 1.185185 を偏差 2.8e-4 (収束残差の 187 倍) で棄却。「0.006% 一致」は
   N ≤ 16・mod-2 族 2 点外挿の分解能不足だった。**「32/27 の理論導出」(v24.0 の
   次期本丸) は対象消滅により中止** — PROMPT/6 の判断 (「artifact を厳密に導出
   しても物理的には artifact のまま」) が正しかった。
2. **「格子エンタングルメント温度 2π/λ」(v23.3–v24.0)**: λ は (i) 方向依存
   (λ_x = 1.185468 ≠ λ_y = λ_z = 1.229430) のまま窓で不変、(ii) 演算子整合後も
   方向別定数が残り (Z_x = 0.8435 ≠ Z_y,z = 0.8134, b ≈ 0)、(iii) 同じ推定器が
   1+1D (XX・Wilson とも) では **UV 層から λ = 1** を返す。温度は方向に依存
   できない — λ の実体は **3+1D staggered の NN ボンド演算子の方向別規格化**
   (η 位相が切断面の向きを特別扱いする regulator 物性)。
3. **G∞ ≈ 1.06a² の「G 窓開通」(v23.6)**: G = a²/(4λs) は定義式で、λ の方向
   スキーム依存だけで G^bare が 3.7% 動く (x: 1.0725 / y,z: 1.0341, s∞ = 0.196639)。
   正名は **G_entropy^bare(a; regulator, 方向スキーム)** — 物理的 Newton 定数では
   ない。Gioev–Klich 対数二択も前提不成立 (点ノードに Fermi 面形を対照していた —
   正しい族の holdout では Fermi 面形が最悪)。

**残ったもの (むしろ強くなった):**

1. **BW 線形形の精密成立**: K_A の NN プロファイルは切片なしの純線形
   (残差 < 2e-6)・taste 清浄 (B_x ≡ 0 厳密)・NNN 清浄 (< 3e-8) — 「形は無罪」
   (v23.1) が 5 桁精度に強化された。壊れたのは規格化の解釈だけ。
2. **1+1D では BW が UV 層から成立** — 推定器と器械の健全性の決定的証明。
3. **定数面積則**: s∞ = 0.196639 (点ノード予想の補正族つき、holdout 検証済み。
   副次項は βlnN/N² を選好 — 解釈は未確定の記録)。
4. **v23.4/23.5 の数値自体**: 全て再現された (5 桁一致 + numpy/LAPACK 独立照合)。
   壊れたのは解釈と外挿。

## 3. 器械知識の追加 (第二十五期の分類学)

- **厳密ブロック分解**: x 開放 staggered 半空間の C_A は (ky ペア, kz) の N²/2 個の
  N×N 実対称ブロック — dense N³ (限界 N=16) → N=192 級へ (~1000 倍)。
- **κ 予算則**: ボンド ξ の測定には |κ| ≈ 8ξ+12 のブーストモードが要る (実測)。
  精度床 (f64: κ≤32 → ξ≤2 / DD: κ≤69 → ξ≤5–6) が信頼窓を決める。
  **クランプ 2 点梯子 (1e-30/1e-26) の一致域のみ採用**が測定規約。
- **対称性 U = (y→y+1)∘((−1)^z ゲージ)**: B_x ≡ 0・A_z ≡ 0 の厳密証明 —
  abs 推定器 (v23.4) は x ボンドで無害だった。
- **対称性 W = swap(y,z)∘((−1)^{yz} ゲージ)**: y-hop ↔ z-hop を写像し合う厳密
  対称性 (η の検算: 旧 y-hop → (−1)^{x+z} = 新 η_z ✓, 旧 z-hop → (−1)^x = 新 η_y ✓)
  — v24.3 で観測した **λ_y = λ_z の 7 桁一致は厳密**と証明された。独立な λ は
  {λ_x, λ_⊥} の 2 つだけ (切断法線に対する縦/横)。
- **登録設計の教訓 (3 件の開発記録)**: ゲートの較正流用は誤発報する
  (f64 ξ*、対称性ゲートの窓外、Wilson の ξ*) — ゲートは器械の実測に貼り、
  物理判定は分岐文へ。
- **外挿より直読み**: 有理数仮説は「候補探索に使っていない大 N」の直読みで裁く。
