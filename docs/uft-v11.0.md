## 2. 第十一期の物語 — 無害に見えた仮定の解剖

出発点は「対角世代対を導出する」だった。結果は 3 回裏切られ、そのたびに理論は強くなった:

1. **導出するつもりが棄却された** (v10.1)。データは対の自由度に Occam 罰以上の値を
   払う。仮定は「無害な規約」ではなく、観測にかかる物理だった。
2. **主結論は生き残った** (v10.2)。対を正直に marginalize しても 3 枚のトーラスが
   勝つ。~10¹³ 項の証拠和は、打ち切り誤差を定理で抑えて厳密に計算できた。
3. **幾何への還元は失敗し、それが発見だった** (v10.3)。「対 = Wilson 細分化 + 向き」
   という綺麗な予想は総当たりで反証され、対は**どの幾何変形とも独立な離散自由度**
   (どの Landau タワー同士を 1 つの 4D 場に束ねるか) だと確定した。
   自分の推測 (v10.1 §4) への正誤表も 1 件発行した。

方法論として残るもの: **仮定は模型空間に昇格させて検定する**。昇格 → 検定 → 棄却 →
頑健性 → 分類、の 5 手が 1 日で回るのは、第七期以来の装置 (退化検査・証明付き誤差・
正誤表の規律) が揃っているからである。

**装置の小さな発見 2 件** (本統合で検出・修正):

   `v*_*` に修正 (集計規則の初の改訂)。
2. v10.1 が物理の答え「No」に [FAIL] トークンを流用していた — [PASS]/[FAIL] は
