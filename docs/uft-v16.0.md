## 2. 第十六期の物語 — 語る機械から裁かれる機械へ

1. **主張の依存が数えられた。** 96 主張・113 辺の DAG が非循環・等級単調 (C2 の定理は
   現象論と解釈に依存しない) であることが Rust と Prolog の独立実装で機械検証され、
   「どの仮定を抜くと何が落ちるか」が集計になった (ASM-LATTICE 51 / ASM-GAUSS 27)。
   その集計が次の的 (連続極限・core 拡張) の優先度を決めた — 監査が研究を導いた。
2. **core がガウスの外に出た。** Gauss 拘束を Hilbert 空間の形として解いた相互作用系で、
   「同じ網の異なる読み出し」が成立し (閉じ込めの 4 つの顔: σ・v・Δ・S 成長)、
   その全量が連続極限監査を通った。
3. **上位原理の候補が秤にかけられ、全て落ちるか未決に終わった。** さらに予言の
   holdout 採点が「この窓は CP を作れない」を突きつけた — σ・Wilson 線の既約性は
   強まり続けるが、探索空間は「CP を作る窓 = 非因子化幾何」へと初めて**観測によって**
   絞られた。傾き T⁴ (v12–v13) は美的挑戦から必要条件になった。
4. **重力の側は梯子が閉じた。** 第一法則 (一次)・正準エネルギー (二次)・BTZ (全次数,
   2D/AdS₃) が同じ格子データの上に立ち、3+1D の一般化だけが残った。
