**攻める問題**: ニュートリノ質量・混合の起源、およびバリオン非対称(初出)。

## 1. 結果

### A. シーソー = GUT スケールの秤
m_ν = m_D²/M_R に観測値(m_ν=0.05 eV、m_D=174 GeV: SO(10) ではトップと同源)を入れると
**M_R = 6×10¹⁴ GeV** — v3.1 の結合統一点(1〜3×10¹⁶ GeV)と独立に同じ場所を指す。
右巻き ν は v3.1 の SO(10) スピノル 16 が予言する場そのもの。

### B. PMNS 混合の統計(FN 電荷 q_L=(1,0,0) + アナーキーな M_R、2000 MC)

| 量 | 予言(中央値 [16%,84%]) | 観測 |

| θ₁₂ | 22° [10°, 41°] | 33.4° ✓ |
| θ₂₃ | 43° [21°, 65°] | 49.0° ✓ |
| θ₁₃ | 15° [7°, 30°] | 8.6° ✓ |
| Δm²₂₁/Δm²₃₂ | **0.03** [0.00, 0.21] | 0.030 ✓ |

クォーク(小混合、v3.2)と対照的に、ニュートリノは**大きい混合がデフォルト**で出る —
観測パターンの定性的説明。

### C. レプトジェネシス
M₁~10¹⁰ GeV の右巻き ν の CP 非対称崩壊 → スファレロン転換で
**η_B ~ 1.3×10⁻⁹(観測 6.1×10⁻¹⁰)— 桁が合う**。
「宇宙から反物質が消えた理由」はニュートリノの重い相棒の崩壊でありうる。

## 2. 考察

右巻き ν を 10¹⁰⁻¹⁵ GeV に置く一手で、(a) ν の異常な軽さ、(b) 大混合、
(c) バリオン非対称、の 3 つの謎が同時に桁で解ける — 経済性は仮説の強い状況証拠。
検証(2030 年代): 0νββ(マヨラナ性)、CP 位相 δ、質量順序。
正直な留保: M_R・FN 電荷は手で置いた(機構の実証であり値の導出ではない)。
