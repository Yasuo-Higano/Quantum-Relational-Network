# 量子情報網理論 v24.4 — 演算子整合: λ は温度ではなく方向別演算子規格化

**第二十五期の本丸 II。判定 (b) — Z の方向依存が確定**。固定 ξ の比 λ(ξ) = πξ/A(ξ)
は「傾き Z」と「境界オフセット b」を縮退させる — 演算子整合
A(ξ) = Z·πξ + b (窓 ξ ∈ [3, ξ*], DD + クランプ梯子) がこれを分離した。

## 2. 帰結 — λ の解釈の確定

PROMPT/6 の判定リスト第 3 項「演算子整合後も方向ごとに別定数が残る →
Lorentz 不変な連続極限が未成立」が該当する。すなわち:

- **λ_dir = 1/Z_dir は温度ではない**。「格子エンタングルメント温度 2π/λ」
  (v23.3–v24.0) の解釈は棄却 — 温度なら方向に依存できない。
- 実体は **NN ボンド演算子の方向別規格化**: K_A の NN 成分は
  (2π/λ_dir)·ξ·h_dir で、λ_x = 1.18547 / λ_y,z = 1.22943。staggered の
  η 位相構造が切断面の向き (x̂) を特別扱いするため、h_x と h_y,z は
  連続 T⁰⁰ への流れ込みが異なる — v23.2 の「長距離成分が modular 重みの
  ~16% を担う」の裏面 (NN 係数の不足分は長距離が運ぶ)。
- 1+1D では同じ推定器が λ = 1.0000–1.0002 を返す (v24.5) — この方向依存
  規格化は **3+1D staggered 離散化の性質** (regulator 物性) であって、
  BW 定理の破れでも推定器の欠陥でもない。
