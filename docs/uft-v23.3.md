# 量子情報網理論 v23.3 — バケット別応答比: 格子エンタングルメント温度 λ の発見

**第二十四期。事前登録 (b) — 増幅説の反証** — だが数字が真犯人を自白した:
**全 NN バケットの応答比が ρ ≈ 1.19–1.23 のほぼ一定係数** — v23.1 の「1% 一致」は
形 (相対プロファイル) の一致であり、**絶対規格化が λ ≈ 1.186 ずれていた**。
格子のエンタングルメント温度は 2π ではなく **2π/λ** である。

## 1. 方法と結果

δ⟨K_A⟩ と δ⟨K_boost⟩ (線形 NN) をバケット別に同時集計し、応答比
ρ_b = δK_boost[b]/δK_A[b] を測定:

| d | ρ(x-NN) | ρ(y-NN) | ρ(z-NN) | 増幅率 A | 1/R₂ 再構成 |
|---|---|---|---|---|---|
| 0.5 | 1.2141 | 1.2098 | 1.2306 | 4.7 | 0.7926 ✓ |
| 1.5 | 1.1899 | 1.2343 | 1.2313 | 345 | 0.8646 ✓ |
| 2.5 | 1.1214 | 1.1925 | 1.3807 | 8061 | −0.0713 ✓ |

- 1/R₂ の再構成が v22.2/v23.1 の実測 R₂ と 4 桁一致 — 装置の自己整合 ✓。
- ρ が 1±3% でない (max 38%) → 登録 (b)。**しかし ρ はバケット間・d 間で
  ほぼ一定 (浅域 1.19–1.23)** — ランダムなずれではなく共通の規格化因子。

## 2. λ の同定

v23.1 の生データを絶対規格化で読み直すと: T(ξ)/(πξ) = 0.8434, 0.8424, 0.8422,
0.8300, 0.8368 (ξ=1..5) — **λ = πξ/T(ξ) ≈ 1.186 が ξ 非依存の定数**。
つまり K_A の NN 成分 = (2π/λ)·ξ·T⁰⁰ の形で、modular 重みの ~16% は
長距離項 (v23.2 の相殺構造の相手方) が担う。R₂ の浅域不足 0.79–0.90 は
1/λ ≈ 0.84 と長距離寄与の d 依存の組み合わせとして定量的に閉じる。

## 3. 帰結 — G 窓の開き方が見えた

λ が (i) 連続極限で 1 に戻る (格子アーティファクト) か、(ii) 有限繰り込み λ* に
留まるかを v23.4 (λ(N) 系列, N = 8..16) が裁く。**どちらでも G 窓は開く**:
(i) なら λ(N) 補正 + 外挿、(ii) なら λ* を温度規格化に組み込んだ読み出し。
「開かない窓」ではなく「2π を素朴に信じた読み方」が障害だった。
