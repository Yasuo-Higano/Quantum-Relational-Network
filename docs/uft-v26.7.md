## 1. 何が凍結されていたか (覗き見なしの証明線)

1. **機構と予言** (v26.3 §3, 計算前に登録): 自由場では全エネルギー流 J^E_tot が
   保存量 → エネルギー双極子の行列要素が消え χ_00 ∝ q⁴。「相互作用を入れると
   J^E_tot は保存でなくなり、χ_00 は q² に戻るはず」。
2. **プロトコル** (spec §7, コミット 157ca53 — 実装前): 1+1D staggered 鎖 (twist,
   半充填)・V Σ n_x n_{x+1} (斥力固定)・V ∈ {0, 0.5, 1.0}・N ∈ {10..16}・
   fit 規則 p = ln[χ(q₂)/χ(q₁)]/ln(q₂/q₁)・**判定 p(V=1.0, 最大 N) < 3.0**・
   モデル交換禁止。predictions.yml PRED-013。
3. **実行前の補足宣言** (spec が未指定だった 1 パラメタ): 採点対象は m = 0.5。
   理由: m = 0 の鎖 + 最近接相互作用は JW で XXZ となり**可積分** — 予言の前提
   (一般の非可積分相互作用) を満たさないため。m = 0 線は対照として記録のみ。
4. **走行前較正** (自由場の厳密参照のみ使用 — 相互作用データには一切触れない):
   S3 対照ゲートを採点サイズ N=16 に設置 (自由 p は N=10 で 3.23 — 有限 q 補正)。
   副産物の発見: **m=0 自由場は N≡0 (mod 4) で χ(q₁) が厳密零** (選択則) —
   可積分対照は p でなく χ 値で記録することを走行前に決定。

## 2. 結果 — q⁴ が q² に折れた

many-body 厳密対角化 (Lanczos + 共役勾配, dim ≤ 12870, 決定的)。器械認証:
V=0 の E₀ と χ_00(q₁,q₂) が 1 粒子 Dirac 海の厳密 Lehmann と一致 (7.8e-13 /
混合許容の 0.000 倍) — 基底・JW 符号・CG の端から端の証明。T_00(0) = H は bit 厳密。

| V (m=0.5) | p(N=10) | p(N=12) | p(N=14) | **p(N=16)** | 多体 gap |
|---|---|---|---|---|---|
| 0.0 (自由) | 3.226 | 3.832 | 3.573 | **3.795** | 1.07 |
| 0.5 | 2.076 | 2.002 | 1.959 | **1.942** | 1.46 |
| 1.0 (採点) | 1.772 | 1.764 | 1.781 | **1.804** | 1.84 |

- **[S4] p(V=1.0, m=0.5, N=16) = 1.804 < 3.0 ⇒ PRED-013 hit。** V を入れた瞬間に
  p ≈ 2 (q²) へ落ち、N 依存はほぼ無い (1.94–2.08 / 1.76–1.80) — 有限サイズの
  綱渡りではない。χ(q₁) は自由場の 2.5e-4 から V=1.0 で 8.0e-3 へ **32 倍** —
  q² 係数 (c₁^00) が相互作用で明確に生成された。
- **誘導重力経路への含意**: v26.3 の「c₁^00 = 0 (q⁴ 保護)」は自由場の保存則による
  偶然であり、**相互作用する matter は T_00 チャネルにも q² 運動項を生成する**。
  誘導重力の source 構造は matter の相互作用に依存する (bare 量の非普遍性の
  もう一つの顔)。
- **[S5] 可積分対照 (m=0, 記録のみ) — 事前の期待は外れた (機構の精密化)**:
  ヘッダに書いた期待「XXZ 可積分性により J^E 保存 → q⁴ 残存」に反し、V≠0 で
  χ(q₁) は 0.045–0.11 (抑制なし)。**後知恵の解剖**: JW 後の V n n は
  Δσzσz + **一様縦場** を生む。可積分性が保存するのは場なし部分の J^E_XXZ で
  あって、**我々の T_00 (V 項込みの局所エネルギー密度) の連続の式に現れる完全
  カレントには spin current 片が混ざり、これは Δ≠0 で保存されない** — 保護は
  壊れて当然だった。「保存 ⟺ q⁴」の機構はこの対照でも整合的 (保存が壊れる理由が
  可積分性の外にあった)。期待の外れは記録として残す。
