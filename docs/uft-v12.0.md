## 2. 第十二期の物語 — 二つの「閉じ」と一つの「測り」

1. **理論空間の外周が閉じた** (v11.1)。単純群の走査は SU(3)×SU(2)×U(1) の窓 (v3.1〜)、
   U(1) の階段 (v6.2/v7.1/v8.2)、そして例外群 (v8.2 の E₆ + 本期の G₂/F₄/E₇/E₈) で
   一周した。カイラル物質が入れる扉は数えるほどしかなく、その全てが SM 1 世代の形を
   している。
2. **core 移行が閉じた** (v11.4)。第七期に始めた「toy の集まりから一つの網へ」は、
   4 模型 + 読み出し群 (幾何・熱・因果・エネルギー条件・**分解**) で完結。最後の
   読み出しは「テンソル分解そのものを状態に語らせる」もので、v1.0 の未解決問題 6-1
   (テンソル分解の起源) に机上の答えでなく装置の答えを与えた。
3. **機構の距離が測れるようになった** (v11.2–v11.3)。タワー整列 σ をめぐる機構の
   はしご — 対角 (−23.61) → 一様オービフォールド (−21.83) → 磁化の向き (−20.86) →
   抽象 S₃ (−19.86) — は、各段の高さを nats で刻む。「あと 1 nat」が、未試行の幾何
   (傾き磁束・固定点・離散トーション) か「σ は幾何に還元されない離散データ」かの
   分岐点として残った。

## 4. 第十二期のテーゼ

> 走査は閉じることで力になり、機構は測れることで科学になる。
