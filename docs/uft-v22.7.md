# 量子情報網理論 v22.7 — DD 全経路核: f64 κ 床の突破

**第二十三期 (PROMPT/4 並列研究 2)。事前登録 (a) 成立 — 床の突破**。
モジュラー核の f64 分解能床 (κ ≤ 32.2 飽和, v19.1 以来の既知の壁) を、
経路全体の double-double (DD) 化で除去した。深部カナリアが完全復活し
(R₁ = 0.99988/0.99991 @ d = 2.0/2.5)、G 窓の測定域が d=2.5 まで開いた。

## 1. 方法 — DD 全経路

1. **DD 部分空間 Newton**: f64 jacobi の占有部分空間を 1 回の Newton 補正で
   1e-30 級へ。X[m,k] = (⟨m|Hv_k⟩ − θ_k⟨m|v_k⟩)/(θ_k − λ_m) — 第 2 項が f64 基底の
   交差汚染を厳密相殺する。閉殻ギャップ (0.2411) が全分母を保証。
   実測: 部分空間残差 9.77e-14 → **1.5e-27** (二次収束)。
2. **DD C_A → DD cyclic Jacobi (864²) → 真の κ**: c_min = 7.2e-18, **κ_max = 39.5**
   (f64 飽和 32.2 の向こう側を解像)。
3. Gauss 対 rank-2 回転・エントロピー・⟨K_A⟩・⟨K_boost⟩ を全て DD で評価。
   α = 0.005/0.010 の漸近対 (DD の分解能が許す微小 α — f64 では雑音に沈む)。

## 2. 結果

| d | R₁(DD) | R₁(f64, 参考) | R₂ (記録) |
|---|---|---|---|
| 1.5 | **1.00000** | 0.99989 | 0.86463 |
| 2.0 | **0.99988** | 0.980 [死] | 1.95696 |
| 2.5 | **0.99991** | 0.947 [死] | −0.07092 |
| 3.5 | 0.43736 | 0.098 [死] | 0.00014 |

- **[境界書き換え]** v22.4 の「カナリア死境界 d ∈ (1.5, 2.0)」は f64 固有の現象と
  確定。真の信号消失 (v19.4 の占有縮退) は d ∈ (2.5, 3.5) から始まる。
- **深部 R₂ の初の有効測定**: R₁ = 1 が完璧な d=2.0/2.5 で、R₂ は 1.96 → −0.07 と
  振動・符号反転。**rank-2 波束プローブは真の modular 演算子 K_A には深部でも完璧に
  結合するが、K_boost (v19.2 凍結の格子ブースト重み) とは d ≳ 2 で乖離する** —
  v22.6 の「演算子レベルの残存仮説」が深部で直接定量化された。浅部の R₂ 不足
  (13–21%) も同根とみるのが自然。
