# 量子情報網理論 v26.4 — c₁ は等方: λ の異方性はバルク応答の性質ではない

**第二十七期第 4 版。判定 (a) — 8 検査 PASS、branch B (等方)。**
v24.x の中心的謎のひとつ「λ_x ≠ λ_⊥ (方向依存の BW 規格化)」を応答関数側で検査した:
誘導重力運動項の係数 c₁ (v26.3) は格子方向に依存するか。答えは **No — バルク応答の
「真空弾性率」は 5 桁で等方**であり、λ の異方性は物性 (輸送・応答) ではなく
**切断に相対的なモジュラー構造の性質**だと確定した。

## 1. 器械 — ブロック対エンジン (q ∥ ŷ)

T(qŷ) はブロック (ky,kz) → (ky+q,kz) を結ぶ (π 跨ぎで成分 {+0,+π} スワップ)。頂点は
全て実 (x ボンド: 振幅 1 / y ボンド: cos(k_y+q/2) / z ボンド: σx⊗cos kz / 質量: σx⊗σx)。
認証: [S0a] 内蔵 dense (v26.2 機構の写経) と **abs 5.1e-14** で一致 (8 値, 折返し込み)、
[S0b] x 経路は REF262 の 20 値 (5.0e-7) と v26.3 の c₁ 公表値 (4.9e-6) を回帰、
[S1] χ_yy(0) の 2 エンジン恒等 2.2e-16。

## 2. 主結果 — c₁ 異方性行列 (N=64; 窓系統 ≤0.6%, N 移動 ≤1.2%)

| m | c₁[xx](x̂) | c₁[yy](x̂) | c₁[xx](ŷ) | c₁[yy](ŷ) |
|---|---|---|---|---|
| 0 | +0.01826(6) | −0.01556(9) | −0.01556(9) | +0.01833(7) |
| 0.5 | **+0.01595** | **−0.01264** | **−0.01264** | **+0.01595** |

- **[S3 branch B] 等方**: 縦組 c₁[xx](x̂) = c₁[yy](ŷ) (m=0.5 で 5 桁一致、m=0 で差 0.4% =
  系統内)、横組 c₁[yy](x̂) = c₁[xx](ŷ) (完全一致)。**バルクの応答テンソルは立方対称**。
  後知恵の理論的整合: staggered の η 位相割当は座標交換のゲージ同値 — バルク応答は
  x↔y 対称でなければならない (v26.2 で見えた x/y 非対称は twist 境界の有限サイズ効果
  で、N とともに消えることも確認済み)。
- **含意 (v24.x の謎の解剖の完成)**: λ_x ≠ λ_⊥ はバルクの異方性では**ない**。
  モジュラーハミルトニアンの規格化は「切断面の向き」に相対的な量であり、その方向
  依存性 (v25.2 で BZ moment として解析的に閉じた) は応答関数には現れない。
  「regulator の異方性はどこに住むか」という問いに、部門の境界線が引けた:
  **応答 (バルク) = 等方 / モジュラー (切断相対) = 異方**。
- **弾性構造**: 縦係数は正 (+0.01595)、横係数は負 (−0.01264) — 等方弾性体の
  2 独立係数 (縦・横) に対応する構造。両方とも質量とともに走る (v26.3 の非普遍性は
  両チャネルで成立)。

## 3. 開発記録 — 転置縮約バグと三者裁定 (器械の教訓として重要)

run1 は S0a が 0.28 で FAIL。numpy による三者裁定 (rust-dense / rust-block / numpy) で
特定した機械的バグ: **chi_pair が頂点行列を転置して縮約していた** (格納規約
v[標的+元·dim] に対し o[k + r·dim] で読んだ)。深い罠: **各片 (x/y/z 単独) は転置で
±自分自身に戻るため、単片の χ は不変 — 片ごとの照合は全て通過した**。壊れるのは
片間の相対符号だけで、それが T_00 の保存キャンセル (χ_00 の q⁴ 保護) を破り 67 倍の
過大評価を生んだ。教訓 (v24.x ゲート較正の教訓群に追加):
**単片照合は転置バグに不感 — 保存則によるキャンセルを含む合成量でこそ照合せよ**。
修正後 S0a は 5.1e-14。物理値は認証後の run のみ公表 (本結果)。
