## 2. 考察

1. **「何が在るか」は観測者の運動状態の関数である。** 同じ量子状態が、慣性系では
   「空っぽの真空」、加速系では「温度 a/2π の熱浴」。粒子数も温度も絶対的実在ではなく、
   状態と観測者の**関係**である — QRN の関係論(A0)の最も鋭い実例。
2. **すべての地平線は同じ機構で熱い。** 加速地平線(本計算)、ブラックホール地平線
   (Hawking 温度 κ/2π → v0.8 の蒸発)、宇宙論的地平線(Gibbons–Hawking 温度 H/2π —
   v2.4 のスペクトル (H/2π)² はまさにこれ)。v0.7 のモジュラー核 β(x) はその静的な種だった。
   **「地平線=情報の遮蔽=熱」の三位一体が、静的(v0.7)・動的(v3.4)の両面で検証された。**
3. 実験状況(正直に): Unruh 効果の直接検出は未達(T=1K に要る加速度 ~10²⁰ m/s²)。
   類似系(アナログ地平線、Hawking 輻射の音響版)では統計的に観測が報告されている。
