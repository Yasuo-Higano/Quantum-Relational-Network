  軽い ν ほど大混合) に必要な**強い非対角性/near-singular 構造**を作れない
  (特異 skip が 0 件だったこと自体が証拠 — 良条件すぎる)。

## 2. 次の構造候補 (登録)

1. **磁束つき φ_B**: B−L 破れ場自身が巻き数を持つ (位相構造が M_R に非対角の
   干渉を作る — CP アークで位相が J を作ったのと同じ型)。
2. **ν_R を別の幾何に**: ν_R だけ異なるシアー/アスペクトのトーラス係数
   (対角化キャッシュの拡張で可)。
3. Weinberg 演算子直行 (シーソーを経ない LLHH の直接重なり)。

## 3. 残高への効き

- レプトン・アークの現在地: 最小 (1/4) → 幾何 M_R (2/4)、**θ23 = 一貫した壁**。
  「重なりの階層性 vs 大混合」の緊張は、クォークの \|V_us\| (対称の病気 → τ で解決)
  と同じ型の**構造診断待ち**の段階に入った。
- PRED-009 (θ12) は幾何 M_R 設計なら hit — ただし**事前登録した採点は v18.3 の
  設計に対するもの**なので status は変えない (設計変更後の hit は新しい登録で
  数えるのが規律 — 本版の帯は QRN-LEP-002 の記録に留める)。
