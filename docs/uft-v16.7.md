## 1. 結果

| | N=18, s=1 | N=36, s=2 (同一 τ_re) |
|---|---|---|
| 縮退幅 | 1.9e-13 | **7.9e-13 (厳密のまま)** |

| \|V_cb\| / \|V_ts\| factor | 2.15 / 2.30 | 1.41 / 1.52 |
| factor 5 以内 | 12/12 | 12/12 |

三つの結論:

1. **\|V_us\| 過大は構造である。** 同一 τ_re で格子を倍にしても 2.51 → 2.57 と不変 —
   格子効果 (A·N^(−p)) ではなく、τ_re = 1/18 のシアー模型がもつ物理的な予測。
   Wilson 粗さ (v16.6 で無罪)・格子の粗さ (本版で無罪) が消え、**残る唯一のレバーは
   τ_re そのもの** — N=36 は τ_re を 1/36 刻みで走査できる。
2. **指数 3 は倍格子+シアーでも厳密。** 傾き磁束が N=18 で割れた (v13.2) のと対照的に、
   シアー族の 3 重縮退は N に対して堅牢 — 世代数はこの族の安定な性質である。
3. **証拠は連続極限で強くなる。** lnZ₁₀ が +2.03 nats 改善し、MAP の \|J\|・\|V_cb\|・
   \|V_ts\| も揃って良化 — 模型の弱点のうち \|V_us\| 以外は格子起源の成分を含んでいた。
   連続極限の方向で 12/12 が締まっていくのは、模型が「格子の偶然」でないことの証拠。

## 2. 残る問いの形

スコアカードの弱点は一点に凝縮された: **τ_re = 1/18 は Cabibbo 角を 2.5 倍過大に出す**。
問いは「どの τ_re が \|V_us\| と J を同時に置くか」— これは v16.3 の綱引き (シアーは
質量階層を壊す) と合わせて、τ_re の一次元走査で答えが出る形になっている。
N=36 の τ_re ∈ {1/36, 2/36, ..., 5/36} 走査が次版の的 (1 点 14 分 × 5 — 時間級)。

## 3. 限界

- N は 2 点 (18, 36) — 冪 fit には 3 点目 (N=54: jacobi 5832² — 時間級) が要る。
  ただし判別 (不変 vs 半減) には 2 点で十分。
- MAP 点評価の改善 (\|J\| など) は谷の移動込み — 事後帯での確認は未実施。
