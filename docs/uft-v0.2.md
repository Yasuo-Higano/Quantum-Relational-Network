## 1. 主張

時間 t を虚時間 τ = it に回す(Wick 回転)と、量子力学の遷移振幅は統計力学の分配関数に**厳密に**変わる:

```
Z_量子 = Tr e^{-iHt/ħ}   --( t → -iτ )-->   Z_熱 = Tr e^{-βH}
経路積分  ∫Dx e^{iS/ħ}   --------------->   ∫Dx e^{-S_E}   (1次元弾性鎖のボルツマン分布)
```

- 量子の「ħ による揺らぎ」と熱の「kT による揺らぎ」は同じ確率構造を持つ
- **逆温度 β = 虚時間方向の周期**(のちの Unruh/Hawking 温度、KMS 状態の核心)
- プランク定数と温度は交換レート: 量子力学とは「温度ゼロでも消えない熱力学」である

これは類推ではなく恒等式である。本バージョンではこの同型を使い、**量子現象(零点エネルギー・トンネル効果)を古典統計力学のモンテカルロ法だけで再現**する。

## 3. 考察 — この橋が統一に対して意味すること

1. **時間と温度は交換可能な概念である。** τ 方向の周期 β が温度を決める。これは後に:
   - 加速観測者が真空を温度 T = a/2π と見る(Unruh 効果)

   の数学的核心となる。「重力 × 量子 = 熱」の最初の顔。

   古典乱数だけで計算できた。量子力学の神秘性の一部は、確率構造の言い換えで消える。

3. **場の量子論への道が開く。** S_E は 1 次元弾性鎖のエネルギーだった。これを 2 次元の膜(格子場)に
   すれば、そのまま場の量子論になる(v0.3)。「粒子の QM → 場の QFT」は次元を増やすだけ。

4. **統一仮説への布石**: 「量子状態 ↔ 熱的状態」の辞書は、v0.5〜v0.7 で
   「エンタングルメント・エントロピー ↔ 熱エントロピー ↔ 幾何(面積)」へ拡張される。
